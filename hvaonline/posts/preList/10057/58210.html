Lệnh ATTRIB tùy vào tham số truyền vào sẽ thực hiện chỉ đọc, file hệ thống, ẩn... và ngược lại, ví dụ bạn muốn hiện những file ẩn trong 1 thư mục c:\windows bạn thực hiện lệnh sau:
ATTRIB -h /s C:\windows tham số -h thể hiện không ẩn ( Not hidden) tham số s để tìm tất cả các file trong thư mục chỉ định.

Trong trường hợp máy tình bị nhiễm virus bạn không thực hiện được việc bỏ thuộc tính hidden hay system của file đó, bạn cũng không biết được file đó có tồn tại hay không thì bạn có một cách nữa để kiểm tra, bạn vào start -> Run gõ vào D:\autorun.inf và nhấn enter. nếu có thì file đó sẽ được hiển thị dưới dạng notepad để mình xem nội dung. trường hợp trên là mình kiểm tra ổ d:\ trên máy có file autorun.inf hay không.
Có khá nhiều biến thể virus file tạo ra file này để thực thi nó khi truy cập vào USB, nếu mình vào theo cách thông thường thì vô tình kích hoạt virus.
