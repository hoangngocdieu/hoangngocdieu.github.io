TroMessenger 1.0 Command Listing
/Cmd - Chạy CMD. Ex: /cmd dir C:\*.txt
/Time - Hiển thị đầy đủ ngày, giờ của máy nạn nhân.
/screenshot - Chụp ảnh toàn bộ màn hình máy nạn nhân và Up lên mạng.
/CloseCD - Closes the CD-ROM Drive Door
/EjectCD - Ejects the CD-ROM Drive
/download - Downloads Một file từ internet.

/getfile - Download file từ máy của Victim
/run - Chạy một file từ máy Victim.(Chú ý : Phải đầy đủ đường dẫn.)
/cancel - Huỷ bỏ lệnh vừa thiết lập.(Đang thực thi)
/restart - Restarts Máy của Victim
/shutdown - Shutdown máy Victim
