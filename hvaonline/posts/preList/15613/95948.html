Tóm lại dùng Search tìm đến file muốn chạy copy nó ra 1 bản rồi đổi tên (giữ nguyên đuôi)
Nếu vẫn không thể chạy -> download về một chương trình tương tự như task manager và kill chương trình nghi ngờ
hoặc dán tasklist>c:\check.txt để xem file check.txt và lại kill các chương trình nghi ngờ bằng taskkill /f /im *.exe trong run chạy nhiều lần hoặc trong shortcut. Nếu kill thì máy logoff -> suspend bằng pssuspend.exe
Nếu cả run, cmd, command, shortcut không chạy được -> viết chương trình kill bằng debug
Nếu debug cũng ngủm luôn thì vào MS-DOS để del
Không về được MS-DOS -> dùng cd rom hoặc ổ A:
Cũng bị khóa luôn thì tháo Pin và xóa CMOS
Nếu không xóa dần windows hoặt move file rồi restart - format c: /s
Đặt ổ E: vào lệnh E: hoặc F: cho chạy autorun.exe gì đó rồi cài lại win để kiểm soát máy
