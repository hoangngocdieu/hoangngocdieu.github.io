Sử dụng file autorun.inf để chạy tự động đĩa CD đã ghi cò thể giúp người sử dụng máy tính tiết kiệm được thời gian khi phải tìm các chương trình phù hợp. Những thứ bạn cần sẽ là 1 trình biên tập text và phần mềm ghi đĩa bình thường(cái này có lẽ bạn biết)
Một cấu hình thông thường của file autorun.inf sẽ tự khởi động chương trình khi bạn đưa đĩa CD vào ổ, hiển thị biểu tượng o chế độ windowns Explorer hoặc 1 loạt danh sách các phần mềm xử lý.File autorun.inf hoàn toàn là 1 file text,có định dạng giống như sau:[autorun]open=myprogram.exeicon=myicon.ico
Trong đó:myprogram.exeicon:Biểu tượng của file cài đặt
File biểu tượng nằm ở thư mục gốc của đĩa CD
Thường thì chương trình bạn muốn chạy sẽ không nằm trong thư mục gốc của đĩa CD
Nếu chương trình nằm trong thư mục gốc,bạn sẽ phải bổ sung thêm đường dẫn:[autorun]open=foder1A/foder1A/myfile.exeicon=myicon.ico
Cũng có đôi khi bạn cần bổ sung thêm 1 đối số (argument) vào chường trình để nó có thể tự động chạy:
[autorun]open=myprogram.exe/argumenticon=myicon
File autorun.inf sẽ tạo rất nhiều thuận lợi cho khách hàng nếu bạn là 1 doàn nghiệp,muốn gữi sản phảm là 1 đĩa CD có chứa nhiều file PDF,HTML,bài thuyết trình.......File autorun.inf sẽ mở 1 file batch và sau đó điều khiển file sử dụng chương trình mặc định để mở.
Một số bạn cứ lầm tưởng file autorun.inf là 1 loài virus.Nếu vậy thì mình viết bài này vô ích rồi .hii.Miễn bình luận nhé

hjz n` ta hỏi 1 câu đơn giản mà trả lời sao thật phức tạp... mình đọc chả hiểu j :-s.
Còn với việc tạo 1 file autorun.inf thì theo mình vô Notepad soạn thảo 1 đoạn văn bản dạng sau:

[AutoRun]
icon=Tên icon (vd: BKAV.exe,0 hoặc autorun.ico) ;Cái này để icon cho ổ đĩa
open=tên file (vd: autorun.exe) ;file autorun.exe nằm trong USB nhé

~phần này nếu bạn muốn làm autorun cho USB thì thêm vào còn nếu để ghi đĩa thì 3 dòng trên là đủ~
shell\open=Open(&O)
shell\open\Command=tên file
shell\open\Default=1
shell\explore=Manager(&X)
shell\explore\Command=tên file
 


sau đó save lại với tên Autorun.inf nhớ chọn file type là All files(*.*) ko nhớ cũng ko sao và lưu vào USB
Cuối cùng, nhấp chuột vào USB ấn F2 và đánh cho nó 1 cái tên rồi enter thì sẽ thấy file Autorun.inf hoạt động như thế nào (qên cái này dành cho USB và ổ đĩa cứng thôi )
