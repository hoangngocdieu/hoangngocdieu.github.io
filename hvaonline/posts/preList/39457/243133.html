theo mình thì c/c++(nếu bạn mới bước chân vào) thì có 3 chủ đề lớn như sau
* các câu lệnh đơn giản ( if,switch,for,....)
* kiểu dữ liệu có cấu trúc ( danh sách liên kết, queue, stack)
* lập trình hướng đối tượng( class )
bạn kiếm khoảng 4,5 quyển sách là ổn.
theo mình thì nhất thiết phải có 2 quyển này "ngôn ngữ lập trình c cơ sở" của phạm văn ất. Và ngôn ngữ lập trình c của quách tuấn ngọc
Nếu nắm vững 3 chủ đề lớn trên kia thì cần khoảng 2 đến 3 tháng. Ít nhất mỗi ngày phải đc 2 bài code
