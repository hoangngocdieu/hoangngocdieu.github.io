ram của VGA không liên quan gì tới ram của máy, card màn hình thì quan trọng là Memory interface, xem thêm về Memory interface ở đây:
/hvaonline/posts/list/20546.html

card của bạn là "Gigabyte PCI Exprees 256 mb - DDR3 - 128bit", memory interface chỉ có 128 bit, nếu bạn mua card memory interface như 256 bit hay cao hơn thì card sẽ chạy với hiệu suất cao hơn.

hình như chỉ có ddr3 là 1066, còn không bạn cũng có thể mua ddr2 rồi overclock lên 1066, nhưng còn tùy ram.
