Bạn cần lưu ý các bước quan trọng sau đây:

1. Mở port 80 trên router (modem)
2. Config IP trên Server APP cho chính xác
3. Vào trang www.No-IP.com để tạo 1 tài khoản free, tạo 1 domain free (http://tenban.ho-ip.com), xong down phần mềm của nó về, đăng nhập bằng tài khoản free để nó cập nhật IP WebServer của mình.
4. Vào đường dẫn sau:C:\Windows\System32\driver\ect (nếu WinServer thì là WinNT), mở file hosts bằng Notepad lên, thêm vào dưới dòng:
127.0.0.1 localhost
<IPLan WebServer> <domain free vừa đăng ký>
rồi save lại.
5. Test domain free
6. Ngồi nhâm nhi càfe thưởng thức thành quả (tùy đường truyền mạng coi nó nhanh hay chậm nữa). Theo kinh nghiệm tui nếu bạn xài đường truyền HOME thì uống khoảng 2 ly nó mới ra đó ^_^...
