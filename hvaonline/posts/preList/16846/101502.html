Câu hỏi của bạn thật khó trả lời vì có thể bạn đang hiểu sai vấn đề về HTTPS

Trang web (tức là phần mã nguồn) của bạn vẫn như vậy, dù có gắn vào HTTP hay HTTPS thì nó vẫn hoạt động bình thường

Hiện giờ trang web của bạn đang đặt ở đâu? Share host hay server bạn có quyền cấu hình?

Nếu trên share host thì bạn xem host của bạn có hỗ trợ SSL hay không và đặt cái phần mã nguồn của bạn vào đúng nơi quy định. Nếu bạn có một server ví dụ như LAMP chẳng hạn, thì bạn cần tìm hiểu cách cài đặt thêm service hỗ trợ SSL

Hy vọng bạn hiểu và không còn hỏi "kỹ thuật lập trình web trên https" như thế nào.
