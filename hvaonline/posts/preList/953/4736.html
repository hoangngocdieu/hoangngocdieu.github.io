Nếu cái máy đó đã từng chạy được giao diện đồ hoạ thì bạn thử gõ lệnh init 5 với quyền root xem.
Nếu nó khởi động chế độ đồ hoạ bình thường thì tốt, sửa file /etc/inittab , tìm dòng
Code:
# Default level
id: x:initdefault:

Thay chữ x bằng số 5 để lần sau khởi động nó chạy level 5 ( đồ hoạ ) mặc định, sau đó gõ lệnh man init để hiểu xem mình vừa làm gì.
Nếu init 5 mà nó ko khởi động đồ hoạ thì xem nó báo cái gì post lên đây tính tiếp.
