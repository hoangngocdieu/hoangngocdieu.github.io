Sẵn đây cho em hỏi. Máy đầu tiên cài Windows Vista. Sau đó vì muốn chơi game và chạy một số ứng dụng nên cài thêm Windows XP2. Muốn dùng cùng lúc 2 chương trình nên em dùng chương trình "Vista Boot Pro" để tạo menu lựa chọn khi boot. Mọi việc tốt đẹp. Về sau, cài thêm Ubuntu 7.04 để học. Trong quá trình cài em cũng chia thêm 2 ổ đĩa, và không đụng chạm gì đến ổ đĩa của XP.
Cài xong Ubuntu. Vô restart lại thì chương trình GRUB chạy, có menu lựa chọn vào Ubuntu hay Windows Vista/Longhorn (loader). Vô Ubuntu thì được, còn chạy cái Windows Vista/Longhorn (loader) thì nó cũng đưa ra 2 lựa chọn: vô Vista hay Xp. Tiếp tục vô Vista thì được, còn vô Xp thì báo lỗi như vầy:

"Error: I/O accessing boot sector file multi(0)disk(0)rdisk(0)partition(1)\BOOTSECT.DOS"

Em đã thử ghost lại Xp mà cũng hổng được, vẫn lỗi y chang vậy. Mấy anh chỉ dùm em cách khắc phục. À, em có 1 ổ cứng chia như vầy:
+ C (primary): Windows Vista
+ Ổ mở rộng gồm:
_D: chứa dữ liệu
_E:Linux
_F:Linux-swap
_G: Windows Xp
_H:dữ liệu
