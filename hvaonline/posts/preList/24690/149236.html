Àh tui chụp hình up lên cho ông xem nhưng forum không có chức năng này thì đẻ tôi viết lại quá:

Bot là chương trình dùng để login account của mình vào một server game online và chơi mà không cần chạy game đó lên, nó có một số chức năng cao cấp hơn auto.
ví dụ đây là phần config server trong file Location của YGX:

IP=210.245.123.70
Port=16101
Protocal=30010 (protocol port cũ của IP 210.245.123.70 mà tôi tìm lúc trước)
PING_IP=203.144.179.134
PING_Port=16010
ServerList=Chiê?nThâ?nCung
ChannelList=Chiê?nThâ?nCung01
CheckUser=1
CheckPassword=0

Khi mở YGX lên chạy và nhấp Login thì nó báo:

Connecting to Game server successfull
Login your account successfull
Invalid Protocol Version
Please try again...

Chủ yếu tôi cần cách để tìm Protocol Version àh.
