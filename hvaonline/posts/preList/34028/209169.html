1. In dealing with a network configuration problem. An IT manager found that the following information on a station: IP address was 199.101.10.32 and the subnet mask was 255.255.255.192. Assume the network ID is correct. What might be the problem?
Đối với IP 199.101.10.32 và sử dụng SM 255.255.255.192 thỳ là /26,
6 bit sử dụng cho host ID, bước nhảy 2^6 = 64

199.101.10.32 network là 199.101.10.0 và BC là 199.101.10.63 em ko hiểu 199.101.10.32 xảy ra vấn đề gì.

2. Divide the IP network address, 140.25.0.0, so that it is capable of supporting 128 network segments.
Ta cần chia cho 128 mạng, vậy số bit cho host cần là 8 bit (2^8 – 2 > 128) (2^7 ko đc)
Có IP address: 140.25.0.0 ----> Subnet mask: 255.255.0.0/24
Bước nhảy: 2^8 = 256
segment 1: 140.25.0.0/24
segment 2: 140.25.1.0/24
segment 3: 140.25.2.0/24
........................................
segment 128: 140.25.127.0/24

3. Divide IP network address, 25.25.0.0, so that it is capable of supporting 65,354 network segments.
Ta cần chia cho 65,354 mạng, vậy số bit cho host cần là 16 bit (2^16 -2 = 65,534>65354)
Có IP address: 25.25.0.0----> Subnet mask: 255.255.0.0/16
Bước nhảy: 2^16 = 65,536
segment 1: 25.25.0.0
segment 2: 25.25.255.255 (sử dụng hết cả lớp B luôn)
Nhưng mà ko chắc lắm ^^.

Em mới học subneting, các anh phân tích giúp em chỗ đúng sai với (không biết có chỗ nào đúng ko nửa ^^) => thank
