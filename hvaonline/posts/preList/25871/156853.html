Nói tóm lại là bạn cần cấu hình DNS trên centos phải không?

nếu vậy thì dùng

# yum install bind bind-devel bind-lib bind-chroot
 


File cấu hình của named nằm trong /var/named/chroot/etc/named.conf.

Mặc nhiên trên centos 5.2 không tự sinh ra file named.conf bạn phải viết nó

Nếu cài đặt gói chroot-bind thì mặc định named.conf nó nằm trong /var/named/chroot/etc
Nếu không cái đặt gói chroot-bind thì mặc định named.conf nó nằm trong /var/named

Tuy nhiên dùng find để tìm kiếm là ok nhất
