HOSTS file hijacking and bank password stealing trojans

Posted by Suzi Turner @ 11:14 pm

HOSTS file hijacking combined with bank password stealing trojans is one of the more egregious spyware tricks currently being seen. Here’s the scenario. A user is infected with a trojan and other malware that, among other things, changes the HOSTS file so that websites commonly used for online banking are wwwected to the spyware pusher/thief’s site which is made to look nearly identical to the real bank site. Everyone in the anti-spyware community knows who these ISPs are.

HOSTS file hijacking can be prevented with a number of apps including several anti-spyware programs and utilities including one of my favorites, WinPatrol.

SunbeltBLOG has an excellent write up describing this trick and a video for demonstration. Host file hijacking is not new on the spyware scene and has been used by CoolWebSearch and similar groups to wwwect users’ browsers to alternative search sites or adware/spyware sites. In many cases the IP address or domain being used to collect users’ IDs and passwords is located outside of the US, but in Sunbelt’s write up, the IP address is right here in River City and belongs to an ISP headquartered in Dallas, Texas, Layered Technology.

The IP address in question is 216.32.94.147, and the whois information can be seen here and here.

Interestingly enough, a Google search for Layered Technologies, Inc. produces a number of links related to blacklists and spam.
The one domain residing on that IP can be seen at http://www.whois.sc/nikavonejalko.com and was registered with incomplete information to a entity in Russia. Let’s hope that Layered Technology acts responsibly and shuts down this site ASAP.

Xin giải thích tóm tắt về Hosts file Hijacking TROJẠN ( Trojan làm thay đổi HOSTs file trong hệ thống ). Khi môt ngừoi sử dụng Internet truy cập đến môt website của hacker thì anh ta có thể bị cài vào hệ thống (máy) của mình môt Trojạn nói trên . Trojan này sẽ thay đổi các địa chỉ IP nằm trong HOSTS file của máy nạn nhân. HOSTS file có tên là "hosts" nằm ở các đừong dẫn sau của từng OS:

Linux and other Unix-like operating systems: /etc
Windows 95/98/Me: %windir%\
Windows NT/2000/XP/Vista: %SystemRoot%\system32\drivers\etc\ is the default location, which may be changed. The actual directory is determined by the Registry key \HKLM\SYSTEM\CurrentControlSet\Services\Tcpip\Parameters\DataBasePath

HOSTS file có tác dụng giúp user truy cập nhanh , trực tiếp đến các website có domain quen thuộc với họ, thí dụ website của môt hay nhiều ngân hàng mà user hay truy cập đến. Bởi vì lúc đó trình duyệt sẽ mapping đia chỉ IP có trong Hosts file ứng với tên miền của website ngân hàng mà user muốn truy cập đến , không cần qua DNS severs.
Nhưng vì IP đã bị Trojạn thay đổi nên user sẽ bị dẫn đến môt trang web giả mạo khá giống với trang web của ngân hàng . Trang web này có IP giả nói trên , dĩ nhiên không phải IP đúng của ngân hàng . Thế là user - nạn nhân cứ việc kê khai username, pass và các thông tin quan trọng khác vào trang web giả của hacker. Trang web giả cũng có thề là môt trang giống như trang đăng nhập của HVA forum ...vân vân .
