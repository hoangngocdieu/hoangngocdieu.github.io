Còn File.DAT mà bác kia nói ..... thì cũng quên đi cho mau giàu...
File *.DAT là viết tắt của file DATA <= qui cái đuôi file như vậy là để sắp xếp loại File thôi
Còn nọi dung trong file *.DAT đó là tùy thuộc vào người viết, tùy thuộc vào cơ chế mã hóa của họ.
Chứ không có cái chương trình nào dịch ra cho bác đâu.

Trước hết bạn tự tìm hiểu DLL đó viết bằng ngôn ngữ nào, ngôn ngữ đó bạn có thành thạo không? Thành thạo ở mức sửa được code mà không có comment. Còn công cụ thì không hiếm đâu! Sửa code mà không có comment, chú thích đã khó. Dễ gì dịch được dll ! Có chăng là tham khảo một vài đoạn quan trọng thôi. Vì với thành thạo NNLT bạn có thể mô phỏng chức năng mà bạn cần ở DLL đó, viết DLL khác thay thế rồi.
