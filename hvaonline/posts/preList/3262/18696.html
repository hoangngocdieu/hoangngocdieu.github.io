==> Rồi, Ok.
Đầu tiên:
Start\Run\ gõ gpedit.msc ==> xuất hiện chương trình Group Policy. (cửa sổ chương trình được chia làm 2 phần)
Chọn User Configuration\System: (nằm ở phần bên trái của cửa sổ)
- Nhìn sang phải, thấy:
Prevent access to command prompt: nhấn đúp vào đó, sau đó chọn Enable. // Cái này để khoá command prompt.

Prevent access to registry editing tools: nhấn đúp vào đó, chọn Enable. // Cái này đề khoá Registry

Nhấp đúp vào dòng Ctrl+Alt+Del Options

Remove Task Manager: nhấn đúp vào đó, chọn Enable. // Cái này để không cho bật Task Manager lên.

Khoá không cho truy cập vào ổ cứng: Chọn
User Configuration\ System\ Windows Components\ Windows Exproler.
Nhìn sang phải, thấy:
Prevent access to drivers from My Computer, nhấp đúp vào đó, chọn Enable. Sau đó, nhấn vào tuỳ chọn Pick one of the following combinations ==>
restrict all drivers // Khoá toàn bộ các phân vùng
... // Tự đọc tiếp.

Không cho vào Internet Options:

Chọn User Configuration\ System\ Windows Components\ Internet Explorer
Nhìn sang phải, thấy một loạt các dòng disable the ... ==> chọn dòng từng cái một sau đó enable hết là được.

Không cho download: Bật IE lên

Chọn Internet Options\ Security\ Custom Level
tìm xuống dòng File download ==> chọn Disable.

Thế đã đầy đủ chi tiết chưa ? (Hy vọng bạn đọc và hiểu tiếng Anh để hiểu mấy dòng đó). Lưu ý, là sau mỗi lần chọn nhớ ấn OK nhé.

Registry Key:  HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer
Data Type: DWORD

Value Data: [0 = Disabled / 1 = Enabled]

Registry Key: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\policies\Explorer
Data Type: DWORD
Value Name: NoViewContextMenu
Value Data: [0 = Disabled / 1 = Enabled]

mình thấy hieubietlungtung nói có lý đó.
giả sử người khác được phép mở 1 tệp tin txt trên ổ của bạn, vậy thì họ ít ra là có thế save as được (cũng không khác copy mấy).
nói tổng quát hơn 1 chút là quá trình copy thực chất là đọc dữ liệu từ máy của bạn và ghi vào máy của người khác. cho nên những gì nó làm trên máy của bạn chỉ là đọc. vậy nếu cho đọc mà cấm copy thì không thể được.

còn cấm cut thì bạn khi share ổ chỉ cần không đánh dấu vào ô allow network user to change my files (dùng trong winxp) là được. chi tiết hơn thì bạn có thể chỉnh được permissions cho từng user với các quyền được thi hành với thư mục đó bằng cách (trong winxp):
- vào folder option\view và bỏ chọn lựa use simple file sharing (recommend) để có nhiều chi tiết hơn khi share.
- bây giờ khi share folder, bạn sẽ thấy có nút permissions, nhấp vào nút đó và bạn sẽ chọn từng user và set quyền cho nó.
- chi tiết hơn nữa thì bạn nhấn vào tab security, ở đó các quyền được liệt kê nhiều hơn, tha hồ cho bạn lựa chọn.
