Tui khẳng định lại : hoàn toàn ko có chuyện ko thể chạy đồng thời cả 2 web server, có hay chăng là ko cần thiết mà thôi. Muốn chạy cả 2 thì bạn set port khác nhau là ok, chẳng hạn IIS chạy port 80 còn Apache chạy 8080. Cách set port trên Apache thì vô file httpd.config, search ":80" sửa lại thành "xxxx" trong đó xxxxx là port mà bạn muốn chạy.
*PS: ko cần thiết phải vô services để tắt service Apache, tui thì khoái xài command hơn. Vào cmd, chạy httpd / Apache - k start | stop | restart | install | uninstall ... (nhớ add cái path đến thư mục chứa Apache vào biến PATH của Win).

1. Extract the httpcfg.exe utility from the support tools area on the Win2003 CD.
2. stop all IIS services: net stop http /y
3. have IIS listen only on the IP address I'd designated for IIS: httpcfg set iplisten -i 210.245.**.***
4. make sure: httpcfg query iplisten (the ip's listed are the only ip addresses that IIS will be listening on and no other)
5. restart IIS Services: net start w3svc
6. start Apache service
