Dùng hàm các api dưới đây có thể dấu được ứng dụng khác đang chạy (VD dấu game đang chơi miễn fi' ở phòng máy thư viện của trường ^^)

bạn paste đoạn code dưới đây vào VB rồi chạy nó sẽ dấu task manger (khi nó đang hiện lên) và cho xuất hiện nó( khi nó đã bị dấu)

Private Declare Function ShowWindow Lib "user32" (ByVal hwnd As Long, ByVal nCmdShow As Long) As Long
Private Declare Function FindWindow Lib "user32" Alias "FindWindowA" (ByVal lpClassName As String, ByVal lpWindowName As String) As Long
Const SW_HIDE = 0
Const SW_SHOWNORMAL = 1
Private handle_task, l As Long
Private Sub Command1_Click()
handle_task = FindWindow(vbNullString, "Windows Task Manager")
l = ShowWindow(handle_task, SW_HIDE)
End Sub

Private Sub Command2_Click()
l = ShowWindow(handle_task, SW_SHOWNORMAL)
End Sub


--> bạn chỉ cần sửa "Windows Task Manager" thành tên tiêu đề (nằm trên cùng cửa sổ) của chương trinh cần dấu
