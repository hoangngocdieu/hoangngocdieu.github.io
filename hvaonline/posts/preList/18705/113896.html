Web nhạc thì cũng giống như bất kỳ 1 web nào ( Tuy có vài điểm khác nhau, mà có lẽ ko cần quá chi tiết )

Nếu bạn là người bắt đầu LKL có 1 vài lời khuyên như sau:
1. Nếu bạn bắt đầu từ đầu:
Học qua html -> php or asp ->...
Thì bạn sẽ nản ngay, vì học mà ko có hành theo cái kiểu học về IT thì chẳng mấy là bù đầu vứt óc mà nản, bỏ cuộc.

Vì thế, theo mình bạn có thể học và hành như 1 chút ý kiến mình đưa ra để bạn tham khảo:
- Trước tiên bạn làm quen với định nghĩa Web tĩnh, với việc học về HTML ( Cái này có vẻ ko quá khó. Nếu bạn nhanh, có thể hòm hòm trong 2, 3 ngày. Nếu chậm, có thể là 1 tuần, hay 1 tháng. còn tùy khả năng tư duy và trí nhớ của bạn. Nhưng theo mình thì nó rất giống với việc bạn học Bảng Cửu Chương )
- Sau khi có căn bản về HTML -> Bạn thử làm cho mình 1 web nhạc đơn giản = html ( hoặc kiếm code trên mạng về web nhạc = html.) Còn nếu ko tìm được thì pm mình, mình sẽ giúp ( Vì nó cũng chẳng tốn tí time nào của mình cả )
- Sau khi đã quen về HTML . Bạn chỉ cần tìm hiểu đôi chút về Web động với việc ngâm cứu 2 anh chàng PHP và ASP ( ở đây thì "và" cũng được, mà "hoặc" cũng được, tuy nhiên và vẫn tốt hơn ^^)
- Đã hiểu được web động, hiểu 1 chút về cấu trúc của ASP hoặc PHP. Bạn bắt đầu đi kiếm cho mình 1 code bằng ASP hay PHP tùy vào việc bạn bắt đầu cái nào trước, hoặc cả 2 nếu bạn thích. Như vậy là bạn đã có thể đủ 1 chút tư duy căn bản về Code.
Còn nâng cao thì vô vàn lắm, cứ đi thì mới biết sẽ đi tiếp tới đâu, chứ cứ nhìn thì muôn ngả, biết đâu mà rẽ .

================
Về vấn đề Môi trường để chạy, để thực thi:
Theo mình, với tất cả các code và ngôn ngữ lập trình trên, bạn nên bắt đầu với định nghĩa về Localhost
Ở Localhost sẽ có:
+ IIS ( Sẵn có và chỉ việc extract trong Windows )
+ Apache ( Windows cũng có, mà Linux cũng có. Tuy nhiên ở Windows, thì bạn cần phải đi download và install ra)

Nếu bạn tiến tới Web động:
+ IIS -> Sẽ hỗ trợ tốt nếu bạn nghiên cứu về ASP và cơ sở dữ liệu ( CSDL ) là Access
+ Apache -> Sẽ hỗ trợ tốt cho bạn nếu bạn sử dụng PHP và CSDL là My SQL

Tuy nhiên, nếu tìm hiểu nhiều hơn, bạn sẽ có thể tự config cho mình được 1 hệ thống mà ở đó IIS có thể hỗ trợ cả ASP và PHP với 2 CSDL tương đương.

Lại nói về CSDL
-> Đó là điều mà Web động ko thể thiếu ( Nơi lưu trữ dữ liệu
-> Vậy cái gì quản lý dữ liệu?
-> Nó cũng là 1 phần mềm
-> Ở đây tớ tạm nói cho bạn dễ hiểu với 2 chương trình căn bản:
+ Access ( Là sản phẩm của Microsoft Office - Cậu cài trong đĩa Office ra full hoặc select thêm mục Access là nó có thêm cho bạn cái đó ngoài Word và Exel ),
+ My SQL ( Là cái mà khi bạn tìm hiểu về PHP tự khắc sẽ thấy người ta nói trong đó. Rồi bạn lên google.com gõ: học cách cấu hình và cài đặt PHP + My SQL -> là sẽ có vô số tài liệu).

Và có 1 lời khuyên: Nếu bạn sử dụng IIS + Access thì ko quá khó. Nhưng nếu bạn sử dụng Apache + My SQL thì bạn nên chọn gói phần mềm giống kiểu All-in-one: Appserver thì cứ cài ra là trong đó nó có tất ( Muốn tìm hiểu cái này: google.com -> Appserver + dowload -> Cài đặt Appserver

Sau khi đã quen với Code, với Localhost -> Là bạn đã ngon roài đó!

Việc tiếp theo chỉ còn là Public cái ý tưởng web của mình cho bàn dân thiên hạ:
Lúc đó bạn có thể học cách tự config 1 server nhỏ, nếu nhà bạn có 1 cái computer, 1 line ADSL và 1 chút kiến thức về "tự tạo server riêng" ( Phần trong ngoặc kép, cũng là phần bạn có thể tìm bằng google.com )

Nếu như cái đó quá khó với bạn, bạn có thể lựa chọn Free host hỗ trợ cho từng loại ngôn ngữ mà bạn muốn thể hiện.

Còn nếu bạn có điều kiện hơn nữa thì có thể tự đi mua cho mình 1 cái host hỗ trợ cho mục đích của mình.

Tuy nhiên cùng với 3 cách trên, bạn luôn cần cho mình 1 cái gọi là domain ( Tên miền) như cái mà bạn nói: nhac.vui.vn chẳng hạn!

Còn cái code mà nó dung là code media của Red Phoenix đấy, hoặc Chí Linh Net cũng được, bạn vào Freecode.vn hoặc manguon.com kiếm nó có cả tá: Ở mục Code > Media

Còn nếu bạn đủ khả năng để tự code thì tốt quá rồi, xin chúc mừng!!!
