Code:

copy *.dll C:\inetpub\wwwroot\bin
cd ..\GUI
copy *.ascx C:\inetpub\wwwroot\UserControls\AMG-137\A0560
pause


Bạn có thể tạo ra nó bằng cách vào notepad và soạn thảo bình thường sau đó Save as lại với extension là ".bat"

Theo như http://en.wikipedia.org/wiki/Batch_file:
In MS-DOS, OS/2 and Windows, a batch file is a text file containing a series of commands intended to be executed by the command interpreter. When a batch file is run, the shell program (usually COMMAND.COM or cmd.exe) reads the file and executes its commands, normally line-by-line. A batch file is analogous to a shell script in Unix-like operating systems.

DOS batch files have the filename extension .BAT. Batch files for other environments may have different extensions, e.g. .CMD in Windows NT and OS/2, or .BTM in 4DOS and related shells. 

Ngắn gọn, nó là một file văn bản bình thường, trong đó là các lệnh cần thực hiện theo lô.

Mình nói theo cách này nhé cho mọi người dễ hình dung:
File bat: Là một tập hợp lệnh Dos được gộp lại với nhau, và lưu trong file có đuôi .bat
Khi chạy file bat, thì những lệnh nào có trong file thì sẽ được thực thi.

Cái này rất hữu dụng khi bạn cần chạy một loạt lệnh liền một lúc, thay vì gõ từng lệnh một, thì chỉ cần chạy file bat là xong

Ví dụ nhé: mình có file test.bat như sau để mở file abc.txt trong thư mục C:\Work
Code:
cd Work
type abc.txt


sau đó chạy file. Ví dụ
Code:
C:\test.bat


Kết quả
Code:
C:\cd Work

C:\Work\type abc.txt


Các command dùng trong file bat chính là các lệnh Dos, không có gì cả. Tuy nhiên bạn có thể tra thêm các tham số dùng cho file bat trong phần help của Windows để biết thêm
Sống là để không chết chứ không phải để trở thành anh hùng
