Bà con xóa msiexec.exe nhẳm mục đích gì ? Bà con xóa nó để nó khỏi chạy khi khởi động ?
Sai lầm to, vì khi bà con thực thi một file setup có đuôi .msi, nó cần msiexec.exe để thực thi.
Chức năng WFP(windows file protection) bảo vệ các file hệ thống của windows. Người dùng xóa file hệ thống trong system32, windows tự khôi phục lại được bằng cách lấy lại file tuơng ứng từ
1. The cache folder (by default, %systemroot%\system32\dllcache).
2. The network install path, if the system was installed using network install.
3. The Windows CD-ROM, if the system was installed from CD-ROM.

Vụ này có liên quan tới malware hay không, bà con lấy biểu hiện gì để phán bừa. Chuyện windows hoạt động với mấy phần mềm cùng hảng với nó, bà con nên tự tìm hiểu. Bà con đừng quá sợ mà phán đại "VIRUS".
