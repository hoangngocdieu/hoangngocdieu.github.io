Trước hết hãy đảm bảo rằng máy bạn không có virus và bạn không sử dụng Warez hay các file cài đặt có nguồn không rõ ràng.

Bỏ đĩa DVD/CD cài đặt Windows vào ổ đọc tương ứng ở máy bạn.
Vào Start => Run => Gõ cmd rồi enter => Gõ rồi Enter:
Code:
sfc /verifyfile=%SystemRoot%\system32\SHELL32.dll

Nếu file SHELL32.dll bị hư và cần được phục hồi, bạn hãy để Windows phục hồi nó. Thực hiện theo các chỉ dẫn của Windows khi được yêu cầu.
Sau đó, tiếp tục vào cmd như trên, gõ rồi enter:
Code:
regsvr32 /i shell32.dll
