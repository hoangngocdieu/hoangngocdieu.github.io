Em có 1 thắc mắc như thế này : khi ta gửi 1 file , bất kể file gì cho người khác qua yahoo messenger thì người đó có thể nhận . Vậy cho em hỏi : yahoo nó đã làm gì để người kia có thể nhận được file . Em có 3 phướng hướng , bác nào giải thích giùm :

+ yahoo lấy IP của người gửi file và dùng lệnh FTP để " Get " file đó về máy ( chắc sai )

+ truyền file đó lên máy chủ của yahoo ( 1 ip khác ) và sau đó đưa file này đến máy người nhận (bước gián tiếp )

+ Up file này lên .... 1 web upload file nào đó và lấy link rồi dùng 1 lệnh gì đó để lấy file này về ( 10% đúng )

Ai am hiểu phân tích giùm em cái , phân tích cả hoạt động luôn .
