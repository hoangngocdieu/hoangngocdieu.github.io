Chào bạn,
Để máy A 192.168.1.10/24 thấy máy B:192.168.4.10/24 thì ta tiến hành cấu hình routing như sau:
1.Máy A add default route(gateway) 192.168.1.1
2.Máy B add default route(gateway) 192.168.4.1
3.Cấu hình routing dynamic RIP cho 3 router R1,R2,R3

a.Config Rip R1

R1>en
R1#conf t
R1(config)#router rip
R1(config-router)#net 192.168.1.0 255.255.255.0
R1(config-router)#net 192.168.2.0 255.255.255.0
R1(config)#do wr

b.Config Rip R2

R2>en
R2#conf t
R2(config)#router rip
R2(config-router)#net 192.168.2.0 255.255.255.0
R2(config-router)#net 192.168.3.0 255.255.255.0
R2(config)#do wr

c.Config Rip R3

R3>en
R3#conf t
R3(config)#router rip
R3(config-router)#net 192.168.3.0 255.255.255.0
R3(config-router)#net 192.168.4.0 255.255.255.0
R3(config)#do wr

Done

Thanks all
