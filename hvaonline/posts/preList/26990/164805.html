.NET là platform; còn Java là một ngôn ngữ lập trình.

Có thể ý bạn muốn hỏi: nên học .NET microsoft hay J2EE (được support bởi rất nhiều vendors).
Theo mình thấy thì đa số anh em SV VN có vẻ "khoái" .NET hơn, vì dễ học, gần với Winblowz, code vài dòng, rê chuột vài cái là "ăn" luôn, không phải nghĩ . Trong khi thực tế thì nhu cầu của các cty hiện nay cần người làm tốt trên J2EE nhiều hơn rất nhiều so với .NET.
Bạn thử google ".NET vs J2EE" sẽ thấy được bức tranh toàn cảnh về 2 thằng này .

Nhưng trước hết thì bạn hãy học C, sau đó là một ngôn ngữ hướng đối tượng (recommend Java, không nên chọn C++).

Tuỳ vào khả năng code java của em đang ở đâu, em viết một cái app ra sao thì sẽ có IDE phù hợp

Nếu như em mới học code: JCreator là một lựa chọn hợp lý.

Sau khi em cảm thấy mình code Java đã có thể coi là "cứng tay" thì chuyển sang Eclipse / Netbeans đều được.

Nếu như em phải code một ứng dụng, có một tẹo phức tạp + muốn cải thiện hiệu suất code thì dùng Eclipse / Netbeans hoặc một cái nào đấy.
