Cậu nên lên youtube trên đó có hướng dẫn cách cài đặt Linux và Window rất là kỹ lưỡng. Nếu muốn trở thành "vọc sĩ" đầu tiên cứ phải cài cái Virtual PC, Virtual Box hoặc VMWare....(1 trong 3 cái đó). Để có thể cài đặt và nghịch 1 cách thận trọng tránh tổn thất dữ liệu. Sau này muốn cài đặt trên máy thật thì cần biết 1 số kĩ năng sau.

1. Burn ISO vào CD/DVD-Disc.
2. BIOS Config.
3. Phân chia Partition, set active, MBR, MsDOS, Linux Command...
4. Xem qua hướng đẫn nếu không thấy tự tin.

Học lập trình thì cũng có các bước cơ bản, bạn nên theo, nếu bạn là thiên tài thì cứ bỏ qua.

1. Học về thuật toán và giải thuật.
2. Chọn một ngôn ngử dễ học để làm cơ sở, và tiện cho mô phỏng thuật toán, tôi nghĩ C là tốt.
3. Tìm sách và đọc sách nhiều và nghiêm túc.
4. Khi bạn hoàn tất 3 bước trên bạn sẽ có kỹ năng viết và debug code nếu khi nào cảm thấy tự tin và viết C tốt. Bạn có thể học Python một cách hiệu quả. Còn nếu bắt đầu ngay với Python bạn chẳng gặt hái gì nếu bạn không phải là superman.

Chọn học về thuật toán và giải thuật giúp bạn có hiểu biết căn bản, và cần thiết.
while(1){}
