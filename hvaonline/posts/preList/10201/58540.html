bạn vô coi lại Local Security Polices.
- Phần :User right Asignment
Xem policy Acess this computer from network xem có everyone trong đó ko,
nếu ko thì add vô
Xem policy Deny access this computer from network, xem có Guest trong đó ko, nếu có thì remove đi
-Phần :Security Option:
dòng Accounts: Guest account status ==> Enable
dòng tiếp theo: chọn Disable.

Tìm đến policy Network access:Sharing and security model for local account => Chọn Classic

Tiếp theo, nếu bạn dùng định dạng đĩa là NTFS thì nhớ set security cho cái thư mục muốn share là Everyone --> read thì mình nghĩ là được đấy.
