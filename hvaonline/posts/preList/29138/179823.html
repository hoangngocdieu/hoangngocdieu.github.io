Theo một số phản hồi từ phía người dùng, các bản Windows 7 RC “lậu” được tải về từ các trang web chia sẻ file trực tuyến có chứa mã độc ngay trong bộ setup.

Mặc dù đã có tin là Microsoft sẽ chính thức phát hành rộng rãi và miễn phí bản Windows 7 RC trên trang chủ của hãng từ ngày 5/5 nhưng không ít người đã không thể chờ đợi và tải về trước phiên bản hệ điều hành này từ nhiều địa chỉ chia sẻ file trực tuyến như BitTorrent hay Mininova.org. Điều đáng nói là những phiên bản này đã bị nhiễm mã độc và có thể gây hại không nhỏ cho người dùng.

"Xin cảnh báo với tất cả mọi người đang và đã tải bản RC rằng ngay trong file setup.exe đã bị nhiễm trojan", một thành viên tự nhận mình là Frank Fontaine viết trên diễn đàn Neowin.net, "Bởi nó được nhúng vào trong file Setup. EXE nên con Trojan này có khả năng tự bung ra và tự chạy. Nếu phân tích kỹ càng, file setup này bao gồm 2 file có đuôi exe là Setup.exe và codec.exe".

Một số phần mềm diệt virus đã nhận ra "codec.exe" chính là một chủng Trojan.

Một thành viên khác phát biểu trên diễn đàn Mininova cho biết thêm con trojan đó có tên là "Falder", có khả năng tự động tải về các phần mềm diệt virus giả để đánh lừa người dùng cũng như lén cài đặt một rootkit để lẩn tránh sự phát hiện của các phần mềm phòng chống virus.

Windows 7 RC không phải là phần mềm lậu đầu tiên có chứa mã độc. Hồi tháng 1/2009, phần mềm iWork '09 của hãng Apple được phát tán qua các địa chỉ chia sẻ file cũng đã bị phát hiện chứa Trojan có khả năng chiếm quyền điều khiển máy tính Mac.

Giới thiệu về chế độ XP Mode (Beta) trong Windows 7


Microsoft has added several new features to Windows Virtual PC to make it easy to use and to help you run many older Windows XP applications in Windows 7.

*Easy setup of Window XP Mode
*One click launch of Windows XP Mode applications
*USB support  


Windows XP Mode & Windows Virtual PC
Q: Windows XP Mode là gì?
A: Windows XP Mode trong Windows 7 hỗ trợ việt cài đặt và chạy các ứng dụng Windows XP dễ dàng hơn trên các máy tính sử dụng Windows 7. Tính năng này sử dụng công nghệ ảo hóa như Windows Virtual PC và cung cấp môi trường ảo Windows XP trên nền Windows 7. Windows XP Mode được cung cấp cho khách hàng dùng phiên bản Windows 7 Professional và Windows 7 Ultimate để dễ dàng sử dụng các ứng dụng cũ trên môi trường Windows XP từ máy tính Windows 7.

Q: Windows Virtual PC là gì?
A: Windows Virtual PC là công nghệ ảo hóa mới nhất dành cho Windows 7. Đây là hạt nhân để tạo ra Windows XP Mode và cung cấp môi trường ảo Windows XP trên máy chạy Windows 7. Với Windows Virtual PC và Windows XP Mode, các ứng dụng XP sẽ sử dụng được trên máy tính và máy trạm Windows 7.

Q: Windows Virtual PC mới khác biệt gì so với các phiên bản trước của VPC?
A: Windows Virtual PC được thiết kế đặc biệt cho doanh nghiệp nhỏ. Phiên bản này tổ hợp với vỏ hệ điều hành Windows 7 và thiết đặt Windows XP Mode. Đồng thời hỗ trợ cho các thiết bị USB và cho phép người dùng chạy các ứng dụng trực tiếp từ máy Windows 7 ở chế độ XP Mode chỉ bằng một nháy chuột.

Q: Windows Virtual PC cộng tác với MDOP như thế nào?
A: Windows Virtual PC cung cấp một cơ cấu ảo toàn thời gian cho giải pháp Microsoft Enterprise Desktop Virtualization (MED-V nằm trong bộ giải pháp MDOP). Tổ hợp trong Windows Virtual PC, MED-V được thiết kế đặc biệt để hỗ trợ các chuyên gia IT khả năng quản trị tập trung và triển khai môi trường Windows ảo hóa nhằm giảm đi độ phức tạp, vận hành tốt và giữ cho hệ thống ổn định ở mức phí tối thiểu.

Mình thấy Ảo hóa trên XP Mode chỉ có tác dụng thực thi các chương trình cũ không tương thích với Win7/Vista thôi. Cho nên, cần j nổi bật với những tính năng không cần thiết chứ.
Nếu dùng VMware để chạy thì phải cài ít nhất là VMWare Player vào, khởi động máy ảo lâu lắc, rồi mới dùng chương trình được. Còn XP Mode đơn giản hơn, chỉ việc cài và gọi chương trình từ ngay trình đơn của Win7. Tiện lợi hơn.

Còn về việc ngốn tài nguyên máy nữa. Làm gì có một cái Vmware XP chạy dưới 50MB RAM và CPU load thì ít nhất 5% ( dùng T7xxx). Còn XP mode thì khác, như mình đã trình bày trên, thông thường chỉ tốn khoảng 20MB RAM và CPU load rất nhỏ.

Bởi thế, mình thấy XP mode đã hoàn thành đúng nhiệm vụ của nó rồi. Không có gì phải phàn nàn nữa.
