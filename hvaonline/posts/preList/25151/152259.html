Máy A có Address là: 192.168.1.4
subnet mask: 255.255.255.0
Defaul Gateway: 192.168.1.1

Máy B có Address là: 192.168.1.6
subnet mask: 255.255.255.0

Sự việc là thế này:
Máy A share 1 thư mục nào đó, máy B dùng lệnh "\\192.168.1.5" hoặc "\\computer name" trong cửa sổ Run thì qua được máy A,còn làm ngược lại là Máy B share 1 thư mục nào đó Máy A cũng dùng lệnh "\\192.168.1.6" hoặc "\\computer name" trong cửa sổ Run thì không qua được Máy A.

Tớ nghỉ bác nên xem lại phần IP đi.......
Bác biết phân biệt IP private và IP Public chứ.
Theo tớ:
IP private là IP dùng trong LAN bao gồm những dảy địa chỉ sau:
- 192.168.0.0/16
- 172.16.0.0---->> 172.31.0.0 hay 172.16.0.0/12
- 10.255.255.255
IP Public là IP dùng cho WAN bao gồm những địa chỉ IP khác IP Private
IP 127.x.y.z là IP test TCP/IP, tức là những máy tính muốn trao đổi dữ liệu với thì phải cùng 1 giao thức.
Tớ chỉ có thể trình bày được như vậy thôi.

1. Có vẻ bạn vẫn chưa lĩnh hội được cách trình bày vấn đề. Lý do:
- Bạn cần cho biết định dạng phân vùng chứa thư mục chia sẻ
- Nhấp chuột phải vào thư mục cần chia sẻ, chọn Properties chụp screenshot của 2 tab Sharing và Security post lên đây
