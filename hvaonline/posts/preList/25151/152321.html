Bác biết phân biệt IP private và IP Public chứ.
Theo tớ:
IP private là IP dùng trong LAN bao gồm những dảy địa chỉ sau:
- 192.168.0.0/16
- 172.16.0.0---->> 172.31.0.0 hay 172.16.0.0/12
- 10.255.255.255
IP Public là IP dùng cho WAN bao gồm những địa chỉ IP khác IP Private
IP 127.x.y.z là IP test TCP/IP, tức là những máy tính muốn trao đổi dữ liệu với thì phải cùng 1 giao thức.
Tớ chỉ có thể trình bày được như vậy thôi.

Theo tôi nghĩ thi` máy tính mà được bạn ping chưa bật chức năng Classic - local users authenticate as Guest trong Local Pocily đó bạn.
Bạn thử vào Pocily:
+ Computer Configuration/ Windows Settings/ Security Settings/ Local Policies/ Security Options/ Network Access: Sharing and Security model for local accounts và chuyển sang chế độ Classic - local users authenticate as Guest.
+ Và Disable chức năng Accounts: Limits local use of blank passwords to console logon only( chức năng này cũng nằm trong địa chỉ trên).
Bạn thử xem sao. Good luck!
