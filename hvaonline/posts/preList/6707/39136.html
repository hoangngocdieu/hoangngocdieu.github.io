Lần sau nhớ sao lưu reg hoặc tạo điểm restore nhé bạn! Cách khả thi nhất là ra DOS sao lưu lại các dữ liệu wan trọng trong partition rùi install lại windows ^^.

Chạy msconfig: bạn có thể ấn tổ hợp phím 'window + r' để gọi run rùi gõ msconfig; regedit để gọi registry; services.msc để gọi các dịch vụ của windows ... hoặc gõ trong windows command cũng dc -> trong run gõ cmd để gọi windows command. (Bạn có thể dùng phần mềm thứ 3 thay thế windows command, regedit, msconfig... )

Vào trong msconfig/services bạn nên mở lại các dịch vụ wan trọng như:
Application management; Even log; IMAPI CD - Burning COM service; Network Connections; Plug and Play; Print Spooler; Remote Access Connection manager; Remote procedure Call (RRC); Shell Hardware detection; Telephony; Themes; Universal Plug and Play Device Host; Windows audio; Windows firewall/Internet Connection Sharing (ICS); Windows Installer; Widows management Instrumentation; Windows Times. 


Từ " http://windowsxp.mvps.org/exefile.htm ":
Nguyên nhân: phần mở rộng .exe trong reg bị gián đoạn. Gây ra bởi virus như: "SirCam" ...
Cách sửa thủ công:
+ chạy cmd (windows command)
+ gõ:
Code:
reg add hkcr\exefile\shell\open\command /v default /t reg_sz /d "%1" %* /f
reg add hkcr\.exe /v default /t reg_sz /d exefile /f

+ Nếu ko gọi red dc thì chạy dòng lệnh:
Code:
copy regedit.exe regedit.com -> 
regedit.com

(dòng này có nghĩa là đổi tên regedit.exe thành regedit.com rùi chạy nó). 

(Bl ko dùng regedit như trong hướng dẫn, mà dùng cmd lun cho tiện ^^)

** Ngoài ra bạn cũng có thể sửa việc ko đọc dc file .exe bằng cách sau mà ko cần phải can thiệp gián tiếp wa registry: từ dòng lệnh windows command bạn gõ
+ Code:
assoc .exe
:để xem phần liên kết các phần mở rộng của file.exe, kiểm tra nếu nó ko phải là 'exefile' thì sửa lại bằng dòng lệnh: Code:
assoc .exe=exefile

+ Code:
ftype exefile
:xem ánh xạ loại file chỉ định thực thi cho các liên kết, kiểm tra nếu nó ko phải là "%1" %* thì sửa lại bằng dòng lệnh: Code:
ftype exefile="%1" %*


Ok ... cho trường hợp các file.exe;

Máy tự đông shutdown lại: dự đoán có thể shell windows gặp sự cố; bạn tìm đến khóa hkey_local_machine\software\microsoft\windowsNT\CurrentVersion\Winlogon
chỉnh lại khóa AutoRestartShell có giá trị là 0
Code:
reg add hklm\software\microsoft\windowsNT\CurrentVersion\Winlogon /v AutoRestartShell /t reg_dword /d 0

Chúc bạn may mắn

Than.
