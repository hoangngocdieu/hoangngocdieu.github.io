Là file template (mẫu) Khi bạn khởi động WinWord lên, Winword sẽ mở file đó, nếu không tìm thấy thì nó mới tạo ra file mới. Bạn chỉ khởi động máy, không mở Winword thì làm sao nó tự tạo được?
Bạn bị với tất cả các file hay chỉ một file thôi. Với tất cả các file thì có khả năng Windows bị lỗi (Bạn có thể mở file .doc bằng chương trình Wordpad - trong mục accessories), nếu chỉ có 1 file --> em đó đã ra đi. Bạn có thể phục rồi một phần bằng cách mở Winword -> File -> Open, ở mục File type chọn "Recover text from any file" để mở file đó xem có phục hồi được phần nào không.
Nhớ là sau khi dùng Winword -> File -> Open, ở mục File type chọn "Recover text from any file", bạn lại phải chọn file type là ".doc" nhé chứ không tất cả các file còn tốt mở ra đều thấy loằng ngoằng cả đấy (cách khác lại xóa file normal.dot đi).
