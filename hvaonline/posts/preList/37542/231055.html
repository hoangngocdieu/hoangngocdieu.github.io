- Về touchpad, có thể bạn cần cấu hình thêm cho nó trong xorg.conf nếu HAL không tự nhận ra (không rõ Ubuntu thế nào). Ví dụ của mình:
Code:
Section "InputDevice"
	Identifier 	"TouchPad"
    Driver		"synaptics"
	Option      "Protocol" "auto-dev"
	Option		"SHMConfig" "true"
	Option		"Device" "/dev/input/event5"
	Option		"TapButton1" "1"
	Option		"TapButton2" "2"
	Option		"TapButton3" "3"
EndSection

Thử tìm hiểu xem của bạn là /dev/input/event mấy nhé.

- Về Fn, bạn cài và chạy thử xev, rồi nhấn tổ hợp Fn + tăng, giảm volume chẳng hạn xem nó bắn ra gì không:
Code:

    root 0xb9, subw 0x0, time 1656218, (529,355), root:(533,431),
    state 0x0, keycode 122 (keysym 0x1008ff11, XF86AudioLowerVolume), same_screen YES,
    XLookupString gives 0 bytes: 
    XFilterEvent returns: False
