Total Commander

Ưu điểm:

- Di chuyển nhanh giữa các thư mục với bàn phím.

- Di chuyển nhanh trong thư mục với Alt+các chữ cái đầu tên file.

- Copy nhanh vì thư mục nguồn và đích luôn luôn cùng hiển thị trên 1 màn hình.

- Đánh dấu lựa chọn file nhanh với Insert, +, -, *

- Copy file chỉ bằng 1 cú gõ phím (F5).

- Vào các file nén .zip, .rar,... như vào thư mục => giải nén cũng như copy. Thê là khỏi cần Winzip, Winrar,...

- Vào FTP server cũng tương tự như vào thư mục thường.

- Xem, tạo, sửa, xóa, đổi tên, dời file và thư mục,... cũng rất nhanh chóng chỉ nhờ 1 cú gõ phím. Đặc biệt ở phiên bản 7 việc xóa, tạo, đổi tên, dời file trong các thư mục cấm cũng rất dễ dàng nhờ lệnh As Administrator.


Nhược điểm:

- Chậm khi download/upload với server xa (vì dữ liệu không được nén trên đường truyền, vì không dùng các kĩ thuật cao kiểu remote increment/delta computation).

- Rất chậm khi copy từ server này sang server khác (cũng vì những lý do trên và vì dữ liệu không chuyển thẳng giữa 2 server mà phải download rồi upload).

- Tìm file chậm (vì không có index mà phải quét từng thư mục/file), nhất là tìm theo nội dung thì cực chậm. Để tìm được vài văn bản theo từ khóa giữa 10 000 000 văn bản thì gần như bó tay.


Nhìn chung, tôi cũng thấy để quản lý file trên một máy đơn lẻ và mạng LAN dùng Total Commander là tiện hơn cả. Mặt khác, trên môi trường mạng, nhất là mạng WAN thì Total Commander còn cần phải cố gắng nhiều.
