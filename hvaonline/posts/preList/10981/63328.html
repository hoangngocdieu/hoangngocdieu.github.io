ngochung_1287 wrote:

còn bác zeros bảo là ( tạo kết nối trên OS )có nghĩa là sao vậy ? mong các bác chỉ giúp nha 

Có nghĩa là tạo kết nối trong hệ điều hành của bạn.
Ví dụ với kết nối quay số Dial Up như bạn yêu cầu
Start / Control Panel / Network Connection / Chọn Create a new connection / ô checkbox chọn Connect to the Internet / Setup my connecttion manually / Connect using a dial up modem / ô ISP name chọn tên nhà cung cấp dịch vụ Internet (Ở đây gõ VNN cũng được) / Phone number gõ số để kết nối (ví dụ là vnn1269) / User name, Pass, Confirm...gõ vnn1269 / đánh dấu tích vào ô Add a shortcut... /Finish
Sau khi làm xong các bước trên thì trên Desktop của bạn sẽ có 1 biểu tượng để kết nối mạng, bạn chạy nó là xong.
PS : nếu không nhầm thì để sử dụng Dial Up bạn phải đăng ký với bưu điện thì mới được, lâu rồi không xài thằng này nên quên ) ) )
