Nhắc nhở bạn nvhck là nếu bạn muốn người khác đọc bài và tham gia giúp đỡ bạn, điều đầu tiên là phải viết tiếng Việt có dấu rõ ràng.

Đầu tiên bạn cần phân biệt rõ, command line interface (giao diện dòng lệnh) không phải là DOS (Disk Operating System). Các hệ điều hành như windows, Mac, linux, vân vân đều có hỗ trợ command line interface. Đó chỉ là một cách khác để giao tiếp và làm việc với hệ thống, thay vì có GUI, chuột, vân vân.

Vấn đề của bạn là do run level của FedoraCore4 mặc định sẽ là level 5, nghĩa là đã có X windows, xdm, vân vân. Bạn dùng ctl-alt-f1 đã có thể log in vào môi trường command line interface. Lý do có chuột là vì: command line interface hoàn toàn có hỗ trợ chuột. Không phải chỉ có GUI mới có chuột :~)

Nếu bạn muốn hệ thống của bạn khi khởi động không dùng X windows mà chỉ có CLI, bạn có thể tham khảo thêm manual về inittab bằng lệnh
Code:
$ man inittab
và tiến hành chỉnh sửa file /etc/inittab cho thích hợp với nhu cầu của bạn.
