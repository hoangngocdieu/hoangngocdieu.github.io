Những lỗi tạo core dump thường do cài không đúng bản (ví dụ máy chạy 32-bit lại cài bản 64-bit) hoặc các thư viện cấp thấp của hệ thống không thoả mãn đòi hỏi của ứng dụng muốn chạy.

Nếu ứng dụng tạo core file thì nên tìm cách phân tích core file để biết chuyện gì xảy ra. Ứng dụng trên chạy trên Java nên chắc chắn phải có java exceptions được log ở đâu đó. Nên đọc kỹ tài liệu sử dụng để nằm rõ những đòi hỏi cài đặt và nơi logs được lưu để tìm thêm manh mối.
