Để web app chạy nhanh thì rất nhiều vấn đề.

- Resource Server
- Config, optimize: Web server, Sql server
- Code chuẩn, các câu truy vấn SQL ( nếu có ) gọn gàng, logic, không tốn resource nhiều
- Hạn chế các file JS, cái nảo cần xài thì nhúng vào.
- Xóa những tag, tab dư...
- Dùng các midleware để tăng tốc
- PHP dùng thêm output ob_gzhandle, IIS thì dùng phần mềm ở Port80software.com
....

Nói chung là rất nhiều, sơ sơ vài cái như vậy.
Cái này phụ thuộc nhiều vào kinh nghiệm hơn.
