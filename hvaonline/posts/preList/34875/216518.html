Ví dụ như link download này: http://www.mediafire.com/?0mzhnmzornd Khi viewsource sẽ có một đoạn Javascript đã bị mã hoá.

Cơ chế chung thì sẽ là thằng mediafire nó đẩy về browser một đoạn javascript, đoạn javascript này đợt một vài giây sẽ tính toán cộng trừ gì đó ra link đích, rồi thao tác với DOM để set cái link thật này ra cho user click vào. Cái rhino kia là một javascript engine cho Java, em dùng nó để mô phỏng quá trình thực thi javascript trong web browser, sau đó chỉ việc đón kết quả trả ra sau khi thực thi javascript là được.
