em cũng có 1 chút vấn đề với Ubuntu 8.10, post lên đây để các bác tham khảo :

Cả desktop và laptop đều chạy OS chính là Xp 2 :

1, File Ubuntu 8.10 ISO down từ trên net và burned ra CD .
2, CD chạy OK trên desktop chề độ "Thử Ubuntu mà không làm ảnh hưởng tới máy" ngon lành, ko thử với install
3, Đĩa CD Ubuntu ko chạy trực tiếp trên laptop với cả 2 chế độ như bạn xamxixixo mô tả ( máy treo cứng - no respond )
4, Cài thành công Ubuntu với file ISO hoặc CD thông qua VMware 6.0 trên laptop : display, sound, Ethernet , USB đều OK.
5, Cài tiếp Red Hat Enterprise Linux 5.2 qua VMware for Windows thành công
6, Thử lại Ubuntu thấy mất Ethernet, cả RHEL cũng vậy, trong khi đó XP2 vào internet bình thường.
7, Xóa và cài lại Ubuntu vẫn tình trạng trên

Bác nào đã thử cài multi-OS trên windows gặp tình trạng trên chưa ạ ?
