có thể bồ đã làm thiếu bước biên dịch.

Sau khi chạy "configure" thì bước tiếp theo là biên dịch chương trình - compile - dùng lệnh sau:

Code:
$ make


Sau đó mới đến bước cài đặt - make install - chạy với quyền root

Code:
$ su
# make install

Oài, thế ra bồ chưa compile source bao giờ à. Như anh conmale nói, đọc kĩ file INSTALL và README để biết phải làm thế nào, cần các dependencies gì để có thể compile thành công.

Về cơ bản, file configure được dùng để tạo Makefile. Sau đó chạy make để compile, và cuối cùng chạy make install để copy files vào những nơi cần thiết, ví dụ usr/bin hay /lib chẳng hạn.
