@phanledaivuong: [ quote ] đâu có giữ được format đâu em. Ví dụ anh muốn đoạn text sau giữ được căn chỉnh và đồng thời bôi đậm dòng thứ 5 là bó tay đấy:
Code:
1	if [ -n "$BASH_VERSION" -o -n "$KSH_VERSION" -o -n "$ZSH_VERSION" ]; then
     2	  [ -x /usr/bin/id ] || return
     3	  tmpid=$(/usr/bin/id -u)
     4	  [ "$tmpid" = "" ] && tmpid=0
     5	  [ $tmpid -le 100 ] && return
     6	  # for bash and zsh, only if no alias is already set
     7	  alias vi >/dev/null 2>&1 || alias vi=vim
     8	fi

Đưa vào [ quote ] thì tất cả mọi dòng sẽ thẳng hàng hết. OK?

Gần 4 năm rồi, đã đến lúc phải sửa rồi anh conmale ơi, không thì copy paste Python code sẽ báo lỗi Indentation ngay: http://www.hvaonline.net/hvaonline/posts/list/41599.html#259567

Em nói thêm cho rõ: mặc định khi bôi đen code và chọn [ code ] tag:

[ code ]dòng1
dòng 2
dòng 3[ /code ]

Lúc này, dòng 1 sẽ bị thò ra, từ dòng 2 trở đi sẽ có một khoảng trắng ở đầu:

Code:
dòng1
dòng 2
dòng 3


Nếu cho dòng 1 xuống dòng như này:

[ code ]
dòng1
dòng 2
dòng 3[ /code ]

thì dòng 1 không còn thò ra nữa, nhưng giờ tất cả sẽ có một khoảng trắng ở đầu dòng, bôi đen sẽ nhìn thấy, copy paste sẽ mang theo:

Code:
dòng 1
dòng 2
dòng 3
