Đây là 1 code Blog bằng PHP, bạn có thể tìm thấy ở Freecodevn.com hoặc manguon.com
Tuy nhiên muốn tự mình chỉnh sửa, cài đặt và quản lý bạn nên tìm hiểu những điều căn bản như sau:

1. Hiểu biết 1 chút về HTML để có thể edit code theo ý mình, theo info về mình

2. Hiểu 1 chút về PHP và CSDL My SQL

3. Hiểu thế nào là domain, thế nào là Host và làm gì để upload dữ liệu cũng như quản lý được Host

4. Học cách cài đặt 1 gói Open Source như Forum, blog...

5. Học cách quản lý, đăng tải thông tin...

Chúc bạn sớm thành công!!!

Tuy nhiên, khi bạn down được source thì trong phân mục cho source bạn cũng có thể tìm thấy hướng dẫn cài đặt.
Nếu thiếu điều gì đó, bạn nên hỏi thăm bác Google.Com thông minh!!!
