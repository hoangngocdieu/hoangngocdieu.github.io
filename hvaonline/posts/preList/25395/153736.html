Bạn thử dùng lệnh attrib của cmd để list các file trong USB.
attrib -h -r -s M:\*.* (thay chữ M bằng tên ổ USB của bạn)

Vào explorer ---> tools ---> Folder Options ---> View , bỏ mục Hide extensions for know file types và chọn show hidden files and folders.

VD bạn có 2 thư mục đặt tên là : HoSo và LuuTru
Thì bạn sẽ thấy 2 thư mục đó mờ đi (do bị ẩn) và bạn sẽ thấy 2 file tên là HoSo .scr và LuuTru .scr có hình giống cái folder. Ngoài ra còn autorun.inf và các file khác nữa. Dùng BitDefender hoặc KAS/KIS chọn những file đó rồi quét là xong. Không biết mình đoán thế có đúng không?
