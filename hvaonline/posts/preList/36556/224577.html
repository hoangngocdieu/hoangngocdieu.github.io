Mình vừa mới Download xong bộ cài Windows Server 2003 Enterprise SP2 từ trang của Microsoft về, nhưng oái oăm thay nó lại bắt phải Burn ra 2 CD: CD1=595Mb - CD2=121Mb cả 2 lại có định dạng là ".IMG".
Bây giờ mình muốn ghép hai CD làm 1 để Burn ra 1 DVD hay thành 1 file ".ISO" dùng để cài Win Server trên VMware Workstation.

- Nhấp OK.
- Chuột phải vào từng file *.img chọn mount như một ổ ảo
