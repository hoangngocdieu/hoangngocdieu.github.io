- trong thư mục hiện tại `/home/dungvip96` không có file nào như thế. Gõ `ls` kiểm tra lại xem.
- file đang cài là .tar.gz chứ không phải .tar.bz2 (muốn extract bạn có thể dùng `tar zxvf ...`)
- mình dùng `ubuntu-tweak` thấy hay bị crash nên mới chuyển sang dùng `gnome-tweak-tool` dùng thấy ổn. Nếu có kết nối Internet thì cài như này:
Code:
sudo apt-get install gnome-tweak-tool
