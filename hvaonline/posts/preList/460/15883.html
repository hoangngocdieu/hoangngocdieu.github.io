Đối với Windows 98 thì việc shutdown windows nằm trong thư viện user32.dll. Bạn chỉ cần tạo một shotcut với link như sau: "c:\windows\system32\user32.dll,exit window" là ok. Còn trên windows 2000/XP trở nên thì nhiệm vụ shutdown windows được đặt trong một chương trình là shutdown.exe trong system32. Bạn có thể vào command Prompt mà gõ "shutdown /?" để biết cụ thể tham số của nó.
Ví dụ bạn muốn máy tính shutdown trong 2 phút nữa: vào command gõ: shutdown -s -t 120
