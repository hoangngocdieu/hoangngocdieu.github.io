à mình chỉ biết thế nầy thôi,yahoo co chức năng luu mật khâu,mình coi 1 tut nhu sau bạn tham khảo nha.

Mà họ hay sử dụng chức năng remember pass yahoo. thì bạn hãy làm các bước sau để có thể login hoặc lấy pass của họ...
bước 1: lén vào run --> regedit --> HKEY_CURRENT_USER --> software --> yahoo --> pager. Bạn chú ý tìm tới khóa: ETS và Yahoo! User ID ( xem hình dưới...)




Bước 2:
Bạn hãy click double vào 2 khóa đó để lấy value của nó. Theo như trong hình của mình ta có 2 value của 2 khóa trên là: ETS:THZwEAADE23q_xvjSyNENz5Tfvv8Hy9NnFjZMUxy80ZLsa sGSGM5SZ2OkVbatkZoQScJ2uu__oeiZk__QRNB2uZ_EOCWMXef z9dw-
và Yahoo! User ID: ds.pro1990( xem hình dưới )


Sau khi đã lưu 2 value đó rồi, ta lén lưu vào 1 file .txt rồi đem về nhà.
Bước 3:
Đem về nhà rồi ta vào run --> regedit --> HKEY_CURRENT_USER --> software --> yahoo --> pager (bên máy ta) tạo 2 khóa ETS và Yahoo! User ID ( bằng cách click chuột phải new-->string value rồi điền name lần lượt là ETS và Yahoo! User ID (xem hình dưới).




Sau đó double chuột trái vào 2 khóa mới tạo rồi điền lần lượt 2 value của 2 khóa mà mình đã lấy bên máy tính bạn mình rồi đóng regedit lại.
Sau đó bạn cứ việc bật yahoo lên và thấy kết quả thế này là ok.

Bước 4:
Giờ bạn click vào để login hoặc có thể sử dụng các chương trình view pass (xpass v2.2 chẳng hạn để view pass là gì, chỉ view được khi phiên bản yahoo của bạn dưới 9.0 nha).
