Dear các Pác!

Em đang học làm web bằng JAVA (jsp/servlet-Struts) các pác ạ! khi làm thì các vấn đề xử lý dữ liệu không dấu với CSDL(SQLserver2000 hoặc MySql) thì không sao nhưng khi đụng chạm đến dữ liệu Tiếng Việt thì toi các pác ạ! Cụ thể như sau:

- Trang jsp (vd: addUser.jsp) để nhập thông tin về 1User bằng Tiếng Việt để ghi vào DB thì việc ghi OK, nhưng dữ liệu vào trong DB thì loằng ngoằng không phải là Tiếng Việt có dấu như đã nhập.

- Trang jsp (vd: listUser.jsp) để hiện thị các User thì hiện thị cũng loằng ngoằng với các Users đã nhập bằng Tiếng Việt trước đó mà không hiển thị nổi Tiếng Việt lên. (nếu nhập bằng TV ko dấu thì hiện thị ko vấn đề).

->Mặc dù em đã:
- request.setCharacterEncoding("UTF-8"); response.setCharacterEncoding("UTF-8"); (trong Servlet)
- trong các trang jsp cũng set về UTF-8.
- Cac cau insert cũng đặt N trước các dữ liệu kiểu varchar
Mà vẫn không được các pác ạ! oằn quá...

Em rất mong các Pác giúp đỡ em để làm sao khi Submit thì ghi được dữ liệu Tiếng Việt vào DB và khi đọc dữ liệu từ DB cũng hiện thị được đúng Tiếng Việt lên các trang JSP.

Chân thành cảm ơn, và đa tạ...

Sau khi cài đặt xog bạn cần phải đổi lại chuỗi liên kết CSDL như sau:
static final String DBDriver = "com.microsoft.jdbc.sqlserver.SQLServerDriver";
static final String strConn = "jdbc:microsoft:sqlserver://localhost:1433;DatabaseName=ten_CSDL";
static final String DBusername = "sa";
static final String DBpassword = "";
(Nhớ copy 3 file:msbase.jar,mssqlserver.jar,msutil.jar trong thư mục C:\Program Files\Microsoft SQL Server 2000 Driver for JDBC\lib vào thư mục \WEB_INF\LIB trong ứng dụng của bạn nhé)
