Khi sử dụng công cụ Encrypting File System (EFS), bạn nên sao lưu các khóa giải mã cần thiết để đề phòng trường hợp hệ thống gặp sự cố vì việc khôi phục dữ liệu khi mất khóa giải mã là rất mong manh.

Bạn hãy tham khảo bài viết tại địa chỉ
Code:

http://www.quantrimang.com/view.asp?Cat_ID=4&news_id=13402

(tiếng Việt)
hay
Code:

http://www.microsoft.com/technet/prodtechnol/winxppro/deploy/cryptfs.mspx

(tiếng Anh)
để biết cách sử dụng EFS.

Ngoài ra, bạn có thể sử dụng phần mềm Advanced EFS Data Recovery
Code:

http://www.elcomsoft.com/aefsdr.html

để phục hồi (decrypt) các file đã bị mã hóa bằng EFS trong Windows 2000/XP/Server 2003.
