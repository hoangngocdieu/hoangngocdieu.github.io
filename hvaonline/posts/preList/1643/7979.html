Như bạn silver nói đó, bạn ấy không thể kết nối được tới server game khi bạn ấy sử dụng proxy (Hình như cái VLTK cũng vậy)
Mình thử dùng proxy server: map tcp port **** đến ip của server game. sau đó mình định route toàn bộ đến địa chỉ của server game qua 1 proxy nào đó. vd:
route add a.b.c.d>* mask **.**.**.** A.B.C.D (Trong đó a.b.c.d nhỏ là địa chỉ ip của server game).
Nhưng khi chạy thì chẳng hiểu thế nào mà ko được, bạn có ý tưởng nào hay hơn không ?
