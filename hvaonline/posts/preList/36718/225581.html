`yum` có option `whatprovides` đấy:
Code:
# yum whatprovides *bin/dot
...
graphviz-2.12-8.el5.x86_64 : Graph Visualization Tools
Repo        : epel
Matched from:
Filename    : /usr/bin/dot


Nếu ntop thích tìm trong `/usr/local/bin` thì bạn có thể compile hoặc chỉnh lại `--prefix`.
