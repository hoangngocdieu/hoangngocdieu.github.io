Em có 2 div a và b.Giờ em muốn cái div b phải nằm trong cái div a, code :

<div id="a">
<div id="b">
</div>
</div>

Mình vừa thử xong với FF và IE với đoạn code sau:
Code:
<html>
<head>
	<style type="text/css">
		#a {
			color: red;
		}
		#b {
			color: blue;
		}
	</style>
<body>
	<div id="a">
		div a
		<div id="b">
			div b
		</div>
	</div>
</body>
</html>

và thấy không có vấn đề gì khi hiển thị trên FF cả, đây là ảnh chụp khi view source bằng FF:
