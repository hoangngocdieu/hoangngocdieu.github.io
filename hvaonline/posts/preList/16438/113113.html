Chào mọi người.

Mình thiết kế web theo CSS, đang gặp phải 1 vấn đề nhỏ thế này: làm thế nào để đặt nội dung của 1 thẻ <div></div> nằm căn giữa màn hình (chỉ theo chiều ngang giống <center></center>, ko có chiều dọc), yêu cầu là bên trong thẻ <div></div> đó có thể có bất cứ thứ gì, kể cả table, header, link, list, ...

Trước mình thiết kế web có 3 cột thì dùng thuộc tính float và margin kiểu như sau để cho cột thứ 2 nằm chính giữa
Code:
#cot1, #cot2, #cot3 {
float: left;
position: relative;
}
#cot1, #cot3 {
width: 100px;
}
#cot2 {
width: 400px;
margin: 0 auto 0 auto;
}

Đối với 3 cột thì nó chạy đúng, nhưng khi mình thử với web chỉ có 1 cột thì ở trình duyệt IE bị lỗi, còn Firefox thì được.

Rất mong mọi người giúp đỡ, xin cám ơn rất nhiều.

Mình mới tìm được 1 cách, áp dụng với thẻ div đã biết sẵn độ rộng width, ai có cách khác xin chỉ giúp:

Code:
width: 400px;
left: 50%; // đặt mặc định lề trái tính từ giữa màn hình
margin-left: -200px; // giá trị âm bằng 1/2 độ rộng, do lề tính từ chính giữa màn hình nên đối tượng sẽ được căn vào giữa
position: absolute;

Ko được ngonchan à, tớ thử đoạn code sau thì thấy bị lệch:
Code:
width: 400px;
left: 50%; // đặt mặc định lề trái tính từ giữa màn hình
margin: 0 auto;
position: absolute;

Đoạn mã margin: 0 auto chỉ là căn lề trái là 0, lề phải tự động, khi đó nếu đặt thuộc tính left: 50% thì lề sẽ được tính từ giữa, và toàn bộ nội dung của thẻ div bị nằm ở nửa bên phải; còn nếu ko có thuộc tính left: 50% thì toàn bộ nội dung nằm lệch về nửa bên trái.

Thực ra lúc nói thuộc tính text-align: center có vai trò là do tớ thử với table mà không để width=100% nên có cảm giác bị lệch, chứ thực ra không cần, chỉ cần 2 thuộc tính width, và margin: 0 auto là đủ rồi.
