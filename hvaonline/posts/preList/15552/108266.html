Các bạn Ping hay Traceroute ( Traceroute= Ping+Ping+Ping+Ping...) mà sử dụng tiện ích (service) intergrated trong Microsoft Windows, nghĩa là sử dụng các gói tin dạng "ICMP usíng Windows DLL" thì làm sao mà reach đến mục tiêu ( là HVA server) đựoc. Vì server này, hay kể cả các device ( router, server ...) trên route có thể đã khóa các cổng ICMP. Thử dùng các gói tin UDP ( UDP packet ) hay các gói tin TCP ( TCP packet) là Ping hay Traceroute đựoc đến tận nơi ngay mà.

Tất cả các tiện ích hay service có trong các Windows OS ( thí dụ Ping, Tracert, Disk Manager, RAID [ soft.]... vân vân), thì đều có các tiện ích tương tự bên ngoài. Chỉ có điều là các tiện ích bên ngoài, của các tác giả khác, thì hay, hiệu quả, đa năng, chuyên nghiệp hơn các tiện ích tích hợp ( intergrated) trong Windows rất, rất nhiều. Phải "chơi" những cái này mới "ra vấn đề được". Chú ý là HVA server luôn luôn chạy, không có khi nào nghỉ cả. Mà hạn hữu có nghỉ, thí dụ để config. chẳng hạn, thì lão conmale sẽ thông báo ngay.
