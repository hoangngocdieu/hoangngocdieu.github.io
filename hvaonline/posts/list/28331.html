mình xài mạng ở trường, phải thiết đặt proxy như vậy. Trong lúc lướt web thì con ubuntu nó đòi upgrade nên mình upgrade = mạng ở trường.
còn mạng ở nhà thì giờ bị khóa sạch, cả passwd wifi bọn nó cũng giấu, bọn English nó điên điên sao ấy .

mình đã google, sau khi xài lệnh Code:
sudo gedit /etc/apt/apt.conf

nó ra kết quả, tạm thời mình để # để ngưng kích hoạt nó
Code:
Acquire::http::proxy "http://10.10.0.50:8080/";
Acquire::ftp::proxy "ftp://10.10.0.50:8080/";
Acquire::https::proxy "https://10.10.0.50:8080/";
rồi thêm đoạn này vàoCode:
Acquire::http:roxy "false";
save lại...

nhưng nó vẫn cứ acquire cái proxy cũ

tự nhiên giờ gõ lệnh nó lại raCode:
tuanhung@kitten-administrator:~$ sudo apt-get update
E: Syntax error /etc/apt/apt.conf:1: Unsupported directive ‘Acquire::http::proxy’

còn nếu xóa sạch thì nó acquire cái proxy cũ là 10.10.0.50
