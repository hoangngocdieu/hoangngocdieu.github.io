Bypass = hiểu sơ sơ là vượt qua. Chứ pass là gì thế vậy bạn.
:top
đặt móc cho goto

set command=rem (tham khảo http://ss64.com/nt/set.html)

Đặt cho cái command là rem, dùng để hiện lên màn hình dòng Command: cho bạn thấy bạn cần chạy cái lệnh (file .exe nào đó
%chdir% hiển thị tên thư mục hiện hành

set /p command=Command: (tham khảo http://ss64.com/nt/set.html )
%command%
echo.
echo.
echo.
goto top (lặp lại, trừ khi gỏ lệnh exit) để khỏi phải thoát hẳng cái màn hình đen thui cmd
Cái này là trick windows lặp đi lặp lại cái dòng lệnh bên trên, đòng thời chạy cái %command% để người ta có thể nhập vào cái gì đó tại dấu nhắc.

PS: tiêu đề cũng tiết kiệm ghê, bypass the cmd.exe restriction
Thử cấm xài cmd như cái dòng thông báo trên kia
"The command prompt has been disabled by your administrator"

windows programs. Add đúng cmd.exe rồi thử với cái file đó sẽ thấy liền.
