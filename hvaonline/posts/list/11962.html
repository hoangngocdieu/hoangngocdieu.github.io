Xong rồi đem về, Unzip vào một thư mục nào đó. Mở Kaspersky lên, vào Setting, chọn Update -> Configure... ->Add -> dẫn đến folder bạn vừa Unzip -> OK -> bỏ chọn Kaspersky Lab's update servers nếu có -> vào Kaspersky chọn Update -> Ok.
