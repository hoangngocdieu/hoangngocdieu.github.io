Các ngôn ngữ lập trình phía server như PHP, ASP, ASP.net thì không cần giấu source vì đối với các client truy cập vào trang web thì các trang đó đều chỉ hiển thị mã HTML. Cũng có một số chương trình giúp mã hoá các file ASP hoặc PHP nhưng không nhiều.

Với các trang HTML thì có chương trình HTMLEncrytor giúp mã hoá source HTML và JavaScript.

Còn những đoạn code như khoá chuột phải, select text... thì thường chỉ có hiệu quả với những trình duyệt thông dụng như IE, FF, NS... Nếu dùng các web browser khác vẫn có thể không có tác dụng.
