Để cấu hình định tuyến, mình xin đưa ra ví dụ cụ thể
Sơ đồ như sau:
172.68.178.196/26(E0)----(Router 1.1.1.1)---S0 -----serial line ----- S0---(Router 2.2.2.2)----(E0)172.68.1.1/26

Cấu hình định tuyến động

Cấu hình Router 1.1.1.1
interface Ethernet0
ip address 171.68.178.196 255.255.255.192

interface Serial0
ip unnumbered Ethernet0

router igrp 10
network 171.68.0.0

Cấu hình Router 2.2.2.2

interface Ethernet 0
ip address 172.68.1.1 255.255.255.192

interface Serial 0
ip unnumbered Ethernet0

router igrp 10
network 172.68.0.0

Hi,

CCNP vẫn có IS-IS nhưng mang tính chất giới thiệu và mục đích để học viên hiểu tổng quát về một giao thức routing cũ là chủ yếu. Tỷ lệ chương trình và câu hỏi về IS-IS khi ra thi cũng rất ít, gặp câu này khi ra thi bạn cứ yên tâm mà chọn OSPF. Sang đến BSCI code mới 642-901 thì phần IS-IS lại càng ít, trong nay mai chắc là sẽ bỏ luôn thôi (hồi trước đọc IS-IS, đúng là hơi nhức đầu với kiến trúc OSI của nó, về sau này loại bỏ IS-IS là quá khoẻ rồi).

Rain
