file NTLDR nằm trên ổ đĩa chứa Windows
Ví dụ C là ổ đĩa cài đặt WIndows của bạn
Copy file C:\NTLDR trong một máy Win XP đang chạy tốt vào đĩa mềm, rồi sang cái máy bị mất file NTLDR
- boot bằng đỉa CD (Hiren Boot CD) . Nếu ổ C đang format FAT32 thì bỏ đỉa mềm vô , copy file A:\NTLDR vào C:\
-Nếu ổ C đang là NTFS, thì chạy phần mềm đọc NTFS ngoài DOS để thấy được ổ C, rồi copy vào
