bạn hãy seach các bài làm đĩa auto ghost hoặc đĩa boot, mình có thể nói qua:
bạn đánh lênh ghost -split=600(chia file ra làm 2 files 600MB) để tạo files ghost sau đó nó sẽ chia thành 2 files
- files thứ nhất có đuôi *.gho, file 2 có đuôi*.ghs
- rồi bạn load 1 files iso chứa hiren boot về dùng ultraiso mở ra và aad files thứ nhất vào
rồi bạn save lại thành 1 files iso, ghi ra đĩa, còn file *.ghs bạn ghi ra 1 cd nhớ là "burn image"(phần này trong nero) nhé
- khi bạn ghost thì boot bình thường, chọn file chứa files ghost ở trong chính CD đó là ok
chúc thành công
