Có bạn nào lại gặp vấn đề với tính năng "Xem trước" như đề cập trong topic này không nhỉ? Ví dụ gửi bài với nội dung:

[ quote ]
abc
[ code ]
xyz
[ /code ]
[ /quote ]

và không chọn "Disable BB Code in this message" thì khi chọn vào "Xem trước" nó hiển thị thế này:

<blockquote>
abc
<span class="quotetxt">Code:</span>
<div class="coded"> <pre class="incoded"> xyz </pre> </div>
</blockquote>

còn khi chọn "Disable BB Code in this message" thì nó hiển thị đúng (nghĩa là gõ thế nào thì hiển thị thế ấy):

[ quote ]
abc
[ code ]
xyz
[ /code ]
[ /quote ]

Mà lựa chọn "Always allow HTML (yes/no)" trong Profile giờ không còn tác dụng nữa rồi vì hình như anh conmale đã disabled hoàn toàn HTML.
