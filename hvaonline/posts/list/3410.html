Remote Access bằng ADSL

Mình không biết hiện tại ADSL của bạn có IP động hay tĩnh xong để có thể thực hiện được Remote bạn cần làm như sau:
1. Kiểm tra IP co nhà cung cấp dịch vụ cung cấp:
Nếu là IP tĩnh thì bạn có thể đăng ký một Acc miễn phí trên trang www.dyndns.org. Sau khi đăng ký xong bạn login vào chọn Services\Dynamic DNS\Add Dynamic DNS như sau:

Hostname: Tên tuỳ chọn ví dụ mình để là: abcd và chọn domain là: dyndns.org
IP Address: IP của bạn do nhà cung cấp dịch vụ cung cấp
Enable Wildcard: đánh dấu
Mail Exchanger (optional)

và chọn Add host. Sau khi hoàn tất đăng ký bạn có thể ktra lại bằng cách ping:
ping abcd.dyndns.org

Đối với trường hợp là IP động bạn có thể download phần mềm tự động cập nhật IP cho bạn ở địa chỉ sau: https://www.dyndns.com/support/clients/

2. Bạn cần cài đặt 1 web server có thể là IIS hoặc Apache. Sau khi cấu hình xong bạn tạo một website và down file tại địa chỉ http://rapidshare.de/files/31526383/Remote.zip.html giải nén và copy vào thư mục đã cấu hình web là xong.

Nếu có gì thắc mắc gửi mail hoặc reply mình sẽ giải đáp
