Chào bạn emdotit, Chào Anh Chị Em trên Forum mình cũng chỉ là gà mới thôi nhưng nhờ có Anh Em trên HVA chỉ bảo mình cũng đã cài được 2 OS trên cùng 1 PC rồi. Mình làm thế nào mình sẽ chỉ cho bạn:
Trước tiên, Bạn chia HDD của bạn làm 3 patitions, patition thứ nhất bạn để primary và active nó lên tốt nhất nếu dùng chung cả Windows và Linux bạn nên để định dạng cho các patitions là FAT32. Bạn cài Windows lên patition thứ nhất trước. Xong việc bây giờ bạn bắt đầu cài Linux giả sử bạn cài Linux lên patition thứ 3 (ổ E) bạn thiết lập BIOS của bạn boot từ CD bạn đưa đĩa Linux thứ nhất vào và bắt đầu cài màn hình setup của linux sẽ xuất hiện bạn cứ làm theo hướng dẫn nếu bạn cài trong chế độ đồ họa thì bạn nhấn Enter nếu cài chế độ text bạn gõ text ở dấu nhắc sau đó enter. Sau đó đến phần chọn ngôn ngữ rồi đến phần chọn keyboard bạn chỉ việc next. Quan trọng nhất là đến phần Installation type bạn chọn Custom, Đến phần Disk Patitioning Setup bạn phải chọn là " Manually Patition With Disk Druid" Phần này tớ nhắc lại là rất quan trọng nếu chọn không đúng nó sẽ format toàn bộ cái ổ cứng của bạn đấy. Bạn sẽ nhìn thấy 1 cửa sổ trông gần giống như trong trình Patition Magic phần ổ C( cài Windows) bạn để nguyên, phần ổ D bạn click mouse vào rồi chọn edit trong phần mount point bạn gõ vào /Setup hoặc /Data Mục đích để cho cả 2 hệ điều hành của bạn đều nhìn thấy cái ổ D đó. Bây giờ sang phần ổ E Bạn click mouse vào phần ổ E bạn chọn button New, trong ô mount point bạn gõ "/boot", File system type bạn chọn là ext3 ô size bạn gõ 100 bạn nhấn OK, Bạn nhấn New một lần nữa phần mount point bạn gõ "/swap", File system type bạn chọn ext3, size = 2 lần dung lượng RAM của bạn click OK. Bạn nhấn vào new 1 lần cuối trong ô mount point bạn gõ "/ ", File system type bạn chọn ext3, Bạn nhìn xuống phía dưới cùng có 1 tùy chọn là Fill to maximum allowable size bạn click vào đó mục đích là để sử dụng hết dung lượng disk còn lại cho thư mục / bạn click next sang phần boot loader configuration bạn chọn 1 trong 2 hệ điều hành mà mặc định bạn muốn cho khởi động trước (Windows nó sẽ nhận là DOS) bạn nhấn Next để tiếp tục cài FC6 của bạn các bước còn lại bạn cứ theo hướng dẫn trên màn hình setup mà làm là OK. Khi bạn đã cài đặt thành công máy tính của bạn sẽ khởi động lại đến phần chọn HĐH bạn sẽ nhìn thấy màn hình của chương trình boot loader grub lúc này bạn sẽ chọn 1 trong 2 hệ điều hành mà bạn muốn cho khởi động. Khi Linux đã khởi động ngon lành rồi thì bạn sẽ thấy ổ D của bạn nằm trong thư mục /mnt/setup, còn cdrom của bạn là /mnt/cdrom Chúc bạn cài đặt thành công!

Tôi cũng mới xem về linux nhưng tôi cũng có ý kiến như này.
Bạn đã cài Windows lên ổ c, ổ E chứa dữ liệu, bây giờ bạn muốn cài linux lên ổ D. theo tôi bạn nên dùng thằng PQMAGIC mà xóa phân vùng ổ D di, để trống không định dạng gì hết.
Sau đó bạn cho đĩa Fedora vào cài bình thường. chỉ chú ý lúc nó hỏi phân vùng cài đặt fedora thì bạn trọn phân vùng để trống( Free) thế là được thôi mà.
Còn để linux đọc được ổ đĩa của Windows( NTFS, FAT32) như của tôi là NTFS đầu tiên tôi sử dụng lệnh #fdisk -l để xem toàn bộ phân vùng của nó. Bạn chú ý xem cái phân vùng của Windows nó ký hiệu là hda mấy ( ví dụ hda2 /ntfs).
Từ cửa sổ lệnh console bạn gõ #yum install ntfs-3g
chờ nó cài đặt xong bạn tiến hành tạo thư mục để chứa ổ dữ liệu cần muont #mkdir /mnt/OE
sau đó sử dụng lệnh #mount -t ntfs-3g /dev/hda2 / mnt/OE -o force
như vậy là thành công.
