Việc không thể resolve các thư viện này, tớ đoán có thể là do lúc install đã bị incorrupt hoặc tương tự thế. Đã remove lần lượt từng cái đi, sau đó reinstall nhưng vẫn "không ăn thua". Mong anh em chỉ giáo.

Nhân tiện nói về update, nếu anh em nào chạy "yum update" và thường xuyên bị time out khi chạy đến gói "filelists.sqlite.bz2" hoặc "primary.sqlite.bz2" hãy thử:
Code:
yum clear all
rm -fr /var/cache/yum/*
hoặc tăng timeout trong /etc/yum.conf lên 45 chẳng hạn

Theo error thì tôi thấy cậu cần install các packages missing như VLC, mplayer, xmms-faad2, ffmpeg-libs, ...thử yum install từng gói, thấy error requires gói nào thì cứ lần theo package từ cái error cuối cùng tới package trở lại mà install.
