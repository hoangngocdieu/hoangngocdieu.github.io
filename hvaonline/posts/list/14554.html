bạn bị đặt virus ở tất cả ổ rồi. bạn nên format lại ổ C, rồi bỏ chế độ ẩn file (cả file hệ thống) rồi mở các ổ còn lại qua Run. (start-run) đừng mở trực tiếp, là nhiễm lại virus đấy. sau đó xóa file *.exe ở trạng thái ẩn đi, và cả file autorun.inf nữa, là ổn.

Chú ý là có thể các file chương trình của bạn có thể đã nhiễm virus rồi, bạn mà mở các chương trình đó là nhiễm lại đó.
chúc thành công.
mà tốt nhất là format tất cả ổ đĩa đi ^^ (tạm chấp nhận vậy)

Xuất phát từ yêu cầu của nhiều người bị nhiểm virus autorun hiện nay, mình có một vài hướng khắc phục cơ bản.

đầu tiên bạn phải diệt virus trước đã tải chương trình BKAV mới nhất:

http://www.bkav.com.vn/home/Download/BHome.exe

và AVG Free Edition

http://fs6.filehippo.com/1178/8b8edab7daba43d79eb5b61d29a1932a/avg75free_487a1130.exe

cả hai đều free và luôn luôn là bản mới nhất.....

sau khi diệt xong thì ta phải thiết lập trở lại registry bằng tay vì lý do nhiều ứng dụng của Win bị virus khoa mất như Registry, taskmanager, Folderoptions....chưa kể file autorun.inf đáng ghét nằm trong mỗi ổ đĩa mà nhiều bạn chưa xóa được.

đầu tiên bạn download chương trình tại đây (sưu tầm và có sáng tạo thêm):

http://www.box.net/shared/uf0d0aj4ug

Sau khi tải về bạn được một file phuc_hoi.rar nặng 2Kb giải nén ra bạn được một thư mục on và 2 file REG_On.bat và show_autorun.bat.

b1: bạn copy file show_autorun.bat vào trong ổ đĩa chẵng hạn ổ D:\ sau click đúp vào bạn sẽ thấy 1 file ẩn có tên autorun.inf bạn xóa file này đi. làm tương tự với các ổ còn lại.

b2: bạn click đúp vào file REG_On.bat chương trình yêu cầu bạn nhấn 1 phím bất kì để logg off bạn cứ Enter.
==>sau khi logg off máy tính bạn trở lại như chưa từng bao giờ, bị nhiểm virus.
