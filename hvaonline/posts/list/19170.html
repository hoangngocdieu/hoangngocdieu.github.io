Bà con thử tạo 1 user , sau đó delete user đó qua cái mục User account trong control panel, hoặc qua control userpasswords2, hoặc xóa user profile trong mục System->Advanced->User profiles.

Trong mấy mục đó có chổ đưa ra yêu cầu lưu lại hệ thống thư mục của cá nhân user, có cái xóa cái rẹt hết. Sau khi xóa sài ngay get databack hoặc tương tự. Thử mới biết.
