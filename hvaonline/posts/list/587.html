Đem mạng internet về nhà
Thời gian đối với bạn là vàng , bạn không muốn bỏ đi những dòng thông tin quang trọng của một trang web nào đó. Bạn save as về usb của bạn.nhưng lúc về đến nhà bạn đang đọc một tin cực kì hấp dẩn thì không thể xem tiếp bởi vì máy bạn không nối mạng
Bạn lại ước nhà mình nối mạng nhưng không đủ kinh phí.
Một máy tính không hề nối mạng internet mà vẩn có thể lướt Web,đọc mail,được hay không?.xin thưa với các bạn hoàn toàn được.chỉ với 3.27mb phần mềm“WWC” có thể làm chuyện tưởng chừng như không thể đó.

Hướng dẩn cài đặt

Bạn không phải cài đặt trên máy của mình mà bạn cài đặt nó trên máy ở ngoài tiệm internet.Chương trình này giúp bạn download cả trang web mà bạn thích về nhà.cả những dòng link hay những kết nối trong trang web đó.Tạo cho bạn có cảm giác như máy của mình đang nối mạng vậy
Sau khi cài đặt trình giao diện như sau:

Chọn next để tiếp tục

Tại mục new project name: bạn gỏ tên mục đề của dự án sắp sữa làm:
Tại mục project calegory : bạn gỏ tên nhãn dự án
Chọn đương dẩn để lưu file:bạn nên chọn đường dẩn về ổ đĩa usb của mình.
Chon next để tiếp tục

Tại mục Action:có những sự lựa chọn sau
--Download web site(s) ownload website với dạng mặt định
--Download web site(s) + questionsownload website với dạng mặt định Và hỏi câu hỏi nếu gặp những dòng link được coi là một file dowload
--Get individual files:download những file dạng đặt biệt như zip, nhưng không thể lấy file thông qua dạng file HTML
--Download all sites in pages (multiple mirror):dowload tất cả những trang web khác xuất hiện trong trang web mà bạn lựa chọn
--Test links in pages (bookmark test) :sẽ kiểm tra tất cả những dòng link có ích(những file đã đánh dấu)
--Continue interrupted download :có thể sử dụng tùy chọn nếu quá trình downlod bị gián đoạn
--Update existing download : cập nhật quá trình download hiện tại
Bạn nên chọn dạng mặt định
Tại mục set option có các tùy chọn để bạn lựa chọn theo sở thích
Sau đó bạn Add URL:gỏ tên website cần dowload
Next/finish:
Có 2 sự lựa chọn khi hoàn thành bạn có thể đánh dấu để lựa chọn 1 trong 2 hoặc không chọn mục nào là : ngắt kết nối hoặc tự động tắt máy sau khi hoàn thành

Sau khi tải song có quá trình kiểm tra lổi các file.và sẽ liệt kê tất cả nhưng file lổi và thông tin của nó để chúng ta có thể tìm thấy nó sữa lổi
Bây giờ bạn đem thư mục vừa download về và chạy file index là song
Bạn có thể mua phần mềm này ngoài cửa tiệm bán đĩa phần mềm .
Một số điều cần lưu ý khi sử dụng:
 Không download website quá lớn
 Không sử dụng nhiều kết nối cùng lúc
 Sử dung băng thông, kết nối ,thời gian,và dung lượng giới hạn
