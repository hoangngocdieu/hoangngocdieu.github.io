Bạn thử dùng một số chương trình sau để phục hồi.

1. Recover4all Professional www.recover4all.com):

Phục hồi file trên các ổ cứng nội bộ có định dạng FAT12, FAT16, FAT32 và NTFS. Tương thích với Windows 9x/Me/NT/2000/XP.

Bạn chỉ cần bấm chuột vào ổ đĩa trong danh sách, chương trình sẽ dò tìm và liệt kê toàn bộ các thư mục/file đã bị xoá và cho biết chúng có thể phục hồi được hay không. Bạn chọn thư mục/file rồi bấm nút Recover để phục hồi, bạn sẽ được yêu cầu chỉ định thư mục lưu trữ.

2. FileRestore www.winternals.com):

Chương trình chạy tốt trong Windows 9x/Me/NT/4.0/ 2000/XP với ổ đĩa phân vùng FAT, FAT32 và NTFS. Có thể phục hồi file trên ổ cứng nội bộ, ổ mềm, ổ Jaz, Zip và cả ổ đĩa mạng nếu được cài trên máy server.

Chương trình này rất dễ sử dụng, bạn chỉ cần xác lập các thông tin sau:

* Gõ tên file vào ô Search for file name, bạn có thể dùng các dãy đại diện như: *, ?. Nếu tìm nhiều file, bạn phải phân cách chúng bằng khoảng trắng (space), dấu phẩy (,) hay dấu chấm phẩy (.

* Chỉ định ổ đĩa/thư mục cần tìm trong ô Look in.

* Để tăng tốc độ tìm kiếm, bạn có thể chỉ định tìm các file có ngày tháng trong một phạm vi nhất định (Last modified), hay giới hạn dung lượng file (Size between).

3. Office Recovery Pro http://www.OfficeRecovery.com):

Phần mềm này chạy tốt với Windows 9x/ME/NT/2000/XP, bổ sung lệnh Recover vào menu File của các ứng dụng Word, Excel, PowerPoint, Access (Office XP/2000/9x/). Có chức năng phục hồi dữ liệu trong các file bị hư hỏng của các ứng dụng nói trên. Chú ý: Không thể phục hồi nội dung Visual Basic, charts, drawing, audio, video clip, password và chương trình chưa hỗ trợ font Unicode. Đặc biệt, bạn có thể dùng lịnh Recover trong Excel có phiên bản cũ để đọc các tài liệu tạo bởi Excel phiên bản mới hơn. Với phiên bản Office Recovery Premium bạn có thể phục hồi cả file của Outlook và Outlook Express (6, 5, 4x). Khi cần phục hồi file, bạn chạy ứng dụng tương ứng (thí dụ: Word) rồi chọn File/ Recover. Trong hộp thoại Recover, bạn chọn file, nhập Password (nếu có). Cuối cùng, bấm nút Recover.

4. Advanced Zip Repairer (AZR):

Chương trình có khả năng phục hồi nội dung file nén - Zip và file nén tự bung - Self Extracting files(SFX). Hỗ trợ tất cả phiên bản của dạng thức Zip và SFX.

Đặc biệt, bạn có thể chạy AZR dưới dạng dòng lệnh sau: AZR filename [/O] [/E]

Với: filename là tên file nén cần phục hồi, /O là mở file sau khi phục hồi, /E là đóng chương trình sau khi phục hồi.

5. Easy Recovery Professional Edition - ERP http://www.ontrack.com):

Với phiên bản 6.0 (mới nhất), đây là một chương trình đa năng phù hợp với trào lưu “tất cả trong một” hiện nay. ERP chạy trong môi trường Windows 9x/Me/NT/2000/XP và có các chức năng sau:

* DriveTests: Kiểm tra vật lý ổ cứng bằng cách thử nghiệm đọc sector đĩa trong một thời gian hay trên toàn bộ ổ đĩa.

* SMARTTests: Kiểm tra tính năng SMART (Self-Monitoring, Analysis and Reporting Technology) của ổ cứng.

* PartitionTests: Kiểm tra cấu trúc hệ thống file (FAT, NTFS) trên đĩa cứng.

* DataAdvisor Diskette: Tạo đĩa mềm khởi động kèm theo các chương trình để kiểm tra vật lý ổ cứng (IDE/EIDE/ATA và SCSI), cấu trúc file hệ thống (FAT), bộ nhớ,...

* AdvancedRecovery: Phục hồi Partition bị hỏng do virus, do hư cấu trúc file hệ thống.

* DeletedRecovery: Tìm và phục hồi file bị xoá. Cho phép tìm file và lọc file theo các yếu tố do người dùng chỉ định.

* FormatRecovery: Phục hồi file từ partition đã bị xoá, bị format hay bị chép đè. Chương trình sẽ bỏ qua hệ thống hiện hành và cố gắng truy tìm hệ thống file có trước đó.

* RawRecovery: Chương trình “quét” tất cả sector trên đĩa cứng để nhận dạng file (trong cơ sở dữ liệu của ERP có khoảng 200 kiểu file), rất hiệu quả khi phục hồi file có dung lượng chứa đủ trong một cluster hay file không bị phân mảnh (chứa trong các cluster liên tục).

* ResumeRecovery: Cho phép bạn tạm ngưng tiến trình phục hồi và tiếp tục lại sau này mà không cần phải làm lại từ đầu.

* EmergencyDiskette: Tạo đĩa mềm khởi động có thể phục hồi dữ liệu khi không khởi động Windows được nữa. Trong đĩa này có phiên bản chạy trong môi trường DOS của ERP.

* FileRepair: ERP có thể phục hồi file Microsoft Access v2.0/9x/2000/XP, Excel 9x/2000/XP, PowerPoint 97/2000/XP, Word 6.0/9x/2000/XP và file ZIP.

* EmailRepair: Cho phép phục hồi file của Outlook 6/9x/2000/XP.

Chú ý: Sau khi dò tìm, chương trình sẽ hiển thị các file tìm được theo đúng cấu trúc cây thư mục nguyên thuỷ của chúng. Tất cả dữ liệu phục hồi phải được sao chép qua partition hay ổ đĩa khác. Cho phép chép qua ổ đĩa cục bộ, ổ đĩa mạng, ổ đĩa tháp lắp. Cho phép nén file (zip) khi sao chép với dung lương tối đa là 4GB và có thể bao gồm 65.535 file trong một file nén. Chương trình sẽ không làm bất cứ thay đổi gì trên ổ cứng bị hư hỏng.

Bạn nên RawRecovery khi không biết rõ ổ cứng hay khi chương trình không thể tự động dò tìm được cấu trúc của ổ đĩa. Khi chọn RAW, chương trình sẽ phân loại các file tìm được theo phần tên mở rộng của chúng và sẽ tạo ra nhiều thư mục để lưu từng loại file.

6. Passware Password Recovers Kit - PPRK http://www.lostpassword.com):

Rất nhiều người sử dụng mật khẩu (password) khi soạn thảo các tài liệu riêng tư nhằm mục đích bảo mật. Tuy nhiên, những người hay đặt mật khẩu cũng là những người hay quên... mật khẩu. Để giúp đỡ những người đãng trí, các hãng phần mềm cho “ra lò” nhiều chương trình dò tìm mật khẩu và các “tin tặc” đã lợi dụng các chương trình này để xâm nhập vào dữ liệu cấm. Thực tế điều này cũng ít khi xảy ra vì nếu bạn không biết được vài đặc điểm của password (để giới hạn việc dò tìm) thì thời gian dò tìm có thể kéo dài vài chục hay vài... trăm năm (với mật khẩu dài 16 ký tự và bao gồm đủ loại ký tự), cho dù tốc độ dò tìm có thể lên đến 1 triệu password/giây với CPU PentiumIII.

PPRK có thể dò tìm mật khẩu của 1-2-3, Access, Acrobat, ACT, Asterisk, Backup (MS, Seagate, Symantec, Norton), Excel, Internet Explorer, FileMaker, Lotus Notes, MS Mail, MS Money, MYOB, MS Office, Lotus Organizer, Outlook, Outlook Express, RAR, Paradox, Peachtree, Project, Quattro Pro, QuickBooks, Quicken, Schedule, VBA, Windows XP/2000/NT, Word, WordPerfect, WordPro, Zip...

Cách sử dụng rất đơn giản:

* Mở Start/Program Files/Passware/xxxx Key (thí dụ: Office Key).

* Mở File/Recover... rồi chỉ định file cần dò tìm mật khẩu. Để tăng tốc độ dò tìm, bạn bấm nút Settings. Trong hộp thoại Settings, bạn xác lập các thông số như: chọn từ điển, chọn số lượng ký tự, loại ký tự, ký tự đặc biệt,...

Đặc biệt, chương trình Windows XP/2000/NT Key sẽ tạo một đĩa mềm gọi là Windows Key có khả năng tạo lại mật khẩu quản trị (Administrator) cho Windows XP/2000/NT.

Phục hồi ổ đĩa:

* GetDataBack: http://www.runtime.org. Tương thích Windows 95/98/NT/2000/XP.

* PowerQuest Lost&Found: http://www.powerquest.com. Tạo ba đĩa mềm khởi động và phục hồi dữ liệu trên ổ cứng IDE, SCSI, JAZ và ZIP.

* Tiramisu Sophisticated data recovery: Hỗ trợ các ỗ đĩa Dos/Windows, Novell, NTFS, Zip. Phục hồi file:

* Fast File Undelete: http://www.dtidata.com/products_ff_undelete.asp.

* BadCopy Pro: http://www.jufsoft.com. Phục hồi dữ liệu trên ổ mềm, CD-ROM, CD-R/W.

* Undelete for Windows NT/2000: http://www.softwareshelf.com/products/undelete2.asp. Phục hồi file trên các hệ thống Windows NT/2000/XP.

* Self-Extractor Archive Recovery: http://www.softpile.com/Utilities/Compression/Review_05089_index.html. Phục hồi file nén tự bung tạo bởi WinZip.

Phục hồi mật khẩu:

* Advanced Password Recovery: http://www.elcomsoft.com/prs.html. Phục hồi mật khẩu của rất nhiều chương trình nén, văn phòng, email, trò chuyện trực tuyến...

* Accent Password Recovery: http://www.passwordrecoverytools.com/en/index.shtml. Phục hồi mật khẩu của Microsoft Office.

Hoặc bạn mua 1 bộ đĩa hirent-boot nó cũng khá hiệu quả để phục hồi
