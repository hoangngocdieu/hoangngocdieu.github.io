Trên MySQL server, `show full processlist\G` để xác định source port:
Code:
*************************** 35. row ***************************
     Id: 27186
   User: driver
   Host: web.server.ip.addr:52181
     db: dbname
Command: Sleep
   Time: 8
  State: 
   Info: NULL

Trên Web server, dùng `netstat` để xác định xem connection nào đang có source port như thế:
Code:
# netstat -natp | grep :52181
tcp        0      0 web.server.ip.addr:52181         mysql.server.ip.addr:3306           ESTABLISHED 7759/httpd


Bạn có thể tham khảo thêm `lsof`, `fuser`, `strace`, ...
