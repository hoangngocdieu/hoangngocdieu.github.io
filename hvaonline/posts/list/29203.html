Cài ra hẳn máy thật đi bạn. Dùng xong khéo bỏ windows luôn!
Cài thì chỉ có câu lệnh với Fedora

Code:
yum install openssh # trong quá trình cài cái này nó tự sinh key luôn, nó cũng tự down cả openssl về cài nếu chưa có.

Đọc thêm cái file sshd_config nữa xem nó có gì.
Một máy khác cậu dùng window cài copssh cũng được.
Copssh vừa là client vừa là server. Dùng nó cũng đủ demo rồi.

SSH secure communication được thực hiện trên cơ chế Public Key.
Client cần Public Key của server trước khi thiết lập kết nối. Client sẽ bị hỏi có đồng ý với Public Key này không. Chọn yes! Sau đó quá trình sẽ được thực hiện dựa trên giao thức na ná SSL. Bạn sẽ bị hỏi password. Muốn không bị hỏi password thì bạn tìm hiểu cơ chế login ssh without password (Gợi ý: file authorized_keys và sshd_config)

Cậu có thể show bằng cách bắt gói tin với tcpdump và ethereal gì đó!
