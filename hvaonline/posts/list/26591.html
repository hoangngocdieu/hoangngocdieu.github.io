Mình làm trên FF với SWF decompiler
Đầu tiên bạn mở trang đó ra trên FF rồi vào tools/page info/ tab media
thấy có 1 file flash là http://comic.nhieusach.com/Flash/truyen.swf
Down nó về rồi mở ra bằng SWF decompiler. Bên phải có các thành phần của Flash thì bạn chọn vào Action để coi code của nó.
Code:
function loadBook()
{
    var _loc2 = new Object();
    _loc2._x = 410;
    _loc2._y = 360;
    _loc2.extXML = [color=red]mi9[/color];
    myBook = _root.attachMovie("FFlippingBookSymbol", "FFlippingBook", 1, _loc2);
} // End of the function

Vậy là nó load một biến để php xử lý lấy ra số tập truyện và truyện đang xem chứ ko phải flash làm
View source trang đó lên:
Code:
<embed src="http://comic.nhieusach.com/Flash/truyen.swf" flashvars="mi9=http://comic.nhieusach.com/getxml.php" quality="high"
> bạn mở trang http://comic.nhieusach.com/getxml.php là tìm được link của các file mà flash đã hiển thị cho bạn xem
