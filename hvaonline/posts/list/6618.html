Bro đang nói cách làm home server qua ADSL khi có 1 domain, nếu vậy thì có các bước sau:
- Đổi cổng default trên web server khác với 80, vd: 8080 (80 là cổng mặc định dùng cho ADSL router)
- Kết nối tới router, cấu hình NAT, cho router forward tới máy đang chạy web server, cần biết IP & Port cuả web server
- Cuối cùng log in vào cái domain control panel, rồi cho domain trỏ tớ cái IP cuả ADSL router.
Có gì sai sót xin mọi người góp ý thêm.
