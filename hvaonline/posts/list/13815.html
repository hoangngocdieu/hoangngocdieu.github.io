Đúng là nó kô phải virus đâu bạn ạ, nhưng nếu trước đây bạn kô thấy nó mà tự nhiên bây giờ thấy tự nhiên thấy rất nhiều thì có thể đó là do một con virus tạo ra những cái đó với mục đích gì đó. bạn hãy open cái file desktop.ini đó ra và paste cái nội dung của file đó lên đây xem có gì lạ kô.

Theo mặc định thì cái file đó nếu được sinh ra thì sẽ được ẩn ở mức hệ thống. nên bạn chỉ nhìn thấy nó nếu bạn mở ẩn sytem thôi. bạn kiểm tra xem ẩn system có bị mở kô nhá.(vào ổ C nếu thấy một đống các file AUTOEXEC.BAT, boot.ini, CONFIG.SYS... hiện ra thì có nghĩa là ẩn OS đang bị mở bạn hãy ẩn nó lại là ok.(tools > folder option > view > tick lại cái dòng "Hide protected...")

Nếu bạn muốn xóa chúng đi thì cũng chẳng sao đâu. hãy search trên toàn ổ với từ khóa desktop.ini rồi Delete all những gì tìm được là xong. nếu sau khi del xong mà lại thấy chúng xuất hiện lại ngay thì chắc là có vấn đề thật đấy bạn ạ.
