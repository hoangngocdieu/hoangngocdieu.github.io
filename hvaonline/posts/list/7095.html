Không nên dùng các software hỗ trợ config webserver . Nên tự download từng phần về , tự config cho quen . Như vậy bồ sẽ hiểu được tác dụng của từng giá trị trong các file php.ini , httpd.conf ... , nó sẽ hỗ trợ bồ rất nhiều trong việc code và security sau này .
Nên nhớ bồ config local webserver 1 kiểu nhưng tụi hosting server nó lại config 1 kiểu khác , nếu ko nắm vững các giá trị của apache , php , khi php hay apache upgrade lên 1 version mới hơn , bồ sẽ gặp khó khăn trong việc fix web-app để nó tương thích với version php , apache mới .
PHPMyAdmin thì chỉ cần sửa 3 giá trị cache , user , pass trong file config.inc.php là ok .
Và cuối cùng : hiện chưa nên dùng apache 2.2.x , dùng 2.0.x trước đi .
