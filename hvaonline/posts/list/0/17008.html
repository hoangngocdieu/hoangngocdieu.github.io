Mình làm quen với Linux chưa lâu nhưng rất yêu thích Linux. Máy tính ở nhà đã dùng Fedora rồi, nhưng mình muốn có một bản nhỏ gọn vào USB để khi đi xa là đem ra xài.
Mình có thử qua vài loại nhưng cuối cùng đã tìm ra loại thích hợp nhất, đó chính là SLAX
SLAX sử dụng giao diện KDE khá đẹp mắt, có nhiều version khác nhau, tích hợp nhiều phần mềm tiện ích cơ bản

Sau đây là bài hướng dẫn mình tìm được, phiên bản gốc tiếng Anh http://www.pendrivelinux.com/2006/09/20/all-in-one-usb-slaxzip/

USB SLAX install
SLAX là một distro Live Linux CD được tạo ra bởi Tomas Matejicek. SLAX về cơ bản là một phiên bản rút gọn của Slackware Linux. Nó sử dụng sự hợp nhất các hệ thống tệp tin cho phép filesystem chỉ đọc vận hành như một filesysem có thể ghi, điều đó cho phép người dùng lưu lại những thay đổi của hệ thống. Chỉ dẫn dưới đây sẽ chỉ cho bạn cách làm thêm nào để cài đặt SLAX vào thiết bị lưu trữ di động USB từ Windows.

SLAX Screenshots:



Những yêu cầu cơ bản:
- Một USB 256MB hoặc lớn hơn
- Công cụ format HP-USB (Tùy chọn)
- File ISO SLAX Linux
- Tiện ích xả nén ISO (7-Zip)
- Syslinux

Chỉ dẫn cụ thể:
1. Download công cụ format HP-USB http://selfdestruct.net/misc/usbboot/SP27213.exe và format USB của bạn sang định dạng FAT hoặc FAT32
2. Download file Slax ISO http://merlin.fit.vutbr.cz/mirrors/slax/
3. Sử dụng công cụ xả nén và xả file ISO SLAX vào thư mục gốc của USB
4. Download file http://pendrivelinux.com/downloads/fixs.zip và cũng xả nén nó vào thư mục gốc của USB
5. Chạy file fixs.bat từ USB
6. Download http://www.kernel.org/pub/linux/utils/boot/syslinux/ và xả nén vào một thư mục có tên C:\syslinux trên máy tính của bạn
7. Từ Windows click start-> run-> cmd
8. Từ command window, gõ cd C:\syslinux\win32
9. Gõ syslinux.exe -ma F: (thay F bằng ký hiệu ổ USB của bạn) để làm cho USB của bạn có thể khởi động được
10. Khởi động lại máy tính và cài đặt hệ thống BIOS khởi động từ USB-ZIP hoặc USB-HDD.
11. Lưu lại BIOS settings. Lần khởi động kế tiếp, bạn sẽ khởi động thành công SLAX Linux của bạn từ USB

- Nếu bạn cần thêm thông tin trợ giúp để giúp flash drive của bạn khởi động, tìm kiếm thêm http://www.pendrivelinux.com/category/bios-usb-boot-options/( our BIOS boot options section) hay http://www.pendrivelinux.com/category/tools/ (Tools Section).

Bài trên là mình dịch lại theo cách rườm rà của bài hướng dẫn mình tìm được.
Bài đó hướng dẫn dùng phiên bản 5.x. Khi làm live USB thì mỗi lần khởi động lại nó quay về trạng thái như lần khởi động đầu tiên, không lưu các thay đổi gì.

Bạn "Mèo" có thể dùng bản 6.x này, không cần tải gì thêm vì nó có sẵn syslinux rồi:
http://merlin.fit.vutbr.cz/mirrors/slax/SLAX-6.x/rc7/ để download bản mới nhất rc7

- Giải nén bằng winrar file ISO cũng được, copy tất cả vào USB
- Vào thư mục boot/syslinux có sẵn sau khi giải nén, copy file syslinux.cfg ra thư mục gốc của USB
- Copy file syslinux.exe vào thư mục C:\Windows
- Vào Command Prompt và gõ:
Code:
syslinux X:

Thay X: bằng ký hiệu USB trong Windows
