Ý bạn là muốn nhờ Mr.Khoai gửi cho cái gì, codec cho MPlayer hả, bạn có thể load trên site của nó mà:
http://www.mplayerhq.hu/design7/dload.html
Sau đó:
Code:
# tar -jxvf essential-20061022.bz2
# cd essential-20061022
# cp * /usr/lib/codecs/

Nếu chưa có thư mục codec, bạn tạo nó trước rồi tạo một soft link của tất cả các file trong thư mục này đến /usr/lib/win32 bằng lệnh
Code:
# cd /usr/lib/win32
# ln -s /usr/lib/codecs/*
Let's build on a great foundation!
