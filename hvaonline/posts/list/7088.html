Sử dụng các chương trình Hide IP thì ISP có theo doi được không, tác dụng của các công cụ Hide IP là gì?  


Theo mỗ được biết thì Hide IP được dùng để che dấu IP thực sự của bạn trên Internet , nghĩa là khi bạn từ máy của bạn ( A ) muốn truy cập một website ở C thì máy của bạn sẽ kết nối tới một server ( gọi là Proxy hoặc Sock ) và nhờ server đó truy cập vào C rồi gửi thông tin của C về cho chúng ta tại A . Như vậy , tại C chỉ biết là thằng server kết nối tới nó chứ ko biết sự truy cập của bạn ( A ).
Tuy nhiên , cái gì cũng thế , không dấu được ISP đâu . Giả sử bạn truy cập vào một website đặt tại Canada , và bạn truy cập gián tiếp thông qua một Proxy Server hoặc Sock Server đặt tại Mỹ . Trước tiên bạn sẽ dùng IP của chính mình ( IP thật ) truy cập đến Server ( Sock hoặc Proxy ) mà để truy cập đến đó thì bạn phải thông qua ISP , rồi từ Server trở đi thì IP của bạn mới được dấu nhẹm hoàn toàn ( tùy Proxy ngon hay dởm ) . Cho dù bạn có liên kết đến một website đặt máy chủ tại Việt nam đi nữa thì cũng phải tuân theo quá trình trên ( nếu bạn Fake IP ) .
