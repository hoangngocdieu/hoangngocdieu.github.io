Database error The Forum's database has encountered a problem có nhiều khả năng xãy ra .
- Mysql server bị down
- Mysql bị crash, tình trạng này thường xãy ra đối với server có chia nhiều account "bán lẽ" (shared hosting)

Giải pháp :
Chuyển sang server mới
Tăng tài nguyên như RAM cho máy chủ .

Bro đăng nhập vào vBulletinCP có phần Session Timeout, tinh chỉnh cho nó cho phù hợp .

Trên đây chỉ là phần "đoán" của mình mà thôi .
