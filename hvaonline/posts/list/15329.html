- Dùng đĩa Hirent's Boot để boot vào môi trường DOS

- Format phân vùng dự định cài hệ điều hành có tích hợp Hirent's ( mọi phiên bản ) với tham số /s ( vd : format c:/s ) ( có thể thêm tham số /q để thực hiện format nhanh ( quick format ).
- Đặt đĩa cài ( Win2k,Xp,2k3 ) vào ổ đĩa rồi reboot.

- Quá trình cài đặt diễn ra đến phần chọn hệ thống định dạng tệp ( FAT/NTFS ) các bạn chọn dòng cuối cùng ( ...no changes ) rồi tiếp tục cài đặt.

- Khi quá trình cài đặt thành công, các bạn mở Window Explorer rồi truy xuất đến ổ C ( ổ đĩa chứa HDH ); vào Tools > Folder Options > Chọn tab Views > Chọn Show
hidden files and folders > Bỏ dấu check tại dòng Hide protected operating system files > Yes > OK để hiển thị những tệp hệ thống bị ẩn theo mặc định.

- Xóa file MSDOS.sys, bỏ thuộc tính hệ thống và thuộc tính ẩn của file Command.com và IO.sys ( có thể bỏ qua bước này nhưng phải chọn chép đè trong bước sau )

- Đặt đĩa Hirent's vào ổ đĩa, sau đó dùng một chương trình như UltraISO hay WinImage để extract phần boot của đĩa CD vào một thư mục bất kỳ ( vd thư mục Boot )

- Vào thư mục Boot, xóa 2 file IO.sys, JO.sys sau đó dùng notepad để mở tệp MSDOS.sys lên, xóa 3 dòng windir=, WinBootDir=, HostWinBootDrv=a rồi save lại.

-Chép toàn bộ các tệp trong thư mục Boot ( nhớ chép đè nếu được hỏi ) vào ổ đĩa C, sau đó chép thư mục BootCD trong CD-ROM vào ổ C.

-Click phải vào icon My Computer > Properties > Chọn tab Advanced > Chọn Settings trong phần Startup and Recovery > Edit > Tại dòng cuối cùng sửa "Microsoft Windows" thành "Hiren's BootCD v.9.3" ( nếu bản LHT của bạn là v.9.3 ) sau đó save lại ( tại đây bạn còn có thể chỉ định thời gian sẽ hiển thị 2 lựa chọn này là bao lâu tại dòng "timeout=" ).

- Đóng tất cả các cửa sổ lại rồi reboot.
