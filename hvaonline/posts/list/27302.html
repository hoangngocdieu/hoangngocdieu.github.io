Windows Vista is Microsoft’s latest Operating system. New features of the system include set up ease, more security and reliability, and compatibility with the widest range of software, devices, and services. Windows Vista contains a number of new security features including built in firewall and spam filter. Windows Vista also has a new design with a high ends graphical user interface and new desktop features. VTC Author Eric Butow, guides users through the basics of the new and updated features of this major operating system upgrade.
