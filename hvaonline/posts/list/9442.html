Kô lo đâu! con này thường thôi.
USB như vậy tôi gặp nhiều rồi. Mýa cái kí tự loằng ngoằng đó là do file Autorun.inf trong USB tạo ra.
Đây là một ví dụ:
[autorun]
OPEN=EXPLORER.EXE
shell\open=??(&O)
shell\open\Command=EXPLORER.EXE
shell\open\Default=1
shell\explore=?????(&X)
shell\explore\Command=EXPLORER.EXE 


Chắc là con virus đã bị Kaspersky diệt rồi nhưng vẫn còn file Autorun thôi(các chương trình AV thường kô delete file Autorun.inf mà chỉ diệt file nguồn của virus thôi)

Bạn làm như sau để vào kiểm tra xem máy có phải bị nhiễm hay kô:
1.mở ẩn như bạn dgthien nói:
vào tool->Folder Options-View sau đó bỏ đánh dấu ở ô Hide protected Operating System files và chọn show hidden file and folder 

2.mở My Computer giống như ảnh bạn đã post lên.
3.vào USB bằng cách click chuột vào USB ở khung bên trái(câ thư mục ấy).Chỉ cần nháy click 1 cái là vào được.

Bây giờ bạn sẽ thấy cái file Auturun.inf ở trong USB. Bạn cứ click đúp vô nó để mở ra và bạn sẽ thấy code của nó. Nhìn xem nó OPEN file nào. Tìm trong USB xem có file ấy kô. nếu kô có thì nghĩa là file virus đó đã bị diệt rồi. Còn nếu có mà Kaspersky kô rú lên đòi delete thì mới đúng là Kaspersky chưa diệt được con này.(tôi cược là kô còn file đó)
Bạn delete file Autorun.inf ấy đi, rồi thoát USB ra, sau đó cắm lại. Bây giờ bạn sẽ thấy USB bình thường(kô còn mấy dòng loằng ngoằng đó nữa.

Làm xong nhớ hồi âm nhá, để người khác còn học hỏi.

Vào USB tạo một cái folder đặt tên là "Autorun.inf" rồi vào dos gõ lệnh: attrib +s +h E:\Autorun.inf

E là ổ USB và trong USB phải đang kô có file Autorun.inf

Từ giờ USB của bạn sẽ kô sợ bị virus tạo ra file Autorun.inf nữa mà USB cũng sẽ kô có cái dòng AutoPlay đáng ghét ở trên đỉnh hay bất cứ cái tác dụng kì quặc nào do file Autorun.inf tạo ra nữa. Trừ khi USB bị format USB hoặc xóa cái folder Autorun.inf đó.
