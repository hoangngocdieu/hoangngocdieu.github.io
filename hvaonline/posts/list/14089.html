Bạn thấy rõ là khi dùng http proxy, pidgin sẽ sử dụng method CONNECT để tunnel với server yahoo và proxy của bạn đã deny method này.
Nếu bạn thích dùng proxy thì tôi khuyên bạn nên sử dụng http://tor.eff.org/. Dùng tor cho sock proxy và privoxy cho http proxy.
