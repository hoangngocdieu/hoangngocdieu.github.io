Hello các bạn. Mình có đọc thấy trên NET có một khóa học về Quản trị mạng doanh nghiệp. Chương trình học cụ thể như sau:
Nội dung chương trình CNAM
CNAM là khóa học chưa đựng đầy đủ kiến thức về mạng doanh nghiệp với 10 phần học sau:

• Part 1 : Computer Installation
o HDD Disk & partition
o Mastering Windows Installation
o Disk duplicate& Windows deployment tool
o Inside Anti-virus process

• Part 2 : Network basic
o LAN cabling
o Addressing layer 2 & layer 3
o Mastering LAN deployment
o Configure ADSL & Dial-up to share internet connection for a real LAN environment.

• Part 3: Other technology
o Wi-fi installation & secure Wi-Fi network
o Boot rom ( run windows without HDD)

• Part 4: Server installation & Fault tolerance
o Mastering Window 2003 Installation
o Disk fault tolerance : SCSI RAID

• Part 5: Network service
o DNS, WINS, DHCP

• Part 6: Standard server deployment
o Domain Controller
o File & Print server
o Backup strategy

• Part 7: High-level server deployment
o HTTP, SMTP, POP3, IMAP
o Web server : IIS 6.0
o Email server : Exchange 2003
o Dynamic Routing : RIP, OSPF
o RAS, VPN, Radius server

• Part 8: Network management
o Deploy a complete Firewall : ISA 2004
o Manage users and other resources
o Network monitor & Access audit

• Part 9: Setup standard networks for a company
o Create IT policy.
o Step by step to built a standard network

• Part 10: Experience in supporting users

Các bạn ai đã học qua chương trình này thì cho mình vài lời khuyên. Hoặc các bạn có tài liệu về chương trình học này thì cho mình xin. Mình cần nuốt chương trình này trong 1 thắng. Rất mong các bạn tham gia giúp đỡ.

Cảm ơn.
