Lệnh AT: The AT command schedules commands and programs to run on a computer at a specified time and date. The Schedule service must be running to use the AT command.
AT [\\computername] [ [id] [/DELETE] | /DELETE [/YES]]
AT [\\computername] time [/INTERACTIVE]
\\computername Specifies a remote computer. Commands are scheduled on the
local computer if this parameter is omitted.
id Is an identification number assigned to a scheduled command.
/delete Cancels a scheduled command. If id is omitted, all the scheduled commands on the computer
are canceled.
/yes Used with cancel all jobs command when no further confirmation is desired.
time Specifies the time when command is to run.
/interactive Allows the job to interact with the desktop of the user who is logged on at the time the job
runs.
Examples:
at 6:50PM "c:\winnt\task.bat"

Performing this command would run the task.bat file at 6:50PM today unless the task is scheduled after the time specified. Because the /interactive switch is not in the command, this batch file will be executed in the background.

at 6:50PM /interactive c:\winnt\beep.bat

Runs the beep.bat batch file at 6:50PM with the ability for the user to interact with the batch file.

Lệnh ATRRIB: About attrib

Attrib allows a user to change the properties of a specified file. Using attrib, the user has the capability of changing the file to have any of the below attributes. Note if you're wanting to change the ACL's of a file see the CACLS command.

Read-only - allowing the file to be only viewed and not written to.

Archived - allowing Microsoft backup and other backup programs to know what files to backup.

Hidden - making the file invisible to standard users.

System - making the file an important system file.

Cú pháp:
Displays or changes file attributes.

ATTRIB [+R | -R] [+A | -A ] [+S | -S] [+H | -H] [[drive:] [path] filename] [/S [/D]]

+ Sets an attribute.
- Clears an attribute.
R Read-only file attribute.
A Archive file attribute.
S System file attribute.
H Hidden file attribute.
/S Processes files in all directories in the specified path.
/D Process folders as well.

Examples

attrib

Typing attrib by itself will display all files in the current directory and each of their attributes. If any file is hidden it will also display those files.

attrib +r autoexec.bat

Add the read-only attribute to the autoexec.bat file so it cannot be modified until the read only attribute is taken off. This is helpful for important system files or any other file that you do not want to have mistakenly edited or changed by another program.

attrib +h config.sys

Add the hidden attribute to the config.sys file causing it to be not be seen by the average user.

attrib -h config.sys

This command will do the opposite of the above command. Instead of hiding the file it will unhide the file if hidden.

Lệnh Break: About break

Break can be used to enable or disable the breaking capability of the computer. For example, if a user wanted to cancel a batch file or another MS-DOS processes, that user could simply press CTRL + C (break), which would then prompt the user if they wish to cancel the current process.

Turning break off will cancel CTRL + C; however, the user will still be able to press CTRL + PAUSE/BREAK and have the capability of getting out of a batch file / current running process.

Note: Newer versions of Windows (Windows ME, Windows 2000, Windows XP, and higher) only include this command for backwards compatibility and turning the break off has no effect.

Syntax

Sets or clears extended CTRL+C checking.

BREAK [ON | OFF]

Type BREAK without a parameter to display the current BREAK setting.

Examples

break on

Turns on the break, which allows the Ctrl + C function.

break off

Turns off the break, not allowing Ctrl + C to cancel a process.

Lệnh ERASE: About erase

Erase is a command used to remove files from your computer's hard disk drive and or other writable media.

Windows 2000 and Windows XP Syntax

Deletes one or more files.

DEL [/P] [/F] [/S] [/Q] [/A[[:]attributes]] names
ERASE [/P] [/F] [/S] [/Q] [/A[[:]attributes]] names

names Specifies a list of one or more files or directories. Wildcards may be used to delete multiple files. If a directory is specified, all files within the directory will be deleted.
/P Prompts for confirmation before deleting each file.
/F Force deleting of read-only files.
/S Delete specified files from all subdirectories.
/Q Quiet mode, do not ask if ok to delete on global wildcard
/A Selects files to delete based on attributes
attributes R Read-only files S System files H Hidden files A Files ready for archiving - Prefix meaning not

If Command Extensions are enabled DEL and ERASE change as follows:

The display semantics of the /S switch are reversed in that it shows you only the files that are deleted, not the ones it could not find.

Examples

Erase c:\windows\*.FAK

Delete any files that end with the FAK extension.

Notice: Users will not receive a confirmation prompt when performing this command. Once executed, all files will be deleted unless the files have a read-only attribute and the /F switch is not being used.

Lệnh Format: About format

Format is used to erase all of the information off of a computer diskette or fixed drive.

Microsoft Windows 2000 and Windows XP syntax

Formats a disk for use with Windows 2000.

FORMAT volume [/FS:file-system] [/V:label] [/Q] [/A:size] [/C] [/X]
FORMAT volume [/V:label] [/Q] [/F:size]
FORMAT volume [/V:label] [/Q] [/T:tracks /N:sectors]
FORMAT volume [/V:label] [/Q] [/1] [/4]
FORMAT volume [/Q] [/1] [/4] [/8]

volume Specifies the drive letter (followed by a colon), mount point, or volume name.
/FS:filesystem Specifies the type of the file system (FAT, FAT32, or NTFS).
/V:label Specifies the volume label.
/Q Performs a quick format.
/C Files created on the new volume will be compressed by default.
/X Forces the volume to dismount first if necessary. All opened handles to the volume would no longer be valid.
/A:size Overrides the default allocation unit size. Default settings are strongly recommended for general use.
NTFS supports 512, 1024, 2048, 4096, 8192, 16K, 32K, 64K.
FAT supports 512, 1024, 2048, 4096, 8192, 16K, 32K, 64K, (128K, 256K for sector size > 512 bytes).
FAT32 supports 512, 1024, 2048, 4096, 8192, 16K, 32K, 64K, (128K, 256K for sector size > 512 bytes).
Note that the FAT and FAT32 files systems impose the below restrictions on the number of clusters on a volume:

FAT: Number of clusters <= 65526 FAT32: 65526 < Number of clusters < 268435446

Format will immediately stop processing if it decides that the above requirements cannot be met using the specified cluster size.

NTFS compression is not supported for allocation unit sizes above 4096.

/F:size Specifies the size of the floppy disk to format (160,
180, 320, 360, 640, 720, 1.2, 1.23, 1.44, 2.88, or 20.8).
/T:tracks Specifies the number of tracks per disk side.
/N:sectors Specifies the number of sectors per track.
/1 Formats a single side of a floppy disk.
/4 Formats a 5.25-inch 360K floppy disk in a high-density drive.
/8 Formats eight sectors per track.

Examples

When using the format command, remember all information on the drive you wish to format will be completely erased.

format a:

Would erase all the contents off a disk. Commonly used on a diskette that has not been formatted or on a diskette you wish to erase.

format a: /q

Quickly erases all the contents of a floppy diskette. Commonly used to quickly erase all information on the diskette.

format c:

This would erase all the contents of your C: hard disk drive. In other words, unless you wish to erase all your computer's information, this command should not be done unless you're planning to start over.

Lệnh GRAFTABL: About graftabl

Graftabl command allows the display to show extended characters in graphics mode from a specific code page. Most monitors do not need the graftabl command to be loaded.

Syntax

GRAFTABL [CODE]
GRAFTABL [/STATUS]

437 United States Code
860 Portugal Code
863 Canada (French) Code
865 Norway and Denmark Code
/STATUS Display the number of the currently selected code page.

Examples

graftabl 437

Loads graftabl for the United States.

graftabl

Typing the command alone will display the current settings as shown below.

Previous Code Page: None
Active Code Page: 437

Lệnh MKDIR: About md

Allows you to create your own directories in MS-DOS.

Syntax

Creates a directory.

MKDIR [drive:]path
MD [drive:]path

Examples

md test

The above example creates the "test" directory in the directory you are currently in.

md c:\test

Create the "test" directory in the c:\ directory.

Lệnh AT: lệnh AT lịch lệnh và các chương trình để chạy trên một máy tính tại một thời gian xác định và ngày. Các dịch vụ Lịch trình phải được chạy để sử dụng lệnh AT.
AT [\ \ computername] [[id] [/ DELETE] | / DELETE [/ YES]]
AT [\ \ computername] thời gian [/INTERACTIVE}
\ \ Computername Chỉ định một máy tính từ xa. Chỉ huy được lên kế hoạch trên
địa phương máy tính nếu tham số này bỏ qua.
id là một mã số được gán cho một lệnh theo lịch trình.
/ Xóa bỏ một lệnh theo lịch trình. Nếu id là bỏ qua, tất cả các dự lệnh trên máy tính
được hủy bỏ.
/ Có được sử dụng với lệnh hủy bỏ tất cả công việc khi không có xác nhận thêm là mong muốn.
Chỉ định thời gian thời gian khi lệnh là chạy.
/ Tương tác cho phép các công việc tương tác với máy tính để bàn của người dùng được đăng nhập vào tại thời điểm công việc
chạy.
Ví dụ:
lúc 06:50 "c: \ Winnt \ task.bat"

Thực hiện lệnh này sẽ chạy các tập tin task.bat lúc 06:50 ngày hôm nay, trừ khi nhiệm vụ theo kế hoạch sau khi thời gian quy định. Vì / tương tác chuyển đổi không có trong lệnh, hàng loạt tập tin này sẽ được thực hiện trong nền.

lúc 6:50 / tương tác c: \ Winnt \ beep.bat

Chạy tập tin batch beep.bat lúc 06:50 với khả năng cho người sử dụng tương tác với các tập tin thực thi.

Lệnh ATRRIB: Giới thiệu về attrib

Attrib cho phép người dùng thay đổi các thuộc tính của một tập tin chỉ định. Sử dụng attrib, người dùng có khả năng thay đổi các tập tin có bất kỳ các thuộc tính dưới đây. Lưu ý nếu bạn muốn thay đổi ACL của của một tập tin xem lệnh CACLS.

Chỉ đọc - cho phép các tập tin được chỉ xem và không ghi vào.

Lưu trữ - Microsoft cho phép sao lưu và chương trình sao lưu dự phòng khác để biết những tập tin sao lưu.

Thể - làm cho các tập tin vô hình cho người sử dụng tiêu chuẩn.

Hệ thống - làm cho các tập tin một tập tin hệ thống quan trọng.

Cú Pháp:
Hiển thị hoặc thay đổi thuộc tính tập tin.

ATTRIB [+ R |-R] [+ A |-A] [+ S |-S] [+ H |-H] [[ổ đĩa:] [path] filename] [/ S [/ D]]

+ Thiết lập một thuộc tính.
- Xóa một thuộc tính.
R chỉ đọc tập tin thuộc tính.
Một tập tin thuộc tính Archive.
S thuộc tính file hệ thống.
H Thể thuộc tính tập tin.
/ S quá trình tập tin trong tất cả các thư mục trong đường dẫn chỉ định.
/ D Quy trình thư mục là tốt.

Các ví dụ

attrib

Kiểu gõ attrib của chính nó sẽ hiển thị tất cả các file trong thư mục hiện hành và mỗi thuộc tính của họ. Nếu bất kỳ tập tin ẩn nó cũng sẽ hiển thị các tập tin.

autoexec.bat attrib + r

Thêm các thuộc tính chỉ đọc đến tập tin autoexec.bat vì vậy nó không thể được sửa đổi cho đến khi các thuộc tính chỉ đọc được lấy ra. Điều này là hữu ích cho các tập tin hệ thống quan trọng hay tập tin nào khác mà bạn không muốn có nhầm lẫn chỉnh sửa hoặc thay đổi bởi một chương trình khác.

attrib + h config.sys

Thêm các thuộc tính ẩn cho tập tin config.sys gây ra nó được không được nhìn thấy bởi người dùng trung bình.

attrib-h config.sys

Lệnh này sẽ làm ngược lại của lệnh ở trên. Thay vì ẩn các file nó sẽ thôi ẩn các tập tin nếu ẩn.

Lệnh Break: Giới thiệu về break

Break có thể được sử dụng để kích hoạt hoặc vô hiệu hóa khả năng phá vỡ của máy tính. Ví dụ, nếu một người sử dụng muốn hủy bỏ một tập tin batch hoặc một quy trình MS-DOS, mà người sử dụng chỉ đơn giản có thể nhấn Ctrl + C (break), mà sau đó sẽ nhắc nhở người dùng nếu họ muốn hủy bỏ quá trình hiện hành.

Turning break off sẽ hủy bỏ CTRL + C, tuy nhiên, người sử dụng vẫn sẽ có thể nhấn CTRL + PAUSE / BREAK và có khả năng nhận ra của một tập tin batch / hiện nay quá trình đang chạy.

Lưu ý: các phiên bản mới hơn của Windows (Windows ME, Windows 2000, Windows XP, và cao hơn) chỉ bao gồm các lệnh này cho tương thích ngược và biến break off không có tác dụng.

Cú pháp

Bộ hoặc xóa mở rộng CTRL + C kiểm tra.

BREAK [ON | OFF]

Loại BREAK mà không có một tham số để hiển thị các thiết lập hiện hành BREAK.

Các ví dụ

break ngày

Lượt về break, cho phép chức năng Ctrl + C.

break off

Tắt phá vỡ, không cho phép Ctrl + C để hủy bỏ một quá trình.

Lệnh xóa: Giới thiệu về xóa

Xoá bỏ là một lệnh được sử dụng để loại bỏ các tập tin từ ổ đĩa cứng của máy tính và hay phương tiện truyền thông có thể ghi khác.

Windows 2000 và Windows XP cú pháp

Xóa một hoặc nhiều tập tin.

DEL [/ P] [/ F] [/ S] [/ Q] [/ A [[:] thuộc tính]] tên
ERASE [/ P] [/ F] [/ S] [/ Q] [/ A [[:] thuộc tính]] tên

Chỉ định một danh sách tên của một hay nhiều tập tin hoặc thư mục. Các kí hiệu có thể được sử dụng để xóa các tập tin nhiều. Nếu một thư mục được chỉ định, tất cả các file trong thư mục sẽ bị xóa.
/ P Prompts để xác nhận trước khi xóa mỗi tập tin.
/ F quân xóa các tập tin chỉ đọc.
/ S Xoá quy định các tập tin từ tất cả các thư mục con.
/ Q Quiet mode, không hỏi nếu ok để xóa trên ký tự đại diện toàn cầu
/ Chọn Một tập tin để xóa dựa trên thuộc tính
thuộc tính R tập tin chỉ đọc các tập tin hệ thống S H Thể tập tin đã sẵn sàng cho một tập tin lưu trữ - Tiền tố có nghĩa là không

Nếu mở rộng của Command được kích hoạt và thay đổi DEL xóa như sau:

Ngữ nghĩa của việc chuyển đổi hiển thị S / được đảo ngược ở chỗ nó cho thấy bạn chỉ các tập tin mà được xóa, không những nó không thể tìm thấy.

Các ví dụ

Xóa c: \ windows \ *. FAK

Xoá bất kỳ tập tin nào kết thúc với phần mở rộng FAK.

Thông báo: Người dùng sẽ không nhận được một xác nhận dấu nhắc khi thực hiện lệnh này. Sau khi thực hiện, tất cả các file sẽ bị xóa trừ khi các tập tin có thuộc tính chỉ đọc và chuyển đổi F / không được sử dụng.

Lệnh Format: Thông tin về định dạng

Định dạng được sử dụng để xóa tất cả các thông tin tắt của một đĩa máy tính hoặc ổ đĩa cố định.

Microsoft Windows 2000 và Windows XP cú pháp

Các định dạng đĩa để sử dụng với Windows 2000.

FORMAT [khối lượng / FS: kiểu tập tin hệ thống] [/ V: tên ổ đĩa] [/ Q] [/ A: kích thước] [/ C] [/ X]
FORMAT khối lượng [/ V: tên ổ đĩa] [/ Q] [/ F: size]
FORMAT khối lượng [/ V: tên ổ đĩa] [/ Q] [/ 1] [/ 4]
FORMAT khối lượng [/ Q] [/ 1] [/ 4] [/ 8]

Ghi rõ khối lượng ký tự ổ đĩa (theo sau dấu hai chấm), điểm gắn kết, hoặc tên khối lượng.
/ FS: Ghi rõ hệ thống tập tin kiểu hệ thống tập tin (FAT, FAT32, hay NTFS).
/ V: Ghi rõ tên ỗ đĩa.
/ Q Thực hiện một định dạng nhanh chóng. ( không dùng cho đĩa mới)
/ C tập tin được tạo ra vào khối lượng mới sẽ được nén theo mặc định.
/ X lực lượng để tháo dỡ đầu tiên nếu cần thiết. Tất cả các xử lý để mở âm lượng sẽ không còn hợp lệ.
/ A: Overrides kích thước phân bổ đơn vị mặc định kích thước. các thiết lập mặc định được khuyến khích mạnh mẽ để sử dụng chung.
NTFS hỗ trợ 512, 1.024, 2.048, 4.096, 8.192, 16K, 32K, 64K.
FAT hỗ trợ 512, 1.024, 2.048, 4.096, 8.192, 16K, 32K, 64K, (128K, 256K cho ngành kích thước> 512 byte).
FAT32 hỗ trợ 512, 1.024, 2.048, 4.096, 8.192, 16K, 32K, 64K, (128K, 256K cho ngành kích thước> 512 byte).
Lưu ý rằng FAT và FAT32 hệ thống tập tin dưới đây áp đặt các hạn chế về số lượng các cụm trên một khối lượng:

FAT: Số cụm <= 65.526 FAT32: Số 65.526 <cụm ><268.435.446

Định dạng ngay lập tức sẽ ngừng chế biến, nếu nó quyết định rằng những yêu cầu trên không thể được đáp ứng bằng cách sử dụng kích thước cluster được chỉ định.

NTFS nén không được hỗ trợ cho các kích thước đơn vị cấp phát trên 4.096.

/ F: Chỉ định kích thước kích thước của đĩa mềm để định dạng (160,
180, 320, 360, 640, 720, 1,2, 1,23, 1,44, 2,88, hoặc 20,8).
/ T: các bản nhạc Ghi rõ số lượng đĩa nhạc mỗi bên.
/ N: thành phần Chỉ định số lĩnh vực cho mỗi theo dõi.
/ 1 định dạng của một phụ duy nhất của một đĩa mềm.
/ 4 định dạng của một 5,25-inch 360K đĩa mềm trong ổ đĩa mật độ cao.
/ 8 định dạng tám phần cho mỗi theo dõi.

Các ví dụ

Khi sử dụng lệnh định dạng, hãy nhớ tất cả các thông tin trên ổ đĩa bạn muốn định dạng sẽ được hoàn toàn bị xóa bỏ.

một định dạng:

Sẽ xóa tất cả các nội dung ra đĩa. Thường được sử dụng trên một đĩa mà không được định dạng hoặc trên một đĩa bạn muốn xóa.

một định dạng: / q

Nhanh chóng xóa bỏ tất cả các nội dung của một đĩa mềm. Thường được sử dụng để nhanh chóng xóa tất cả thông tin về đĩa này.

format c:

Điều này sẽ xoá tất cả các nội dung của C: ổ đĩa cứng. Nói cách khác, trừ khi bạn muốn xóa thông tin của tất cả các máy tính của bạn, lệnh này không nên được thực hiện trừ khi bạn đang lập kế hoạch để bắt đầu lại.
