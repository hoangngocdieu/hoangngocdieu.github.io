Định dạng phân vùng đó là FAT32 hay NTFS ạ ? 

Phân vùng của mình là FAT32. Mình đã dùng cách này rồi mà vẫn không được HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer

>> If there is no NoSecurityTab listing then this is not the problem.
>> Value Name: NoSecurityTab
>> Value Type: REG_DWORD
>> Value Data: 0 or 1
>> 0 = Security tab will be visible
>> 1 = Security tab will not be visible
