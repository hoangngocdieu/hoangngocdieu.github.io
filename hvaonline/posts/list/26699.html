Hehe, lại nói về mấy bác YM và Skype này. Tình hình là cho dù mấy bác chạy port 443 và 80. Nhưng firewall vẫn cấm được như thường.

Trên firewall có command như sau:
config firewall profile

edit "IM" <-- Profile IM appy vào policy trong Firewall

set log-spam enable

set log-web-ftgd-err enable

set ftp splice

set http scan bannedword exemptword rangeblock urlfilter

unset https

set imap fragmail

set pop3 fragmail

set smtp fragmail splice

set nntp no-content-summary

unset im

set comment "Deny chat"

set aim enable-inspect block-im inspect-anyport <-- cấm AIM

set aim enable-inspect block-im inspect-anyport <--- cấm em Yahoo Mess

set icq enable-inspect block-im inspect-anyport

set msn enable-inspect block-im

set p2p enable

set bittorrent block

set edonkey block

set gnutella block

set kazaa block

set skype block <--- Cấm em Skype luôn

set winny block


 


Nhưng thực sự mình cũng chưa rõ, nó làm cách nào để cấm được mấy thằng này. Anh em có ý kiến gì không?
