1/ Loại forum backup : VBB 3.5.4 nguyên thủy ko có hack mod gì hết

2/ Em backup = chức năng có sẵn trên cpanel

3/ Sang host mới em cũng restore = chức năng có sẵn trên cpanel

------> restore thành công
--> tạo user mới
--> add user đó vào DB vừa restore

---> vào file config.php ( trong thư mục includes ) sửa lại tất cả những thông tin của SQL mới ( userDB,pasDB,DBname )
---> upload đè lên file cũ trên host

Lỗi của bạn là do thời gian backup vượt quá 300 giây cho phép.

Để chỉnh sửa điều này, bạn thử dùng phpMyBackupPro và chỉnh lại trong phần cấu hình của nó thời gian thực hiện dài hơn, có thể là 3000 giây chẳng hạn.

Ngoài ra, bạn có thể chia nhỏ cơ sở dữ liệu của mình ra thành nhiều file nhỏ hơn để thời gian backup từng file nhỏ hơn 300 giây cho phép.
