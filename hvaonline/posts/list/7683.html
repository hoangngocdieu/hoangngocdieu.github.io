Bạn đọc lại câu trả lời của mình, mình đã nói rõ đến thế còn gì nữa. Nhắc lại nhé: nếu dùng trên Win thì bạn copy file .dll đó vào thư mục extension trong thư mục php của bạn, sau đó thêm 1 dòng vào file php.ini, ví dụ:
Code:
extension=php_celedar.dll

Sau đó restart lại server thì có thể dùng được ròi.
Còn chuyện dùng những function của library đó như thế nào thì mình nghĩ không ai có thể trả lời giúp bạn được )
