Cả 2 tuần nay tôi muốn viết về cái console (giao tiếp) này, nhưng chẳng có thời gian (+1 chút hứng thú ) ). Không ngờ hứng thú lại đến vào cái lúc oái ăm như thế này: 1h30 am... Bắt đầu gõ vậy.

Microsoft Management Console (viết tắt là MMC) là một trình tiện ích giúp bạn quản lý mạng, PC, các dịch vụ và những thành phần khác.

Khởi động MMC:
+ Tại cửa sổ Run gõ "MMC"
+ Tại dấu nhăc của Command Prompt gõ MMC
+ Hoặc có thể chạy trực tiếp file mmc.exe trong thư mục \\WINDOWS\system32\

Thêm một mục mới vào MMC:
Trên menu File click vào Add/Remove Snap-in, click Add và chọn mục cần thiết. Đến đây bạn có thể chọn máy bạn cần quản lý: Local Computer (máy bạn đang dùng) hoặc một máy khác trên mạng (Đòi hỏi có User Admin).

Lưu ý:
Bạn có thể sao lưu Console hiện tại để mở ra lúc cần thiết, tránh trường hợp phải add các items trong mỗi lần sử dụng.

Thiết lập Console:
Trong mục File\Option cho phép bạn lựa chọn chế độ quản lý cho lần khởi động MMC thứ 2:
Author mode: Nói ngắn gọn là quản lý toàn bộ, và bạn có toàn quyền
User mode full access: Có toàn quyền đối với các snap-in đã được cung cấp và không cho phép add hoặc remove các snap-in mới.
User mode limited access, multiple window Người sử dụng có thể tác động đến console tree hiện hữu, có thêm các cửa sổ mới nhưng ko thể đóng cửa sổ hiện tại.
User mode limited access, single window ... ko thể mở các cửa sổ mới.

Chi tiết:
MMC là console quản lý chung, vì thế bạn có thể add rất nhiều Snap-in. Số lượng snap-in phụ thuộc vào các trình tiện ích đã được cài đặt. Một số tiện ích sắn có của Windows:
Certificate
Computer management
Device manager
Disk defragmenter
Disk managerment
Event Viewer
Folder
Local Computer Policy
......

Các phần về Computer managerment, Event viewer,... các bạn có thể đọc tại http://hvaforum.net/hvaonline/posts/list/3542.html

Phần còn lại tôi muốn viết về Local Computer Policy - vì nó cho phép chỉnh sửa một số chức năng của Windows - rất hữu ích trong việc quản lý PC của mình.

Bạn có thể khởi động Computer policy trong menu Administrative tool ---> Group Policy, hoặc acc Local Computer Policy (LCP) vào MMC.
Các mục con của LCP nằm ở bên trái cửa sổ gồm:
1. Computer configuration
2. User configuration

Sự khác biệt giữa 2 phần này: Computer configuration thiết lập các thay đổi chung cho tất cả user, start up, shut down... Còn User configuration thiết lập các thay đổi cho User hiện tại, log on, log off...

Ta sẽ xem qua về Computer configuration và một số thay đổi thông dụng. Khi đã làm quen với console rồi, bạn sẽ tìm hiểu thêm các thay đổi còn lại.

Bắt đầu với Computer configuration\Windows settings\. Ở đây có 2 mục nhỏ:
1. Script: cho phép bạn add 1 script trong quá trình khởi động cũng như tắt máy.
2. Security Settings: cho phép thiết lập bảo mật hệ thống.

2.1 Account Policies: thiết lập các thông số của các account:
2.1.1 Password policies: các thiết lập về password: độ dài, lưu password trong history,...
Bạn có thể thay đổi thiết lập rất đơn giản bằng cách click vào mục cần thay đổi và chọn thông số phù hợp.

2.1.2 Account lockout policy: Thiết lập các thông số quy định lockout các account sau 1 số lần log on không chính xác, xác định thời gian lockout....

2.2 Local Policies:
2.3 Public key policies:
.....
-)) Phần này nhiều quá lại hơi phức tạp trong vấn đề diễn đạt nên có lẽ các bạn tự tìm hiểu thì tốt hơn.

Tôi sẽ đưa ra 1 số setting hữu ích để thấy được tiện dụng của Computer policy, và cũng để minh họa cho bài viết.

Remove Folder option: User configuration ---> administrative templates ---> Windows Components ----> Windows Explorer ----> Remove the Folder option from... double click bạn sẽ có lựa chọn Not config, Enable hoặc Disable. Xem thêm tab Explain để lựa chọn thích hợp.

Không cho phép thay đổi Home page của IE: User configuration ---> administrative templates ---> internet Explorer ---> Disable changing home page settings---->Not config, Enable hoặc Disable

Tắt Task manager: User configuration ---> administrative templates ----> System ---> Ctrl + Alt + Del option ---> Remove Task manager

) Có lẽ chừng này cũng có thể thấy sơ sơ sự tiện lợi của Computer Policy. Có thể console giúp bạn quét được ối con virus Yahoo trong thời gian gần đây.
Tôi có "cảm tình" với console MMC vì nó rất trực quan, chịu khó đọc, tìm hiểu tí là bạn có thể có những thiết lập như mong muốn.

Xin mời các bro đóng góp thêm ý kiến về MMC và Group Policies để cho nó phong phú + hữu ích hơn.
