1. Thứ nhất bạn không nói rõ về User mà bạn đang log on: đây là account admin do bạn tạo ra hay là account admin mặc định của Windows. Thông thường thì sau khi cài đặt Windows người ta thường đổi tên account admin mặc định của Windows, set pass cho acc đó và ghi nhớ, không cho người khác biết, đồng thời thiết lập policy để không cho hiện thị user name log on lần cuối. Mục đích của việc này dùng để làm gì? Nó dùng khi bạn thiết lập GP quá chặt chẽ, ví dụ khoá account sau mấy h mới được enable lại chẳng hạn v..v.., khi đó có một người nào đó cố tình nghịch máy của bạn và acc admin do bạn tạo ra bị lock, lúc đó bạn sẽ thấy sự cần thiết khi phải làm như tôi nói ở trên.

2. Theo như tôi biết bạn đã dùng tài khoản admin để set Policy và khoá hết các chương trình. Việc dùng admin để set Policy nếu tôi không nhầm sẽ ảnh hưởng lên toàn bộ hệ thống bao gồm các account khác vì đây chính là việc bạn set "Local Computer Policy". Do đó trong trường hợp của bạn theo tôi nghĩ là có lẽ nên chào thua, không biết khởi động bằng Safemode như bạn gì gợi ý có ăn thua gì không. Nhưng theo tôi nghĩ là không thể!!

3.Để tạo một user có quyền Admin bạn có thể sử dụng cách sau bằng command line trong Windows :

a. Tạo một user :
c:\net user kienmanowar 123456 /add

b. Add user này vào nhóm Administrators :
c:\net localgroup Administrators kienmanowar /add

Vậy là bạn đã có một acc có quyển Admin rồi đó.

kienmanowar

Mình có cách này bạn có thể làm thử, đảm bảo okie
- Đầu tiên boot vào DOS bằng Hiren's BootCD
- Ở đây mình sẽ sử dụng chương trình "hạt tiêu" REG có sẵn ở \Windows\System32\ để xoá key trong Registry
+ Để tránh xoá nhầm các key khác, mình nên xem trước key cần xoá cho chắc ăn, bạn gõ câu lệnh bên dưới vào:
REG QUERY HKCU\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer /s
Đây rồi, key cần xoá là thằng này đây: RestrictRun
+ Bắt đầu xoá:
REG DELETE HKCU\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer /v RestrictRun
Câu thông báo Delete the regitry value RestrictRun (Yes/No)? xuất hiện hỏi bạn có thật sự xoá hay không, nhấn Y thế là okie. Giờ khởi động lại Windows xem thế nào, hehe...

1,Start =>Run => go secpol.msc de chay Local Security Settings
2,Local Policies => Security Options
3,Tim đến "Recovery console:Allow floppy copy and access to all driver and folders" thiết lập disabled thành enabled
4,Log Off xong bắ đầu nghịch.
