Nếu bạn đã VPN thành công, nhưng khi VPN xong thì ra net không được. Cái này là do khi VPN đã được kết nối nó sẽ lấy kết nối VPN làm Default Gateway đó. Bạn có thể bỏ chế độ này bằng cách vào Properties của kết nối VPN, chọn TCP/IP, trong phần advan.. nó có check vào dòng là Use Conenction.... Default Gateway.
Bỏ chọn nó thì bạn ra net vô tư. Chúc bạn thành công.

Bạn làm VPN thông qua ADSL hả? Không có dòng default gateway thì sao được. Máy bạn phải hướng default gateway về modem ( ở đây là: 192.168.1.1) phải đảm bảo Ip Publish lúc test là phải trùng nhau. Bạn phải config Modem cho nó NAT vào máy của bạn. Phải đảm bảo những thông số khi config VPN( Máy ngoài) là phải đúng. Đặc biệt là IP. DHCP ở modem thì bạn phải enable - còn ở máy bạn thì đừng config gì đến DHCP. DHCP là 1 Tool dùng để cấp phát IP động. Bạn không enable nó ở modem thì làm sao ISP cung cấp IP cho bạn được! Đó là những gì mình biết. Có gì mong chỉ bảo thêm.
