Xin chào mọi người
- Nobita có sử dụng dòng lệnh trong CMD để hiện các file và folder ẩn (do bị virut, và đã diệt xong):
attrib -s -h C:\* /S /D 

Nhưng nobita chỉ muốn hiện các file và folder ẩn trong thư mục hiện tại mà thôi, cụ thể ở đây là tất cả các file và folder trong ổ C, nhưng câu lệnh trên sẽ thực hiện trong tất cả các subfolder nên sẽ rất lâu, nobita không muốn thế. Vậy nobita phải làm thể nào, hay là kết hợp 2 lệnh attrib + for ?(cái này nobita cũng ko rành lắm)
- Cho nobita hỏi thêm, nobita vẫn sử dụng câu lệnh trên đối với ổ D:
attrib -s -h D:\* /S /D 

Nhưng nó thực hiện được tất cả các folder phía trên thư mục System Volume Information nhưng riêng folder System Volume Information thì nó báo là
Access denied - D:\System Volume Information 

và nó ngừng lại đồng thời các folder và file phía sau cũng ngưng không thực hiện luôn
- Cho nobita hỏi câu cuối nữa nghen: Bây giờ do máy đã bị virut phá tất cả các file hệ thống của Window, nên nobita muốn repair lại tất cả các file đó, và nobita đã sử dụng đĩa Windows XP SP2 cho vào ổ đĩa và khởi động lại vào chế độ Repair. Vậy nobita phải sử dụng câu lệnh nào để thay thế toàn bộ file hệ thống trên.
PS: Tất nhiên là không sử dụng phần mềm để thực hiện. Và nobita không muốn thực hiện câu lệnh trên cho từng file và folder vì giả sử ổ đĩa nhiều dữ liệu mà thực hiện từng cái chắc tiêu.

Try this if your Windows XP Pro is using the NTFS File System

1.Click Start, and then click My Computer.
2.On the Tools menu, click Folder Options.
3.On the View tab, click Show hidden files and folders.
4.Clear 'Hide protected OS files (Recommended)' check box. Click Yes when you are prompted to confirm the change.
5.Click OK.
6.Right-click the System Volume Information folder in the root folder, and then click Sharing and Security.
7.Click the Security tab.
8.Click Add, then type the name of the user to whom you want to give access to the folder. Choose acct location if required (either local or from the domain). Typically, this is the account with which you are logged on. Click OK, and then click OK.
9.Double-click the System Volume Information folder in the root folder to open it
