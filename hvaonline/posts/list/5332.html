Bạn vào Run, gõ gpedit.msc để mở Group Policy.
Chọn User Configuration->Windows Explore trong Left Panel
Click phải lên mục Removes the Folder Options menu item from the Tools menu, chọn Properties, sau đó chọn Disable.
Vào Run, gõ gpupdate /forge để cập nhật các thay đổi.

Các bước trên là để mở Folder Option trong trình đơn Tools của Windows Explore đã bị virus giấu đi. Giờ thì bạn có thể vào Folder Option để chọn xem các file ẩn

Bạn kiểm tra lại xem cái này còn không hay bị delete rồi

HKEY_CURRENT_USER\software\microsoft\windows\currentversion\explorer\advanced

Bạn hãy vào START --> RUN , gõ vào REGEDIT rồi chọn OK. Một cửa sổ REGISTRY EDITION sẽ hiện ra. Hãy tìm đến khóa : HKEY CURENT USER/Microsoft/Windows/Current Version/Policies/Explorer. qua cửa sổ bên phải, right click -->New-->Dword Value , điều chỉnh giá trị của khóa này là "0"

Tiếp theo vào : HKEY LOCAL MACHINE/Microsoft/Windows/Current Version/Policies/Explorer , làm tương tự như trên
