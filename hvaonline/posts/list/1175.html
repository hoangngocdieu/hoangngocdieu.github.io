Bạn muốn trình duyệt hỗ trợ công cụ download file Flash luôn thì dùng FireFox đi, hiện đã có bản Firefox 1.5.0.4 rồi đó ( http://www.mozilla.com/products/download.html?product=firefox-1.5.0.4&os=win&lang=en-US)


Sau khi cài đặt xong, bạn chạy trình duyệt Firefox và tải thêm Download Embedded, đây là một Add-ons cho FireFox (hiện đã có version 0.4). Add-ons này cho phép bạn xem và download các embedded được nhúng vào trang Web như movies, mp3s, flash, quicktime, hoặc là các file embedded khác nữa... Download ở đây: http://releases.mozilla.org/pub/mozilla.org/extensions/download_embedded/download_embedded-0.4-fx.xpi
Cài đặt xong, khởi động lại trình duyệt Firefox và mở một trang web có file Flash thử xem, khi đó ở góc bên phải, cuối trình duyệt Firefox sẽ có biểu tượng sau:
