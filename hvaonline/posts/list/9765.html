Bạn muốn tắt USB ở chế độ an toàn (safe mode) nhưng windows không cho phép. Bạn làm theo 2 cách sau:

1. Log off windows sau đó log on trở vào và tiến hành tắt USB ở chế độ an toàn.
2. Không cần log off, click chuột phải lên USB chọn Eject. Sau đó rút USB ra. Cách này lúc được lúc không.

Vírus nếu có trong USB, nó phải tồn tại dưới môt hay nhiều file ( visible hay hiden). Khi virus hoạt động trong hệ thống, thì được hiểu là file của virus cũng đang open.
Đôi khi hệ thống file ("file system") trong USB bị corruption thi OS cũng không cho rút USB ra ( trừong hợp này dùng trình Erro-check trong windows để sửa lại hoặc dùng trình CHKDISK trong Hiren Boot để sửa lại.)

Khi tắt máy(Shutdown) xong có thể rút USB ra khỏi máy an toàn. Nhưng tốt nhất là Restart để Windows tự động chạy CHKDISK trong quá trình RESTART và fix lại các error file system trong USB nếu ta nghi ngờ file system của USB bị hỏng tại một vài sector nào đó.

Trong USB hệ thống File luôn là FAT
Hỏng File system hoàn toàn khác với trường hợp hỏng các file của hệ thốnghay của USB
