- Trước tiên, bạn tải về file SP3.torrent (dung lượng 15KB) từ địa chỉ http://tinyurl.com/27pf2j hay http://tinyurl.com/2dh3q9.

- Tiếp đó, bạn sử dụng một chương trình chuyên dụng tải file torrent như FlashGet, uTorrent hay BitComet mở file này và bắt đầu download toàn bộ dữ liệu chứa trong nó (tổng dung lượng khoảng 339,56MB bao gồm một file wb-xpsp3b3180.rar, một file wb-xpsp3b3180.sfv và 24 file dạng wb-xpsp3b3180.r00, wb-xpsp3b3180.r01... cho đến wb-xpsp3b3180.r22). Tại thời điểm này, số lượng seeder là khá đông nên tốc độ tải file cũng tương đối nhanh.

- Khi quá trình tải dữ liệu hoàn tất, bạn chạy file wb-xpsp3b3180.rar để nối tất cả file lại thành một file windowsxp-kb936929-sp3-x86-enu.exe có dung lượng 332MB.

- Kích hoạt file này để bắt đầu nâng cấp Windows XP SP2 lên SP3, nhưng lưu ý ổ đĩa cài đặt Windows phải còn trống ít nhất 1.1GB và trước đó bạn nên sao lưu hệ thống cẩn thận nhằm đề phòng bất trắc.

- Quá trình cài đặt diễn ra khá lâu nên bạn hãy kiên nhẫn chờ đợi. Lúc hoàn tất, bạn khởi động lại máy để thay đổi có hiệu lực.

- Khi vào lại Windows, mở hộp thoại System Properties (dùng tổ hợp phím Windows + Pause/Break), bạn sẽ thấy “mác” Service Pack 3 đã lộ diện.
