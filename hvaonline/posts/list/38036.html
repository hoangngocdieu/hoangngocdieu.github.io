Mọi người cho mình hỏi?
Mình mới cài lại win 7, kiểm tra dung lượng thì thấy nó lên tới 30 GB ( trong khi mình chưa cài đăt soft gì nặng ngoài cài win). Mình muốn xoá bớt file ko cần thiết để cho bớt dụng lượng ổ C. Lưu ý là máy mình ko có virut đâu, cái này mình khá chắc.
Mình đoán là do file windows.old tạo thành khi mình cài mới lại win nên mới chiếm nhiều dung lượng vậy. Nhưng khi vào phân vùng win tức là ổ C thì chẳng thấy file win...old đó đâu cả mặc dù mình đã cài chế độ hiện file ẩn cũng chẳng ăn thua.
Mình vào ổ C rồi chọn hết tất cả các file rùi tỉnh tổng dung lựong là bao nhiêu thì chỉ mất 13GB, nhưng khi kiểm tra dung luong trên ổ C thì nó lại là 30GB. Thế là sao nhỉ rõ là chỉ có 13GB thui mà, còn gần 20GB kia nó ẩn núp đâu vậy sao mình ko thấy, ko hiển thị. Mọi người người giúp mình nên làm sao đây???

Mình xài winrar rùi vào ổ C thì phát hiện có một số file ẩn, nhưng ko thấy file windows.old đâu, đó là:
Các file: pagefile.sys + hiberfil.sys mỗi cái nặng hơn 1 GB
Và vài thư mục như Recovery và Documents and Settings, System Volume Information
Mình đoán có thể 20GB đó nằm trong thư mục Documents and Settings nhưng ko mở đc, ko chỉ vào để kiểm tra dung lượng đc.

Noi chung là mình ko biết làm sao xoá bớt dung lượng thừa trong ổ C đi cả - đã cài sof Glary Utilities Pro xoá file rác cũng ko đáng kể.
Xin hay cho mình ý kiến phải làm sao, xin cảm ơn !

Để hiển thị các file hệ thống bị ẩn, bạn bỏ tích trong mục "Hide protected operating system file (Recommanded)"
trong Folder options --> View. Nếu thấy file Windows.old thì bạn xoá nó đi còn không bạn dùng cách:
Chuột phải vào ổ C: --> chọn properties --> Disk Cleanup --> bấm nút Cleanup System File --> đánh dấu tích vào mục Previous Windows Installation(s) rồi nhấn OK. Đợi trong chút lát để chương trình chạy là xong.

Sau đó bạn tắt chức năng Hibernate trong Win bằng cách:
Vào search --> cmd --> chuột phải vào cmd chọn Run as administrator
trong cmd gõ thật chính xác lệnh powercfg -h off rồi nhấn Enter (sau khi tắt Hibernate bạn đã loại bỏ được file hiberfil.sys khá nặng rồi.
trong trường hợp bạn muốn bật lại chức năng bạn chỉ cần gõ lệnh powercfg /hibernate on.

- Giả sử bạn mua USB dung lượng 4GB (=4096MB) chẳng hạn, cắm vào PC và kiểm tra dung lượng thấy mất đi một phần nào đó -> nhiều người tưởng rằng mình mua USB lởm nhưng hoàn toàn sai, vì sao sai? tự giải thích.

- Hay cũng giống như rất nhiều bạn hỏi tại sao tôi mua ổ cứng 80GB (=81920MB) thế mà khi kiểm tra thấy thiếu và lại cho rằng mình mua ổ cứng rởm song nghĩ như vậy cũng sai nốt, vì sao sai? tự giải thích.

- Và khi bạn cài bất kì một chương trình nào trên PC, điều đầu tiên là phải đọc những yêu cầu về hệ thống của chương trình ấy trước, trước khi cài nó vào máy. Và một trong những "system requirements" là không gian trống của đĩa cứng tối thiểu dành để cài đặt cho nó.

* Còn trên windows 7 có một số folder ẩn (Documents and Settings, System Volume Information ...), vì lý do bảo mật nên Microsoft giới hạn quyền user truy cập vào những folder đó. Nếu bạn thực sự thấy cần thiết truy cập vào Documents and Settings thì bạn hãy sử dụng tab Security thuộc Properties trên folder đó để Take ownership nó là có thể vào được.
