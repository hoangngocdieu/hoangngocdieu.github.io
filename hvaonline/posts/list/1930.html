C:\Documents and Settings\Administrator\ntuser.dat

Cái file này có lé như thế này: Nó làm cho em khi chat YM khi click vào 1 link thì nó không mở link mà nó chạy vào C:\Documents and Settings\Administrator nên em mới tìm cách xoá.

Khi xoá thì nó lại bảo như sau:

Can't delete ntuser: It is being used by another program or person. Close any program that might be using the file and try again.

Nhung chắc chắn là em đã close tất cả rồi và không có read only file nữa.
