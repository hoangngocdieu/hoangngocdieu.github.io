Hi Các bạn,

Server IBM của công ty đang chạy HDH Windows Server 2003 Ent, bị lỗi này rất lạ.
- Mỗi khi làm việc xong trên Sever mình hay nhấn tổ hợp phím Ctrl+ Alt + Del, Sáng nay mình vào bật màn hình lên thì thấy xuất hiện màn hinh màu xanh xám. Không có hiện lên chế để cho mình login. Mình vẫn rê chuột được.
- Mình đã kiểm tra nguồn vào cable màn hình vẫn ok. sau đó mình dùng máy mình remote vào server rồi restart lại Server vẫn tình trạng như cũ. Màn hình vẫn là màu xám xanh.
- Có điều rất lạ là các client vẫn truy cập vào data trên Server bình thường. Và mình remote trở lại thì báo có nhiều kết nối không thể remote vào được.
- Firewall fortinet và router FPT không thể vào được để kiểm tra (错误
您所请求的网址（URL）无法获取

--------------------------------------------------------------------------------

(60) Operation timed outThe remote host or network may be down. Please try the request again.
