panfider wrote:
python hình như hơn hẳn shell script thì phải ? 


Không thể so sánh Python với shell script có sẵn trên *nix được bởi vì chúng được tạo ra ở hoàn cảnh khác nhau, thời điểm khác nhau và cho mục đích khác nhau. Xét về khía cạnh kỹ thuật, shell script chỉ là scripting đơn giản và hoàn toàn không có những đặc tính mà Python có như: Object-oriented, modular và portable. Chỉ riêng ở khía cạnh OO, Python đã là một công cụ nằm ở biên độ hoàn toàn khác với shell script rồi.

@ Jino_Hoang: Python được "port" sang Symbian nhưng nó không phải là ngôn ngữ lập trình chủ lực cho các ứng dụng trên Symbian. Thật ra, nếu dùng Python làm ngôn ngữ lập trình cho ứng dụng trên PC vẫn có thể sống được. Python hay cái gì cũng chỉ là công cụ lập trình. Cái quan trọng là ý tưởng và cách ứng dụng ý tưởng mà thôi.

Nếu nói Python là ngôn ngữ ưa thích của hacker cũng không sai đâu vì Python rất powerful (tính powerful của nó thế nào thì chỉ có thể nhúng tay vô thử dăm ba ngôn ngữ tương tự như Perl, Ruby... để so sánh).


@ all: cá nhân tớ nhận thấy Python có phần "string manipulation" không bằng Perl nhưng tính gọn nhẹ trong syntax, đặc biệt trong phần conditional và looping thì Python hơn hẳn Perl.

Nên biết rằng Google dùng Python rất nhiều trong software engineering của họ.
