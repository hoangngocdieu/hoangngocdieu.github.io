Tốt nhất bạn nên xóa IIS rồi cài lại, bạn nên khởi động tất cả services mà tôi đã chỉ ở trên để IIS chạy được (tất nhiên bạn có thể tắt các services không cần thiết, nhưng để IIS chạy được bạn nên chọn Enable All)

Bước 1: Xóa IIS, start/ settings/ control panel/ add or romove programs/ chọn "add/remove program windows components" ở bên trái/ hủy dấu check tại "internet information services (IIS)"/ chọn next......

Bước 2: Khởi động lại máy

Bước 3: cài lại ISS, start/ settings/ control panel/ add or romove programs/ chọn "add/remove program windows components" ở bên trái/ đánh dấu chọn tại "internet information services (IIS)"/ bỏ đĩa CD XP vào / chọn next....
