Trước đây lão Bin có 1 bài này rồi, move qua forum mới hình như không còn giữ đc.
Bài này giới thiệu cho bà con cách cài đặt windows tự động mà không cần nhập các thông tin lúc cài đặt như: CDKey,ngày giờ, tên đăng nhập.v.v.v và mấy lần click Next. Chỉ cần nhét đĩa boot vào nó sẽ chạy đến lúc cài đặt xong.
Tạo 1 tên file winnt.sif có nội dung như sau:
Code:
;SetupMgrTag
[Data]
    AutoPartition=1
    MsDosInitiated="0"
    UnattendedInstall="Yes"
[Unattended]
    AutoActivate="no"
    CrashDumpSetting="0"
    DisableDynamicUpdates=Yes
    DriverSigningPolicy="Ignore"
    FileSystem=NTFS
    Hibernation=0
    UnattendMode=FullUnattended
    OemSkipEula=Yes
    OemPreinstall=No
    Repartition=Yes
    TargetPath=\WINDOWS
    UnattendSwitch="yes"
[GuiUnattended]
    AdminPassword=*
    EncryptedAdminPassword=NO
    OEMSkipRegional=1
    TimeZone=20
    OemSkipWelcome=1
[UserData]
    ProductKey=JJWKH-7M9R8-26VM4-FX8CC-GDPD8
    FullName="User"
    OrgName="MyOwn"
    ComputerName=*
[TapiLocation]
    Dialing=Tone
    AreaCode=844
[Branding]
    BrandIEUsingUnattended=Yes
[Proxy]
    Proxy_Enable=0
    Use_Same_Proxy=1
[Identification]
    JoinWorkgroup=WORKGROUP
[Networking]
    InstallDefaultComponents=Yes


Tiếp đến tạo file winnt.bat có nội dung:
Code:
rem SetupMgrTag
@echo off
rem
set AnswerFile=.\Winnt.sif
set SetupFiles=E:\i386   		   
E:\i386\winnt32 /s:%SetupFiles% /unattend:%AnswerFile%


Sau đó bạn copy 2 file này vào thư mục i386. Nếu cài đặt từ DOS bạn chỉ cần chạy file winnt.bat (chú ý gõ đầy đủ cả .bat vì trong thư mục i386 còn có 1 file là winnt.exe)

Còn nếu bạn muốn boot từ CD thì cần có 1 file img/iso của WinXP, dùng công cụ như WinISO để copy 2 file trên vào thư mục i386
Thay file autod.bat ngoài thư mục root bằng file autod.bat có nội dung như sau:
Code:
i386\winnt.bat /s:%SetupFiles% /unattend:%AnswerFile%


OK, xong. Bạn burn file img/iso đó ra CD rồi cài đặt. Bây giờ bỏ đi uống cafe rồi quay về chắc máy cài xong rùi )

Chúc bà con vui vẻ.
