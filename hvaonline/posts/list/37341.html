Bạn đã kiểm tra xem GNOME và X11 đã được cài đặt vào hệ thống chưa?
Nếu rồi (GNOME + X11) thì kiểm tra xem đã có Display Manager nào được cài đặt rồi?
Một số display manager: gdm (của GNOME), kdm, xdm, slim.
Nếu đã có GNOME + X11 + Display Manager thì chạy cái Display Manager từ dòng lệnh (thường là với quyền root).

Quay trở lại topic của bạn: Để lấy thông tin từ /var/log/Xorg.0.log thì bạn có thể xem qua phần đầu file, nó có mô tả các kí hiệu quy ước., ví dụ:
(EE) - Lỗi
(WW) - Cảnh báo

Ví dụ để xem xét trong file log nói trên có bao nhiêu lỗi, bao nhiêu cảnh báo thì dùng
Code:
$ cat  /var/log/Xorg.0.log | grep -e "(EE)" -e "(WW)"


Còn nếu bạn chưa quen làm việc với file log, thì hãy post nó lên pastebin rồi dẫn link vào đây. Đừng post vào diễn đàn.
