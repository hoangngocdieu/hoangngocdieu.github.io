Bài viết được lấy từ vniss.net, tác giả necromancer có bổ xung thêm.

Bài viết này mục đích dùng chương trình Resource Hacker để đọc 1 số file hệ thống của windows, và chỉnh sửa 1 phần giao diện của nó.

Công cụ: Resource Hacker (download tại http://users.on.net/johnson/resourcehacker)

Các vị trí chỉnh sửa bao gồm nút Start, dòng chữ welcome khi khởi động máy và màn hình khi đăng nhập. Mình chỉ xin nêu chi tiết chỉnh sửa nút Start, còn các chỉnh sửa khác hoàn toàn tương tự.

1. Chỉnh sửa nút Start:

Chú ý: nên copy file C:\WINDOWS\explorer.exe (có thể khác nếu bạn cài win ở ổ khác) sang 1 vị trí khác để thực hành, đề phòng khi có sự cố có thể hồi phục lại được.

Chạy chương trình Resource Hacker, nhấn Ctrl-O để mở file, chọn đến file explorer.exe đã được copy đó.
Sau khi được nạp lên rồi, ở phía bên trái, tìm đến String Table\37\1033, khi đó ở cửa sổ bên phải sẽ hiện ra một loạt các chỉ mục. Ở chỉ mục số 578, bạn sẽ thấy "start".
Bây giờ muốn thay đổi nút start thành chữ gì chỉ việc thay dòng "start" đó bằng dòng chữ bạn muốn, ví dụ "bat dau".
Nhấn vào nút Compile để biên dịch và Ctrl-S để ghi lại các chỉnh sửa ở file đó. Thoát khỏi Resource Hacker.
Đổi tên file vừa được chỉnh sửa xong (ví dụ thành myexplorer.exe, chú ý lúc này sẽ có 2 file explorer.exe và explorer_orginal.exe, file cần đổi tên là file explorer.exe) và copy nó trở lại thư mục C:\WINDOWS.

Sao lưu lại registry để đề phòng sự cố.
Sau đó bạn mở registry lên, tìm đến khoá HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsNT\CurrentVersion\Winlogon (khoá shell), và chỉnh giá trị của nó thành "myexplorer.exe".
Khởi động lại máy tính, bạn sẽ có 1 nút "bat dau" mới.
Khi nào bạn không muốn thay đổi, muốn trở lại như cũ chỉ cần vào registry, chỉnh sửa lại giá trị của khoá trên thành "explorer.exe" là xong.

Chú ý: dòng chữ được thay chỉ ở dạng ascii, không gõ tiếng việt có dấu được.

Hơn nữa, bạn cũng có thể chỉnh sửa dòng chữ "Click here to begin" thành dòng chữ gì tuỳ ý bằng cách tìm đến vị trí String Table\34\1033 trong Resource Hacker.

2. Chỉnh sửa nút welcome:

Cách làm tương tự như ở trên, tuy nhiên file cần chỉnh sửa là C:\WINDOWS\system32\logonui.exe.
Khi nạp bằng Resource Hacker tìm đến vị trí String Table\1\1033 thực hiện các thay đổi ở đó. Compile, Save, đổi tên và copy ngược trở lại thư mục C:\WINDOWS\system32.
Sau đó vào registry, thay đổi giá trị của khoá HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsNT\CurrentVersion\Winlogon (khoá UIHost) thành tên file mà bạn đã chỉnh sửa.

Các thay đổi dưới đây là của màn hình khi đăng nhập (khoá ở registry không thay đổi, vẫn là HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsNT\CurrentVersion\Winlogon (khoá UIHost))
- chỉnh dòng chữ "Turn off computer": String Table\1\1033
- chỉnh dòng chữ "After you log on, you can add or change accounts.\nJust go to Control Panel and click User Accounts.": String Table\2\1033
- chỉnh dòng chữ "To begin, click your user name": String Table\2\1033

Chú ý quan trọng:
1. Copy file cần chỉnh sửa sang 1 vị trí khác, đề phòng thất bại có thể sửa lại được.
2. Sau khi chỉnh sửa, nhớ đổi tên và copy ngược trở lại thư mục nguồn.
3. Sao lưu registry.
4. Khi xảy ra sự cố (hoặc muốn trở lại như cũ), hãy vào win bằng chế độ Safemode và chỉnh lại các khoá ở registry như ban đầu.

Còn có 1 số vị trí trong các file này mà bạn có thể khám phá thêm, ví dụ như có thể thay đổi được các nút bấm, các biểu tượng, hình ảnh khi đăng nhập...

mình đâu có nói chỉ có Resource Hacker làm được đâu. nó là đọc file exe và giúp mình tìm nhanh được đến chỗ cần đồi thôi. có thể dùng các chương trình chỉnh sửa file exe khác để làm việc này.
tuy nhiên cần chú ý là nếu chỉnh sửa trực tiếp trên các file exe thì độ dài các chuỗi thay thể thường không được vượt quá độ dài chuỗi ban đầu.

còn để chỉnh sửa nút ấn, hình ảnh, ví dụ hình biểu tượng windows ở màn hình đăng nhập, bạn cũng mở file logonui.exe, chọn phần Bitmap\127, nhấp chuột phải vào đó và chọn Change resource (có thể tên khác, vì mình đang dùng bản Resource Hacker tiếng Nga), khi đó sẽ hiện ra bảng danh sách các hình ảnh.
chỉ cần chỉ ra file có chứa hình ảnh (file này phải có đuôi exe, dll, res, br) và chọn 1 trong các hình ảnh chứa trong đó. chú ý đến kích thước của ảnh nên để bằng với kích thước ảnh gốc.
sau đó compile và save, rồi làm tương tự là được.
