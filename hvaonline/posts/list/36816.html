Khi truy cập vào IP public của modem, mà bạn được yêu cầu nhập username, password của modem có nghĩa là:

1. Bạn đang enable internet access control (hay remote router access), nên tắt tính năng này đi (cấu hình trong phần Adminnistration hoặc Access control, hay security của modem).

2. Bạn chưa NAT port 80 cho web server. Đối với các modem ADSL thì nó thường được gọi là Port forwarding, hoặc virtual server. Vào đó mà cấu hình để forward port 80 về IP của web server. Trong một số trường hợp, bạn phải đổi port web của modem sang 8080 hay bất kì port nào bạn muốn. Với một số modem (như Dlink) thì không có chỗ để cấu hình đổi port này, thì không cần thiết phải đổi, chỉ cần disable Remote router access.

Nếu đã NAT port, thoát khỏi cái web interface của modem mà bạn vẫn không vào web server được thì tắt firewall của web server đi là okie
