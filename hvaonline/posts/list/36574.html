Đã thử ngó qua nhưng thấy hơi nhức mắt. Chả đọc nữa.
Thử theo cách này xem.

Từ Windows Explorer -> Tool -> Folder Option -> File Tpyes -> chọn vào cái Drive -> Advanced -> New
  • Action: đặt tên cho thực thi
  • Application : ứng dụng thực thi ( chọn cái nào cảm thấy ít có thể nguy hiểm thì chọn : calc, notepad .... )

Có thể tạo 1 cái Application đơn giản, ví dụ như hiện lên 1 thông báo chẳng hạn.
Tiếp theo là lưu lại và Set Default cho nó.
