Chao Ke Kho,

Hoàn toàn không có cách "tháo bỏ GRUB". Bạn chỉ có thể: Cài một GRUB khác đè lên MBR, hoặc cài LILO hoặc các bootloader khác, hoặc sử dụng BIOS để boot. Cách sử dụng BIOS để boot thường được xem như là tháo/xóa bootloader, nhưng thật tế vẫn là sử dụng một bootloader khác đè lên MBR mà thôi.

Để chỉnh lại, bạn có thể chỉnh file cấu hình của GRUB. Bạn có đĩa Ubuntu có thể boot vào livecd, chỉnh lại file config vậy là được.

Hello Ke Kho,

Một cách *nôm na*, lệnh fdisk /mbr chỉ đơn thuần khôi phục lại Windows Loader mà thôi. Nói cách khác, lệnh này sẽ nạp trình Windows Loader vào MBR.

Bản chất của MBR chỉ đơn thuần tìm active (bootable) partition trong partition table của nó để trao quyền khởi động cho partition đó. Vì vậy, bạn có thể yên tâm sử dụng lệnh này mà không sợ hư hết bột đường đâu. )

Tại sao phải cài lại. Khi gặp 1 vấn đề nào đó, nên tìm hiểu nguyên nhân và fix nó hơn là ngồi cài lại từ đầu. Chẳng hạn với vấn đề của bạn, nếu bạn remove linux, có thể dùng
Code:
fdisk /mbr

để boot được vào windows
Còn nếu bạn cài lại windows và không vào được linux, thì có thể cài lại boot loader lên MBR bằng lệnh
Code:
GRUB: grub-install /dev/hda
LILO: /sbin/lilo

hoặc active patition như sau: (không cài lên MBR)
Code:
dd if=/dev/hda3 of=/mnt/C/grub.bin bs=512 count=1

dd: dump disk
if: input file
of: output file
bs: boot sector
count: số lần copy
