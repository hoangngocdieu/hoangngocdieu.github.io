Hướng dẫn trên mạng: ./configure, make, sudo make install là hướng dẫn chung để compile một package từ source theo chuẩn, nhưng không phải gói nào cũng cài như thế. Trong mọi trường hợp, tốt nhất là đọc Installation guide trên trang chủ hoặc đọc file INSTALL, install.txt, ... và README sau khi giải nén.

Ở đây, sau khi `cd` vào smplayer-0.6.9 thì mở file Install.txt ra và đọc:
Code:
$ less Install.txt
