Ken wrote:
SMTP dùng để gửi mail đi thôi. Còn muốn vừa gửi vừa nhận thì fai dùng POP3 

Theo mình biết thì:
_SMTP dùng để chuyển e-mail từ server --> server và từ client --> server (gửi e-mail).
_POP3 dùng để chuyển e-mail từ server --> client (nhận e-mail)
Còn cái POP3 dùng để gửi mail ra sao thì mình chưa rõ.
