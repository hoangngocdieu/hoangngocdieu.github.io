Nhiều bác hỏi cách cài private sock quá nên em viết bài này,hic hic.

Viết cho bác nào chưa biết tự mình setup 1 con sock,nếu bác nào biết roài thì thôi nhé



Đầu tiên các bạn đăng kí 1 host free ví dụ như



http://freehostia.com



Sau đó down file này về



http://www.freewebs.com/lukyforme/mocks%2D0.0.2%5F3.rar



Giải nén file ra bên trong có 1 folder và 1 file cgitelnet



Tìm file cgitelnet và mở nó ra



Mở file đó ra tìm dòng này :



$Password = "123456"; # Change this. You will need to enter this

# to login.



Thay chỗ "123456" bằng password của mình



Tiếp theo mở folder mocks ra tìm file mocks.conf ,mở file đó = notepad,tìm dòng:



PORT = 9559



Thay số 9999 bằng số bạn thích,đây chính là port của sock sau khi mình làm xong



Dòng MOCKS_ADDR = 64.72.112.12



Thay số 64.72.112.12 bằng ip host của bạn,ví dụ bạn đăng kí ở freehostia.com host của bạn sẽ có dạng



nguyenthihai.freehostia.com vao Start ==> Run gõ ping nguyenthihai.freehostia.com sẽ ra ip của host,thay số ip đó vào dòng MOCKS_ADDR



Up tất cả lên host,sau đó CHMOD 3 file này sang 744:

mocks.conf

mocks

cgitelnet.pl



Một số bạn không biết cách CHMOD,các bạn có thể dùng tool như leap FTP hoặc CHMOD luôn trên FTP cũng được.,ở trang host free này đã hỗ trợ sẵn việc CHMOD







Sau đó các bạn chay file cgitelnet.pl (ví dụ chạy trên host này link nó sẽ có dạng http://tênban.freehostia.com/cgitelnet.pl



Hiện ra 1 cửa sổ tối ôm,và có hỏi pass các bạn điền cái pas lúc mình thay đổi ở file cgitelnet.pl vào







Nếu điền đúng pass nó sẽ dẫn đến 1 trang các bạn gõ lệnh



cd mocks



vào để vào thư mục mocks







Sau đó gõ tiếp lệnh



./mocks -f -c mocks.conf start







Nếu thành công nó sẽ báo



Starting mocks ... OK

Killed it!



Bạn muốn chạy sock thì gõ lệnh



./mocks start



Tắt sock:



./mocks shutdown



Sau khi hoàn thành con sock của bạn sẽ có ip là ip của host port là port bạn đặt

Hướng dẫn :

-Bước 1 :
Đăng ký 01 free host :
www.cosmopod.com - Home
Sau khi đăng ký xong, bạn sẽ có usename và pasword.

-Bước 2 :
1-Chạy chương trình Bitvise Tunnelier .
tải tại đây nè
+http://quanviethack.vn

2-Chọn tab Login, khai báo như sau :
Host : cosmpod.com
Port : 443
Usename : usenam của bạn.
Initial method : chọn password .
Password : password của bạn.
3-Chọn tab Services : đánh dấu vào phần Enabled, các thông số khác để mặc định.
4-Click Login. Chọn Accept and Save. Một cửa sổ nhỏ hiện lên, bạn tắt cửa sổ này
và thu nhỏ Bitvise Tunnelier xuống tastbar.

1-Với IE : bạn vào Tool > InternetOptions. Click Connections >

2-Với Firefox, bạn vào Tool > Options, trong tab General chọn Connection Settings >
click Manual proxy configuration.
Trong phần Socks hosts : 127.0.0.1, port : 1080, >OK.
