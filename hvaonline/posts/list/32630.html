- Down bản vá win về trước.
- chạy Tools chuyên diệt conficker (search kido-conficker hoặc fixdwndp.log hoặc symantec w32.downadup removal tool....)
- Vào safe mode (Để hiện tất cả file ẩn và cả file ẩn hệ thống) rồi Documents and Settings rồi vào local settings, vào Temp và Temporary Internet Files, vào Content.ỊE và Windows rồi Temp, xóa hết các file và "folder" trong đó.
