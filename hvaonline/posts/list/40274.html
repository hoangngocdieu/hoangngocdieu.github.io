bạn download http://www.mediafire.com/?f33ri5f7sh0pw3q về và làm theo cách sau:
khởi động lại máy vào safe mode, chạy lần lượt ( hoặc đồng thời ) 3 file 0,1,2 lên, đây là 3 tool để gỡ bỏ conficker ra khỏi hệ đièu hành máy bạn, sau đó tiến hành chạy lần lượt các file sau 3,4,5,6 đây là các bản update windows. chú y quan trọng nhất là bản vá số 5 ( KB 958644 ) đây là bản vá lỗi conficker . sau khi làm các bước trên xong bạn khởi động lại máy thì bạn sẽ không bị dính lại conficker nữa. vì nếu bạn chỉ dùng tool 0,1,2 để gỡ bỏ conficker mà không update windows ( các bản vá 3,4,5,6) thì bạn sẽ bị dính lại.
cách trên mình đã test thành công trên windows xp, nếu bản vá không tương thích với windows của bạn thì bạn có thể download bản vá dúng với windows của bạn.
