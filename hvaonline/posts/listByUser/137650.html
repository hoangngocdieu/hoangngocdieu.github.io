Máy tôi giờ dùng hai hệ điều hành là XP và Server 2003. Cài trên hai phân vùng khác nhau. Hệ điều hành XP thì tôi cài đặt trước, sau đó mới cài Server 2003. Nếu bây giờ xảy ra tình huống hoặc hỏng hệ điều hành Xp, hoặc format ổ cài XP, để cài lại Windows XP mới. Thì khi đó bootloader trước đã bị xóa như vậy khi cài mới bootloader sẽ không khởi động được thằng Win Server 2003 kia. Bây giờ tôi muốn hỏi là làm cách nào để tiếp tục khởi động mà trên màn hình có thể lựa chọn được cả XP mới cài, và thằng Win Server kia?
Tôi đã nghĩ dùng cách này xem có được ko, vào thằng Win XP mới cài, vào chỉnh trong file BOOT.INI như sau:
[boot loader]
timeout=0
default=multi(0)disk(0)rdisk(0)partition(1)\WINDOWS
[operating systems]
multi(0)disk(0)rdisk(0)partition(1)\WINDOWS="Microsoft Windows XP Professional" n /fastdetect
từ file bootloader của thằng XP này em thêm vào dòng :

multi(0)disk(0)rdisk(0)partition(2)\WINDOWS="Microsoft Windows Server 2003" /fastdetect
