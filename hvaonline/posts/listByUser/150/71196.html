In MS-DOS, OS/2 and Windows, a batch file is a text file containing a series of commands intended to be executed by the command interpreter. Và để bung nén? Cũng dùng ngay lệnh tar (hoặc dd tùy trước đó bạn làm thế nào) là được.
Chi tiết thì bạn có thể tham khảo phần giúp đỡ ngắn gọn của 2 lệnh:
Code:
$tar --help
$dd --help

Hoặc chi tiết hơn, tham khảo man:
Code:
$man tar
$man dd

Giới thiệu qua chút về lệnh tar mà bạn sử dụng (lệnh dd có lẽ bạn tự tham khảo nhé ).
Code:
$ tar -cvf file_name.tar a b c

Lệnh mà bạn dùng có 3 phần, các tùy chọn, tên file (là file_name.tar), các thư mục bạn chọn (là a, b, c). Các tùy chọn dùng ở trên:
-c, --create create a new archive
-f, --file=ARCHIVE use archive file or device ARCHIVE
-v, --verbose verbosely list files processed 

-c dùng để tạo một file nén mới.
-f để chỉ ra cái file ta cần dùng.
-v liệt kê cái tiến trình.
Ngoài ra, còn thêm một tùy chọn nữa bạn cần lưu ý để giải nén là:
-x, --extract, --get extract files from an archive 

-x dùng để giải nén.
Ví dụ, tui muốn bung cái file bạn nén trên kia, ta làm như sau:
Code:
$tar -xf /mnt/hda1/backup.tbz

Chi tiết bạn tham khảo phần giúp đỡ của lệnh.
