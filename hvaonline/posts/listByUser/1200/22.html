Để bạn đỡ è cổ, tớ gợi ý bạn tí xíu:
Đúng như tên của nó, LDAP là một giao thức giúp thiết kế và truy xuất dữ liệu dựa trên mô hình client-server (directory service protocol), với đặc tính dễ sử dụng và mở rộng. Directory ở đây không phải là thư mục dạng cây đơn giản khi duyệt bằng Windows Explorer, mà nó là một loại hình database có tính mô tả cao, được tối ưu cho việc tìm kiếm.

Như vậy trong một hệ thống, LDAP thường thấy ở nơi chứa dữ liệu. Khi người ta cần một database có những ưu điểm như trên Trong Windows server có thể dễ dàng nhận thấy ở Active Directory vì nó dựa trên nền tảng LDAP.

Code:
Text1.Text = "Không có"
.....
Dim ctrl As Control
' duyệt từng control
For Each ctrl In Me.Controls
If (ctrl.Left = 240 And ctrl.Top = 240) Then
Text1.Text = "Có"
Exit Sub 'có rồi thì không cần tìm thêm
End If
Next
