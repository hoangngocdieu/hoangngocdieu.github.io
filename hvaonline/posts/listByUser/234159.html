Trong Windows7 hay Windows XP, khi bạn nhấp phím Shift 5 lần thì chương trình StickyKey sẽ hoạt động cho dù bạn đã login hay chưa. File thực thi của chương trình này là C:/Windows/system32/sethc.exe. Bạn dùng WinXP mini trong đĩa HirenBoots hay Dos để sao lưu lại file sethc.exe vào một nơi cần thiết, chép file cmd.exe và đổi tên thành sethc.exe (cái này giống lừu tình thằng Windows thật ), hoặc bạn có thể chép file exe nào cũng được miễn là đổi tên được rồi. Khi mở máy tính mà không có pass đăng nhập Windows, bạn sẽ dừng lại ở màn hình login, lúc này nhấp Shift 5 lần rồi cmd.exe hiện lên, gõ explorer.exe rồi làm gì thì làm mà pass Win vẫn được bảo toàn.

Mà tớ khuyên một câu nhá, dùng HirenBoots phá toi cái pass Windows đi cho nhanh, kêu thằng bạn làm lại.
