mình có 1 home network như sau:

Code:
Internet
|
|
| |- Comp1
Router 1 --- switch -|- Comp2
| |- Comp3
|
|
|
| |- Comp1
Router 2 --- switch -|- Comp2
|- Comp3

Router1 có cấu hình:
IP từ 192.168.0.100 ---> 192.168.0.200
Gateway: 192.168.0.1
Netsmak: 255.255.255.0
DNS: 192.168.0.1

Router2 có cấu hình:
IP từ 192.168.1.100 ---> 192.168.1.200
Gateway: 192.168.1.1
Netsmak: 255.255.255.0
DNS: 192.168.1.1

Chuyện là thế này: Comp1 ở router 2 mình chạy 1 file server, nếu mình share mạng trên chung 1 router thì có thể truy cập file trên Comp1 đó dễ dàng. Nhưng hiện tại vì phải block 1 số ports ở 1 số máy nên mình chạy 2 router để block 1 số máy ở router 2, còn router 1 thì không có block ports. Nhưng vì 1 số lý do nên không thể cho Comp1 ở router2 lên router1 được nên phải nằm dưới router2. Nhưng mình không biết làm cách nào để truy cập vào Comp1 ở router2 từ những máy ở router1 như 1 file server được.

Các máy đều chạy windows, Comp1 ở router2 chạy windows 2003 server.
Router1 là wireless router của linksys (WRT54GS), Router2 là wired router của D-link (DI-604).
