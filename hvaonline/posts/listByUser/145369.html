cryptswap1 là swap partition được encrypted. Mở /etc/fstab và /etc/crypttab ra, bạn sẽ thấy:

Code:
# swap was on /dev/sda7 during installation
#UUID=c53789a4-f307-4157-84f4-e4fdd354336f none swap sw 0 0
/dev/mapper/cryptswap1 none swap sw 0 0


Code:
cryptswap1 /dev/sda7 /dev/urandom swap,cipher=aes-cbc-essiv:sha256
