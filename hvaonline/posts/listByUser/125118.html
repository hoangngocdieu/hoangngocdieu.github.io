Theo mình biết thì đó là IP modem chứ không phải gateway modem, IP modem chính là gateway cho LAN của nó hay nói cách khác IP modem là nơi mà default gateway TCP/IP các PC trỏ về để truy cập internet.
Để chỉnh cho IP 2 modem khỏi trùng nhau :
- Đăng nhập vào cái modem 2 (tạm thời tắt cái 1 đi), trong mục Initial Setup đổi ip address mặc định của nó từ 192.168.1.1 thành 192.168.1.2 ; subnet mark 255.255.255.0 (giữ nguyên). Nhớ save lại --> reboot.
- Modem 1 vẫn để IP mặc định 192.168.1.1 ; subnet 255.255.255.0
PC nào có default gateway trỏ về 192.168.1.1 thì truy cập modem 1.
PC nào có default gateway trỏ về 192.168.1.2 thì truy cập modem 2.
