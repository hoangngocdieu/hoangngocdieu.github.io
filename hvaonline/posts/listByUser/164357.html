1. Chức năng của người dùng Buyer - Buyer Admin Panel :

A, Để sử dụng chức năng này, bạn đăng nhập vào website bằng Tài khoản sau đây :

Url : www.localhost/c2cdemo/c2cbuyer
Tài khoản : buyer@gmail.com
Mật mã : buyer

B, Chức năng chính :

- Gửi, Quản lý thư cá nhân của chính Buyer cho các Coder

- Quản lý thông tin cá nhân

- Thay đổi mật khẩu cá nhân

- Liên hệ với Admin website C2C (Code To Coder)






2. Chức năng của người dùng Coder - Coder Admin Panel :

A, Để sử dụng chức năng này, bạn đăng nhập vào website bằng Tài khoản sau đây :

Url : www.localhost/c2cdemo/c2ccoder
Tài khoản : coder@gmail.com
Mật mã : coder

B, Chức năng chính :

- Gửi, quản lý thư cá nhân của chính Coder cho các Buyer

- Upload source code để giao dịch

- Quản lý, Cập nhật Source Code của chính Coder

- Quản lý thông tin cá nhân

- Thay đổi mật khẩu cá nhân

- Liên hệ với Admin website C2C (Code To Coder)







3. Chức năng của người dùng Quản trị Website - Admin Panel :


A, Để sử dụng chức năng này, bạn đăng nhập vào website bằng Tài kkhoản sau đây :


Url : www.localhost/c2cdemo/adminpanel
Tài khoản : admin
Mật mã : admin

B, Chức năng chính :

- Cập nhật, Quản lý Thông báo Website

- Cập nhật Trang Thông tin Trợ giúp dành cho Buyer, Coder

- Cập nhật, Quản lý Tin tức Website C2C

- Trả lời, Quản lý Câu hỏi Người dùng website C2C (FAQ)

- Upload, Cập nhật, Quản lý Source Code của các Coder up lên Website C2C

- Quản lý, Trả lời Liên hệ của Người dùng Website C2C qua Email trực tiếp (sử dụng giao thức SMTP để gửi mail trên Web)

- Quản lý Thông tin Member (Coder, Buyer) Đăng ký sử dụng Website C2C, Gửi Email liên hệ trực tiếp đến Member (Qua giao thức SMTP trên Website C2C)

- Quản lý Thông tin PM (Personal Messgae) của các Member (Coder, Buyer) trên Website C2C.

- Cập nhật, Quản lý Danh mục Source Code

- Cập nhật, Quản lý Banner Quản cáo

- Cấu hình Hệ thống Website như : Yêu cầu Kích hoạt tài khoản khi đăng ký thành viên hay không ? (Confirm Open) , Sau khi đăng ký Thành viên Member có cần Duyệt trước khi có thể Đăng nhập bằng Tài khoản mình hay không ? (Authorize Open)

- Quản lý Thông tin Tài khoản Quản trị Website C2C (Admin)

- Thống kê Toàn bộ thông tin về Website như : Số thành viên đăng ký, Số bài gửi Liên hệ, Số câu hỏi FAQ, Tổng các Source Code được Up lên Website C2C, Số các bài viết về Tin tức, Số lần Người dùng truy cập website C2C, Tổng số lần Download các Source Code trên Website C2C.
