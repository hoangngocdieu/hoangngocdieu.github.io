Cài đặt để biến máy mình thành web-server bồ có thể dùng gói WAMP:
Code:
http://www.wampserver.com/en/download.php


Cài gói này máy sẽ chạy: Apache (webserver), MySQL (Cơ sở dữ liệu), và hỗ trợ PHP (Ngôn ngữ lập trình web) - wordpress được viết trên PHP. Sau khi cài xong nhớ đọc hướng dẫn sử dụng để biết thư mục nào sẽ trở thành webroot (nơi chứa tất cả trang web của localhost server - tui giải thích hơi thô thiển, cố hiểu nhé ). Và làm sao để "chạy" web ...

Tiếp theo kiếm thằng wordpress:

Code:
http://wordpress.org/download/


Giải nén sau đó tống nó vào cái thư mục webroot rồi ... làm theo hướng dẫn
