Hướng dẫn cấu hình router 2821 + FTTH + FPT

Mô hình:

Internet => Converter => Router => Firewall (ISA) => LAN


Mô tả:

Interface GigabitEthernet 0/0 kết nối với Converter
Interface GigabitEthernet 0/1 kết nối với Interface của Firewall (Ip address: 192.168.1.1)
Firewall với IP: 192.168.1.2, Subnet mask : 255.255.255.0 , Default gateway: 192.168.1.1, DNS : 210.245.0.11

Code:
ip nat inside source static 192.168.1.2 118.70.x.y


Trong đó: 118.70.x.y địa chỉ IP WAN được FPT cấp, tôi đang sử dụng gói bạc của FTTH (FPT), gồm: 1 IP front và 4 IP WAN nhưng thực tế chỉ dùng được 2 IP WAN vì 2 IP còn lại dùng cho IP network và IP boardcast.

Code:
ppp pap sent-username HNFdl-xxxxxx-xxx password xxxxxxxxxxxx

Nhập username và password theo hợp đồng của bạn.

Cấu hình Router theo thông số sau:

interface GigabitEthernet0/0
no ip address
ip route-cache flow
ip tcp adjust-mss 1452
duplex auto
speed auto
pppoe enable
pppoe-client dial-pool-number 1
!
interface GigabitEthernet0/1
ip address 192.168.1.1 255.255.255.0
ip nat inside
ip tcp adjust-mss 1452
duplex auto
speed auto
!
interface Dialer0
ip address negotiated
ip mtu 1492
ip nat outside
encapsulation ppp
dialer pool 1
dialer-group 1
no cdp enable
ppp authentication pap callin
ppp pap sent-username HNFdl-xxxxxx-xxx password 7 xxxxxxxxxxxx
!
ip classless
ip route 0.0.0.0 0.0.0.0 Dialer0
!
no ip http server
