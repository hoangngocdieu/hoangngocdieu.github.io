What are HIBERFIL.SYS and PAGEFIL.SYS?

These are system-generated files. They are used by Windows for hibernation and virtual memory control.

* HIBERFIL.SYS is a file the system creates when the computer goes into hibernation mode. Windows uses the file when it is turned back on. If you don't need hibernation mode and want to delete the file you need to turn the hibernation option off before Windows will allow you to delete the file...
o Start | Control Panel | Power Options
o Go to the Hibernate Tab
o Uncheck the Enable Hibernation box if you don't need the hiberation function

* PAGEFIL.SYS is the virtual memory file. Typically, the size of the file is around 1.5 times your physical memory size. Windows uses this file for its normal operation however if you really need the space you can delete it after turning the virtual memory option off...
o Start | Right Click on My Computer | Select Properties from the menu
o Select the Advanced Tab
o Select Performance Settings
o Select the Advanced Tab
o Under virtual memory use the Change button to either set the size you want or turn it completely off. Note that if you turn it off or make the value too small you may notice a system slowdown. Windows wants to use this file and if it's set to zero then in addition to a slowdown while running, on system shutdown you may think Windows has hung due to the extra time involved. You may have to experiment a bit if you set it lower than the recommended minimum but most people find setting the size to the amount of RAM is a reasonable compromise.

WPA Wifi Protected Access là 1 tập con trong chuẩn tạm thời IEEE 802.11i, được dùng để thay thế WEP (Wired Equivalent Privacy).

WPA bao gồm những phương thức mã hóa rất an toàn như Temporal Key Integrity Protocol [TKIP], Message Integrity Check [MIC], initialization vector [IV] mở rộng…Nó có hai cơ chế xác thực, một là cơ chế xác thực dành cho người dùng doanh nghiệp sử dụng chuẩn 802.1x & giao thức xác thực mở rộng (Extensible Authentication Protocol-EAP), cơ chế thứ hai kia là dành cho người dùng gia đình sử dụng khóa mã Pre-Shared Key (PSK).

WPA2 là bổ sung đầy đủ của chuẩn 802.11i và sử dụng thêm kiểu mã hóa AES (Advanced Encryption Standard) khá là mạnh mẽ. Cả hai kiểu WPA & WPA2 đều an toàn hơn WEP.

Đối với những thiết bị không dây cũ, để sử dụng được WPA/WPA2 bạn cần phải nâng cấp phần mềm điều khiển cho thiết bị. Với những thiết bị chuẩn 802.11b hoặc 802.11a thì không thể nâng cấp lên được.

~~~Source : www.vninformatics.com ~~~~
