Máy mình bị lỗi như sau:

A problem has been detected and Windows has been shutdown to prevent damage to your computer.

If this is the first time you’ve seen this stop error screen, restart your computer. If this screen appears again, follow these steps:

Check to be sure you have adequate disk space. If a driver is indentified in the stop message, disable the driver or check with the manufacturter for driver updates. Try changing video adapters.

Check with you hardware vendor for any BIOS updates. Disable BIOS memory options such as caching or shadowing. If you need to use Safe Mode to remove or disable components, restart your computer, press F8 to select Advanced Startup Option, and then select Safe Mode

Techinal infortion:

***Stop: 0x0000008E(0x00000005,0x8056452A,0xEDE49C18,0x00000000)

Physical memory dump complete

Contact your system administrator or technical support group for further assistance.





Sau khi khởi động lại máy tính và chọn Last Known Configuration (Your most recent setting that worked) … tôi vào trong Window vào phần Computer Management/ System Tools/ Event Viewer/ System thấy báo lỗi sau:



Printer RepliGo failed to initialize because a suitable RepliGo driver could not be found.



For more information, see Help and Support Center at http://go.microsoft.com/fwlink/events.asp.



Event ID 23



Sau đó tôi có vào Add/ Remove Programs gỡ bỏ chương trình Convert to Repligo. Sau đó khởi động lại máy và vào Window nhưng không làm gì thì thấy xuất hiện “màn hình xanh”:



A problem has been detected and Windows has been shutdown to prevent damage to your computer.

PAGE_FAULT_IN_NONPAGED_AREA

If this is the first time you’ve seen this stop error screen, restart your computer. If this screen appears again, follow these steps:

Check to be sure you have adequate disk space. If a driver is indentified in the stop message, disable the driver or check with the manufacturter for driver updates. Try changing video adapters.

Check with you hardware vendor for any BIOS updates. Disable BIOS memory options such as caching or shadowing. If you need to use Safe Mode to remove or disable components, restart your computer, press F8 to select Advanced Startup Option, and then select Safe Mode

Sau khi khởi động lại máy tính và chọn Last Known Configuration (Your most recent setting that worked) … tôi vào trong Window vào phần Computer Management/ System Tools/ Event Viewer/ System thấy báo lỗi sau:



The Computer Browser service terminated with the following error:

This operation returned because the timeout period expired.



For more information, see Help and Support Center at http://go.microsoft.com/fwlink/events.asp.



Event ID 7023



Tiếp đó, tôi vào Control Panel / Administrative Tools / Services chọn Computer Browser để Disable dịch vụ này lại nhưng không được.

Sau đó tiếp tục xuất hiện hiện tượng trên “màn hình xanh” dù chỉ là chơi game nhỏ nhu Solitaire, Minesweeper … , hay là đang soạn thảo văn bản trên Word 2003. Nhưng nội dung lỗi có khác chút ít:



A problem has been detected and Windows has been shutdown to prevent damage to your computer.

Driver_IRQL_NOT_LES_OR_EQUAL

Check to make sure any new hardware or software is properly installed. If this is a new installation, ask your hardware or software manufaceturer for any windows updates you might need.

If problems continue, disable or remove any newly installed hardware or software. Disable BIOS memory options such as caching or shadowing. If you need to use Safe Mode to remove or disable components, restart your computer, press F8 to select Advanced Startup Option, and then select Safe Mode

The Computer Browser service terminated with the following error:

This operation returned because the timeout period expired.



For more information, see Help and Support Center at http://go.microsoft.com/fwlink/events.asp.



Event ID 7023



Đôi khi xuất hiện lỗi:



The description for Event ID ( 10016 ) in Source ( DCOM ) cannot be found. The local computer may not have the necessary registry information or message DLL files to display messages from a remote computer. You may be able to use the /AUXSOURCE= flag to retrieve this description; see Help and Support for details. The following information is part of the event: application-specific, Local, Activation, {0C0A3666-30C9-11D0-8F20-00805F2CD064}, HVAN, IWAM_HVAN, S-1-5-21-1844237615-688789844-839522115-1012.



Event ID 10016



Hiện tượng trên xuất hiện khoảng 5-6 lần liên tiếp thì máy không vào được Window nữa, trong khi nguồn điện vẫn ổn định và đèn ổ cứng có hiện lên … Sau đó tôi tháo nguồn và pin CMOS để ra ngoài khoảng 1 tiếng rồi lắp lại thì vào Window bình thường nhưng chỉ sau 2 tiếng lại xuất hiện “màn hình xanh” … Khởi động lại thì ko vào được Window nữa nhưng đèn ổ cứng vẫn hiện. Tôi thử sử dụng phần mềm Test trong đĩa Hirent Boot thì chỉ vào nửa chừng là “đơ máy”.
