Thông thường, trên hệ thống *nix những gói .tar.gz là những gói chứa mã nguồn của chương trình. Để cài đặt nó, mình phải biên dịch nó ra các file dạng thực thi được và các file thư viện... Muốn làm được việc này thì hệ điều hành phải cài sẵn trình biên dịch (gcc, make...).

Các bước cài đặt thường gồm 3 bước:
configure: bước này dùng để kiểm tra hệ thống, chuẩn bị các bước cần thiết trước khi biên dịch.
make: biên dịch mã nguồn thành các file thực thi, thư viện...
make install: cài đặt (copy) các file đã được biên dịch của bước make vào những vị trí cần thiết trên hệ thống.

Đó là cách thức chung, còn với những phần mềm khác nhau có thể có cách cài đặt khác nhau, bạn đọc file README hoặc INSTALL đi kèm theo mã nguồn để biết thêm chi tiết.
