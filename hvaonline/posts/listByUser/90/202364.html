Mình mới cài FC11 và cài đặt dhcpd

Tắt FireWall

Khi cấu hình ip tĩnh cho Eth0, mình config file /etc/dhcpd.conf cho assign 1 subnet đơn giản nhất

rồi start dhcpd lên, nhưng nó báo lỗi Fail và hiện thông báo ko thể lắng nghe trên interface

Not configured to listen on any interfaces! 


Sau đó mình thử vào file /etc/sysconfig/dhcpd cấu hình

Code:
# Command line options here
DHCPDARGS=eth1


rồi start service network và dhcpd nhưng vẫn báo fail.

Đây là file /etc/dhcpd.conf của mình các bạn xem dùm

Code:
ddns-update-style interim;
ignore client-updates;
subnet 10.0.0.0 netmask 255.255.255.0 {
range 10.0.0.150 10.0.0.225;
}


còn ip address trên ETH0 thì chỉ cấu hình ip 10.0.0.1 netmask 255.255.255.0
