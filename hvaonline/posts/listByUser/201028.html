mfeng wrote:
Nếu biên dịch bằng gcc, bạn dùng dòng lệnh sau:
Code:
gcc -osimple simple.c

Tùy chọn -o chỉ định file compiled ra là simple, thay vì tên file mặc định là a.out

Để chạy chương trình vừa biên dịch, dùng lệnh sau:
Code:
./simple
