Các bước bạn cần phải làm như sau:
1-Xem chính xác xem server là con RX300 S mấy.
2-Down driver SAS ở cái links mình đã gửi tương ứng với con server
3-Down xong quăng nó vào 1 cái usb, không cần phải extract ra, các file driver SAS thường đã có dạng disk inmage và thường có dung lượng bằng dung lượng 1 chiếc đĩa mềm. Centos đọc thoải mái
4-boot vào bằng đĩa cài Centos bình thường, tại cửa sổ welcome thay vì ấn enter bạn gõ "linux dd" (nhớ là có dấu cách giữa linux và dd. Linux dd = linux disk driver)sau đó enter. Cắm usb chứa driver SAS vào và đợi
5-Centos sẽ nhận ra usb, tìm đến thư mục chứa driver SAS chọn nó để load, bạn không cần phải install bất cứ thứ gì, server sẽ tự động load. Load xong thì rút usb ra, nếu đúng driver thì Centos sẽ nhìn thấy ổ cứng của bạn, và bạn tiến hành cài OS bình thường.
