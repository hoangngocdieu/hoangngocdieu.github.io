Bữa trước mới làm xong cái đĩa vista kèm SP1, giờ post lên cho những ai muốn tham khảo thêm

1. Chuẩn bị:
- Đĩa windows vista và file cài đặt SP1 - http://www.microsoft.com/downloads/details.aspx?FamilyID=b0c7136d-5ebb-413b-89c9-cb3d06d12674&DisplayLang=en hoặc http://www.microsoft.com/downloads/details.aspx?FamilyID=874a414b-32b2-41cc-bd8b-d71eda5ec07c&DisplayLang=en
- Vlite - http://www.vlite.net/download.html
- Đĩa trắng DVD và ổ cứng trống vài GB

2. Tích hợp
- Bỏ đĩa DVD gốc vista vào, cài đặt Vlite
- Vào thư mục chứa windows vista (ở ổ DVD), Vlite sẽ copy toàn bộ file vào một thư mục ở ổ cứng của bạn


- Giờ bạn phải xác định rõ phiên bản vista nào cần được tích hợp SP1 vào

- Kế tiếp chọn "Service Pack Slipstream" và "Bootable ISO" -> tiếp -> chọn file cài đặt Windows Vista SP1.


- vLite cần khoản 60-90 phút để tích hợp tất cả file SP1 vào bộ cài Vista. Khi hoàn thành thì chọn sang chế độ "Direct Burn".



Ngoài ra khi bạn cài Windows Vista SP 1 thì bộ cài SP1 sẽ không xóa những file cũ của vista. Những file đó cần để khi bạn muốn uninstall SP1 về sau. Vì thế ổ cứng sẽ tốn thêm một số dung lượng. Nếu bạn muốn xóa chúng thì vảo Run gõ "vsp1cln.exe".
