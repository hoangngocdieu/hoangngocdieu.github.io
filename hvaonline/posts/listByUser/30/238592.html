Bạn vào Trang everywebspace.com đăng ký 1 tài khoản.
Sau đó Quay lại trang CO.CC cài đặt DNS cho tên miền của bạn.
Với các thông số là:

Service Type : Name server (DNS)
Name server :
ns1.everywebspace.com
ns2.everywebspace.com
ns3.everywebspace.com
ns4.everywebspace.com
ns5.everywebspace.com

Mình thấy bạn đang dùng dạng URL Forwarding, cho nên bạn bỏ chế độ đó đi, và chuyển sang cài đặt DNS nhé (phần trên).
Lên Google tìm download các File Index.
Quay lại trang everywebspace.com (sau khi đã kích hoạt xong), bạn vào DOMAIN chọn Addon Domains, nhập tên miền của bạn vào (có đuôi co.cc).
OK...
Giờ bạn cài đặt trên HOST. Vào File Manager 1, Vào public_html, chọn Upload, Tìm File INDEX (nhớ là dạng .ZIP), Up lên host.
Quan trọng là trong phần File INDEX, bạn chỉnh sửa lại các thông số cho phù hợp với Trang Forum của bạn, cụ thể là phải có 1 đường dẫn đến Forum của ban.
