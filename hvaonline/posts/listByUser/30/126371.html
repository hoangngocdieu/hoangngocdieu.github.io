Những ai sẽ bị Red Code hại?

Red Code chỉ tấn công vào các hệ thống có cài đặt Internet Information Server (IIS) các phiên bản 4 và 5. Như vậy những ai đang dùng Windows 95, Windows 98, Windows Millennium, Windows XP, Windows NT phiên bản 3505 trở xuống thì không cần làm gì cả vì IIS 4 và 5 chỉ chạy trên Windows NT 4 và Windows 2000.

Để biết được máy bạn có đang chạy IIS hay không thì có thể làm như sau:
1. Nhấn 3 phím Ctrl - Alt - Del một lượt
2. Trong cửa sổ Task Manager, chọn tab Process
3. Tìm trên cột Nam xem có Inetinfo.exe không, nếu có thì bạn đang chạy IIS

Làm sao để trị được Red Code?
Trước hết tải phần chương trình phụ trợ xuống từ trang web của Microsoft như sau:
1. Tạo một thư mục tên IIS_patch
2. Nếu dùng Windows NT 4 thì tải xuống từ địa chỉ
http://www.microsoft.co...ease.asp?ReleaseID=30833
Nếu dùng Windows 2000 thì tải xuống từ địa chỉ http://www.microsoft.co...ease.asp?ReleaseID=30800
3. Sau đó vào thư mục IIS_patch chạy chương trình vừa tải xuống. Sau khi chạy xong, khởi động lại máy
Theo Microsoft, sau khi khởi động lại, máy của bạn sẽ hoạt động bình thường. Red Code là loại virus chỉ tìm kiếm lỗ hổng trong IIS mà lây lan theo đường truyền internet. Nó không có khả năng tự vệ lại quá trình tiêu diệt trên cho dù nó đang nằm trong bộ nhớ của máy bạn. Vì vậy sau khi cài đặt phần mềm bổ trợ, máy của bạn hiện "tạm" an toàn. (biết đâu còn có Red Code 3)

Lưu ý: Nếu khi cài đặt bạn nhận được thông báo lỗi thì có thể là do máy bạn chưa có service pack, trường hợp này bạn phải cài đặt service pack trước. Bạn có thể tải service pack ở các địa chỉ:

Windows NT 4: http://www.microsoft.co...icepacks/sp2/default.asp
Windows 2k: http://www.microsoft.co...commended/SP6/allSP6.asp
