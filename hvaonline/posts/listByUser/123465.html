Pác ợ, đường dẫn tương đối với tuyệt đối là do pác thôi. Nếu pác có 1 cái host và domain (http://abc.com/), pác up code (file html) của pác lên folder A trong abc.com đó và pác up thêm 1 file video cũng lên file A luôn. Lúc này pác có thể dùng đường dẫn tương hay tuyệt thì tuỳ pác:
- Tương đối: thì pác cứ chèn code như w3school src="movie.mp4" tức là khi pác chạy file html ở trên và src="movie.mp4" thì nó sẽ tìm 1 file tên là movie.mp4 trên folder A (tức là folder chứa html đang chạy). Lưu ý nếu file HTML của pác nằm ở folder B mà movie.mp4 ở A thì bó tay (lúc này nó sẽ tìm ở Folder B xem có file movie.mp4 ko).

- Tuyệt đối: thì pác cứ src="http://abc.com/A/movie.mp4" thì lúc này pác chạy file html ở chỗ nào (dù là A hay B) thì nó vẫn chạy cái file movie.mp4 ở trong folder A.
