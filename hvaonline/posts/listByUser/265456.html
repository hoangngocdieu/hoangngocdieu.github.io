Mình có 1 vấn đề khó khăn cần giúp đỡ.

Trong Mini Xp của hiren boot có chương trình Seagate DiscWizard (bản cũ chỉ chạy trên mini XP). Seagate DiscWizard (SDW) khi chạy có sử dụng file snapman.sys trong thư mục driver của windows (I386\System32\drivers).
Theo yêu cầu công việc mình cần chạy thêm Acronis True Image Home 2012 (ATI 2012) trên mini XP này. ATI 2012 cũng sử dụng file driver snapman.sys nhưng file snapman.sys này có vesion mới hơn thì mới chạy được. Mình đã thay snapman.sys mới để chạy được ATI 2012. Nhưng SDW thì lại chạy không được vì nó chỉ chạy với file snapman.sys cũ mà thôi.

Tức là ATI 2012 chỉ chạy với file snapman.sys mới, còn SDW chỉ chạy với file snapman.sys cũ. Mà snapman.sys này dù mới hay cũ thì nó bắt buộc phải nằm ở thư mục I386\System32\drivers. Trong 1 thư mục thì làm sao có thể có 2 file có tên giống nhau được. Đây là khó khăn của mình.
Đồng nghĩa với việc muốn dùng SDW thì mình để bản mini XP cũ, còn muốn dùng ATI thì phải dùng 1 bản mini XP mới. Như thế thì bất tiện. Mà vấn đề cốt lõi nằm ở file snapman.sys.
