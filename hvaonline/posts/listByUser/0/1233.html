Với thông báo "'The request operation has failed' " thì LM đoán là bạn dùng Apache trên Windows.
Kiểm Tra error log để biết thêm. Nhưng có một số tình huống cơ bản có thể biết nguyên nhân.

1.Cài PHP cho Apache trên windows thì bạn kiểm tra xem các file trong config của apache mà php yêu cầu add vào httpd.conf có tồn tại và đúng path hay không. Dù cài CGI hay Apache Module thì như nhau

Action application/x-httpd-php "/php/php-cgi.exe"

LoadModule php<version>_module "c:/php/php<version>apache<version_apache>.dll"

Nếu bạn cài apache 2.2.x mà dùng PHP như apache module thì load phải là file php<version>apache2_2.dll

2. Trường hợp 2 là bạn cài các extension nhưng thiếu lib, ví dụ như Mysql không copy file libmysql.dll vaò %system32% chẳng hạn.

3. Trường hợp hiếm là các file trên bạn nhúng mất quyền Read/Execute.
