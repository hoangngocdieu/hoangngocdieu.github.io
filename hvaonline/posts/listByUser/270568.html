• Thiết kế, cấu hình, cài đặt và duy trì hạ tầng mạng và hạ tầng an ninh

• Theo dõi tình trạng của hệ thống phần cứng, phần mềm và lên kế hoạch nâng cấp các hệ thống này
• Set up and monitor commercial server system
• Participate in designing, monitoring high scalability commercial software system
• Set up and maintain IT system
• Design, configure, set up and maintain network and security infrastructure
• Monitor networking infrastructure including core network switches, routers, firewalls, intrusion, servers, VPN concentrators
• Record status of hardware and software system and schedule upgrades
YÊU CẦU CÔNG VIỆC
• Tốt nghiệp đại học
• Có hiểu biết về Khoa học Máy tính
• Có kinh nghiệm thiết kế, triển khai các hệ thống có khả năng mở rộng
• Có ít nhất 2 năm kinh nghiệm ở vị trí Kỹ sư Hệ thống
• Có thể làm việc theo nhóm và làm việc độc lập

• Thành thạo hệ điều hành *nix
