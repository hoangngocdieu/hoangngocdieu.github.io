Cài đặt Jforum.
I. Tạo Database:
Trong bài này mình sẽ sữ dụng MySQL để tạo cơ sở dữ liệu. Các bạn có thể download MySQl tại trang web: http://mysql.com (Phiên bản mới là 5.0) Sau khi download về ta tiến hành cài đặt file set up. Sau đó ta sẽ tạo database. Bạn vào Start -> All programs -> MySQL -> MySQL 5.0 ->MySQL Server Instance Config Wizard.


Ta chọn Detailed Configuration. Tiếp lick Next.


Chon Developer Machine.Next


Ta chọn Multifunctional Database.Next


Chọn ổ đĩa.Next


Ta chon option Online Transaction Processing(OLTP)




Ta sử dụng tiếng việt nên chon Best Support For Multilingualism. Next


Các cậu nên check vào ô: Include Bin Directory in Windows PaTH.


Đặt password cho account root. Next


Click Execute để bắt đầu configuration.

Sau khi làm các bước như hình demo, là chúng ta đã thiết lập xong MySQL. Để tạo database cho forum. Ta vào Start -> All programs -> MySQL -> MySQL 5.0 -> MySQL Command Line Client.
Ta đánh password để vào MySQL, ngay dấu nhắc mysql ta đánh lệnh tao database:
Code:
mysql> create database jforum;

nếu báo là : Query OK, 1 row affected (0.00 sec) là database đã tạo xong

II. Cài đặt JForum:
1. Tải phần nềm:
Để có thể chạy được Jforum ta cùng có những phần nềm sau:
A Trước tiên cậu phải down cái java tên là J2SE(TM) Development Kit 5.0 Update 8 từ đây:
http://java.sun.com/javase/downloads/index.jsp

B Sau đó down luôn cái Apache Tomcat về:
http://mirrors.24-7-solutions.net/pub/apache/tomcat/tomcat-5/v5.5.17/bin/apache-tomcat-5.5.17.exe

C Tiếp theo là đưa jforum về để sử dụng : jforum đây .
nếu link trên không hoạt động dùng link dưới : http://jforum.net/download.jsp

Ta cũng cài đặt J2SE(TM) sau đó mới tiến hành cài đặt Apache Tomcat. Cài đặt xong Tomcat ta mở trình duyêt lên và đánh đĩa chỉ : http://localhost:8080 , nếu thấy trang web của Apache Tomcat xuất hiện là nó đang chạy bình thường cài đặt đã hoàn tất.

2. Cài đặt jforum:
Ta tiến hành giải nén file JForum-2.1.6.zip sau đó ta đổi tên thư mục JForum-2.1.6 thành jforum(các cậu có thể để tên khác nếu muốn và nên viết thường không viết hoa). Ta chép thư mục jforum và thư mục webapps của Apache Tomcat.( C:\Program Files\Apache Software Foundation\Tomcat 5.5\webapps).
Ta mở file SystemGlobals.properties.(C:\Program Files\Apache Software Foundation\Tomcat 5.5\webapps\jforum\WEB-INF\config)
Sửa
Code:
default.container.encoding = ISO-8859-1

Thành
Code:
default.container.encoding = UTF-8

Sau đó Save lại.

Tiếp theo ta mở file mysql.properties.( C:\Program Files\Apache Software Foundation\Tomcat 5.5\webapps\jforum\WEB-INF\config\database\mysql)
Ta tìm dòng này:
Code:
# If you're going to use MySQL 4.1 or newer,
# set this attribute to "true"
database.support.subqueries = false

Vì ta dùng bản MySQL 5.0 nên ta chuyển dòng “false” thành “true”
Code:
database.support.subqueries = true

Save file lại.

Sau khi đó ta mở trình duyệt lên và đánh đĩa chỉ sau: http://localhost:8080/jforum .
Trang web cài đặt jforum xuất hiện:


Chọn Database Type là MySQL


Connection Type : JDBC.
Database name: là jforum
Database Username: root.
Database Password: là password lúc cài đặt MySQL ta đã đặt.
Database Password[ confirm]: đánh lại password.
Database Encoding: UTF-8
Use Connection Pool: Yes.


Ta đặt password cho admin trong forum. Sau đó Next Step.


Bước này thông kê lại những gì mình thiết lập. Nếu cần sửa đổi ta click vào Modify Configuration. Còn không có thì lick Begin Install


Check vào option I have done….. Sau đó lick vào Click here to access the forum.
Forum xuất hiện. Viêc cài đăt jforum đã hoàn tất.

Chúc các cậu cài đặt thành công.
) Cheer )

Bạn có thể tạo một Cascading Style Sheets(CSS) để thay đổi lại tag link trong forum. Đây là đoạn code mẩu :
Code:
a:link { color: #01336b; text-decoration: none; }
a:active { color: #01336b }
a:visited { color: #01336b; text-decoration: none; }
a:hover { color: #dd6900;text-decoration: underline }

-"link": có nghĩa là màu của đoạn link nó có màu gì và text-decoration chính là điểm bạn lưu ý nếu muốn không có gạch chân thì cậu để none.
-"active": chính là màu của link bạn nhấn chuột xuống link.
-"visited": màu của đoạn link sau khi bạn đã lick vào link đó rồi.
-"hover": màu của đoạn link khi cậu re chuột vào link.
"link,active,visite" thì cậu nên để cùng một màu nha.
