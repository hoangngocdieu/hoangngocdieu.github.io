start -> settings -> network connections -> right lick lên biểu tượng "Local area connection 2" chọn use the follwing IP address :
phần IP address gõ : 192.168.1.x ( x= 5 -> 20)
subnet mask : không chỉnh gì cả
defaul gatewa : 192.168.1.1 hoặc 192.168.1.254 tùy model (thường của bọn FTP là 192.168.1.254)

Rùi bây giờ bạn thử vào 1 trang web nào đó xem đã có mạng chưa nếu chưa thì bạn dùng lệnh sau trên cmd : ping 192.168.1.11 -t và ping 203.162.0.11 -t sau đó post kết quả lên cho mọi người biết lỗi để sửa típ.
Gook luck.
