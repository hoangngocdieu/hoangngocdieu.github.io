Em đã tìm hiểu về vấn đề này bằng help của Windows thì chỉ thấy nói như sau:

master boot record (MBR)
The first sector on a hard disk, which starts the process of booting the computer. The MBR contains the partition table for the disk and a small amount of executable code called the master boot code.

Rồi nói về cái chương trình này DiskPart.exe .Nhưng trong lệnh của nó không có cái nào dùng để sao lưu MBR

Còn cái này chỉ nói đến cách tạo MBR mới.
FixmbrRepairs the master boot record of the boot disk. The fixmbr command is only available when you are using the Recovery Console.
fixmbr [device_name]
Parameter
device_name
The device (drive) on which you want to write a new master boot record. The name can be obtained from the output of the map command. An example of a device name is:
\Device\HardDisk0.

Example
The following example writes a new master boot record to the device specified:

fixmbr \Device\HardDisk0
