Để người dùng có thể gửi mail đến cho bạn, bạn phải dùng một một server script như ASP, PHP hoặc JSP .v.v. HTML không làm được.

VD sau sẽ tạo trang gửi mail bằng PHP:

Trước hết, chỉnh lại trang html một chút ) :
Code:
<html>
<head>
<title>Góp ý</title>
</head>
<body>
<form method='POST' action='sendmail.php' name='send'>
Tiêu đề : <input type='text' size='10' name='subject' value='Thư góp ý' /><br />
Nội dung : <textarea name='message' rows="10" cols="25"></textarea> <br />
<center><input type="submit" value="Send" /></center>
</form>
</body>
</html>


Trang sendmail.php:

Code:
<?php
$subject = trim($_POST['subject']);
$message = trim($_POST['message']);
if($subject && $message){
if(mail('email@domain.com', $subject, $message)){
echo 'Cảm ơn bạn đã góp ý. <br />';
echo 'Click <a href="http://lalatt.110mb.com/"> vào đây </a> để trở lại. <br />';
}
else{
echo 'Xin hãy thử lại sau. Cảm ơn.<br />';
echo 'Click <a href="http://lalatt.110mb.com/"> vào đây </a> để trở lại. <br />';
}
}
else{
echo 'Bạn chưa nhập nội dung hoặc tiêu đề. Vui lòng thử lại.';
}
?>


Ngồi mò mò, không biết có lỗi không, bạn test xem hen. À, nhớ sửa cái email@domain.com thành mail của bạn nghen.
