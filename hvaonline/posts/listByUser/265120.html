Đầu tiên là bạn check history của browser xem có web nào lạ lạ không.
Nếu dùng linux thì vào terminal lệnh last để xem các lần đăng nhập cuối cùng gần đây, vào /var/log để xem các log hệ thống
Còn dùng window có thể check trong menu recent item, cái này depend vào version của windows, bạn tự tìm vì lâu không dùng win không nhớ
Dùng chức năng search file với tuỳ chọn date modified gần đây để xem các file tạo / sửa gần đây, ngoài ra enable các log trong adminstrator tool để theo dõi máy tính, với window vista trở lên khi mở menu start bạn sẽ thấy được các ứng dụng gần đây

Cuối cùng là thắt chặt an ninh cho máy của bạn, bằng cách đổi password admin, chỉnh lại các quyền access đến các file và folder.
