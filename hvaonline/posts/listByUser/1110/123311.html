Trước hết theo tinh thần của topic: "Thảo luận về JForum" tớ muốn đưa ra một số điều mà bản thân nhận thấy khi mày mò với cái JForum này.

1. Những thứ cần khi cài đặt:
+ Server Apache Tomcat. (hay là một server nào đó hỗ trợ JSP)
+ CSDL (ở đây tớ dùng MySQL)
+ Source của JForum
+ Bài hướng dẫn của Khủng Long (HVA) )

2. Những thứ cần biết khi muốn nhúng tay vào "quậy" cái JForum:
+ HTML, Javascript.
+ Java
+ FreeMarker
+ MySQL
+ Apache Tomcat.

Về việc chỉnh sửa sơ giao diện:
- Lang thang trong thư mục của JForum, view source code vài trang tớ thấy
rất nhiều trang có dòng này: #include "header.htm" ở đầu và #include "bottom.htm" ở cuối. => Muốn chỉnh sửa giao diện cơ bản thì cứ đè 2 "thằng" này ra mà chỉnh.

+ Mấy cái vụ Image, icon,... thì xem trong /templates/default/images

+ Agreement xem trong /templates/agreement

+ Avatar, Smiley xem trong /images

+ Tìm file Việt hóa ngôn ngữ ở: /WEB-INF/config/languages

------------------

- Chỉ sơ bộ từng đó thôi, nhưng tớ mất biết bao nhiêu thời gian. (Hehe, đây là lần đầu tớ làm việc với Web). Cũng chỉ tại cái tội: Cắm đầu cắm cổ "cày" Java, chẳng để ý đến mấy cái thư mục. Nhưng giờ khi muốn hiểu cái JForum này quả thật chẳng dễ tí nào, chẳng khác gì một mớ bòng bong. Xem code mấy file căn bản thấy có chèn mã của FreeMarker -> phải đọc FreeMarker, có Javascript -> phải đọc Javascript, có .class -> học Java. Bên cạnh đó còn có MySQL và Apache nữa chứ.

+ Download "http://prdownloads.sourceforge.net/freemarker/freemarker-2.3.9.tar.gz" khá nhẹ, có kèm cả Documentation. Bản thân FreeMarker cũng không quá phức tạp để hiểu. Nhưng vấn đề là ở chỗ... tớ ko biết làm sao để chèn một đoạn code FreeMarker cả, chèn vào đâu? như thế nào?

+ Gặp câu hỏi của Duza về vụ /forums/list.page, lại tiếp tục lần mò. Xem trang index của / rõ ràng có dòng "document.location = /forums/list.page" nhưng chẳng biết nó lôi ở đâu ra cả. Tớ suy nghĩ theo hướng Apache Tomcat sẽ định nghĩa Url đó theo cách của nó, thế là lại lôi Documentation của Tomcat ra đọc -> 1 mớ tiếng Anh...

+ Giờ thật sự tớ đang ở trong mớ bòng bong hỗn độn, chưa có lối ra, chắc sắp tẩu hỏa nhập ma quá. Hic hic.
