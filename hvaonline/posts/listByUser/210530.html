Muốn xem địa chỉ nơi mà user dùng làm nơi chứa web trong sever thì cat file passwd ra xem. Mật khẩu trên hệ thống *nix được lưu trong file /etc/passwd, một file chứa các tên người sử dụng, UIDs, GIDs, và các mật khẩu được mã hoá cho người sử dụng trên hệ thống. Ngoài ra, file này còn lưu các thông tin khác, như tên đầy đủ của người sử dụng, đường dẫn thư mục, hay hệ thống shell. Từ đó lần ra mối quan hẹ của user và đường dẫn.
