slapd -d1 -h "ldap:///"

- Chạy Master:
Code:
slapd -d1 -h "ldap:///"

- Chạy slurpd trên máy Master:
Code:
slurpd -f /path_to_slapd.conf -r /path_to_replog

Code:
dn: dc=vt04a,dc=com
objectClass: top
objectClass: domain
domainComponent: vt04a

lên Master:
Code:
ldapadd -x -D "cn=Manager,dc=vt04a,dc=com" -W -f base.ldif

-Nếu thành công thì trên Slave sẽ có một Entry được Add vào giống hệt trên Master.

conmeomeo wrote:
Chào các bạn mình có cài cái Apache để chạy mấy file PHP ở localhost. Xin cho mình hỏi là phải cấu hình Apache như thế nào để mỗi khi mình mở một thư mục nào đó thì nó không liệt kê tất cả các file có trong thư mục đó. 


Để làm việc đó bạn có các cách sau:

1/ Disable mod_autoindex. Với cách này thì bạn cần phải chỉnh lại một số cấu hình mặc định trong httpd.conf nữa.

mod_autoindex:

http://httpd.apache.org/docs/2.2/mod/mod_autoindex.html

2/ Bạn có thể cấu hình cho thư mục đó, ví dụ:

Code:
<Directory "/Thu muc chua web">
Options -Indexes
Order allow,deny
Allow from all
</Directory>


Bạn chú ý thêm -Indexes vào sau Options.

Theo mình nếu bạn làm lập trình thì với mỗi thư mục bạn nên vứt vào đó một trang index.html hoặc index.htm cho chắc ăn.
