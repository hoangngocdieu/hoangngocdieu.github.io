Hi all,
tôi đang cài 1 PXE server trên Ubuntu 12.04LTS để cài đặt centos cho các máy trong LAN.
- Tôi đã cài đặt TFTP, DHCP, NFS.
- Từ client, tôi đã thấy được menu boot nhưng không thấy được source cài đặt tôi share bằng NFS. (no driver found)
file pxelinux.cfg/default:
Code:
DEFAULT vesamenu.c32
TIMEOUT 600
ONTIMEOUT BootLocal
PROMPT 0
MENU INCLUDE pxelinux.cfg/pxe.conf
NOESCAPE 1
LABEL BootLocal
localboot 0
TEXT HELP
Boot to local hard disk
ENDTEXT
MENU BEGIN CentOS
MENU TITLE CentOS
LABEL Previous
MENU LABEL Previous Menu
TEXT HELP
Return to previous menu
ENDTEXT
MENU EXIT
MENU SEPARATOR
MENU INCLUDE centos/centos.menu
MENU END

file centos/centos.menu
Code:
LABEL 1
MENU LABEL CentOS 5.8 (32-bit)
KERNEL centos/5.8/i386/vmlinuz
APPEND netboot=nfs nfsroot=172.16.1.101:/srv/install/centos/5.8/i386/ lang=us keymap=us ip=dhcp ksdevice=eth1 noipv6 initrd=centos/5.8/i386/initrd.img ramdisk_size=10000 quiet splash --
TEXT HELP
Install CentOS 5.8 (32-bit)
ENDTEXT
