Máy chủ nên cài Windows Server 2000 hay 2003, lên domain và cài đặt 2 dịch vụ sau: DNS và IIS

Hướng dẫn cài đặt DNS ( Domain name services):
Vào Add/Remove Program -> Windows Component -> Network Services -> DNS

Cấu hình DNS:
vào Administrator tools -> DNS -> Nhấp vào dấu + chổ tên máy chủ -> mở Forward Lookup Zones và xoá 2 dòng trong đó đi
Nhấp phải vào Forward Lookup Zones chọn New Zone -> next ...-> Zone name ( tên domain cuả máy chủ) ->next-> chọn Allow both nonsercure and sercure dynamic update ->next -> Finish

Tiếp theo, right click lên Zone vưà tạo, chọn New Alias -> ô Alias name gỏ www -> ô Fully... chọn Browse -> nhấn chọn tên máy chủ _> Forward Lookup Zone -> chọn domain name -> chọn ámy chủ -> ok

Cấu hình cho Reverse Lookup Zone:
Nhấp phải vào Reverse Lookup Zone chọn new Zone -> Next... _> Network ID: gõ 3 cột đầu cuả IP cuả server ví dụ: 192.168.1 -> Next -> cho Allow both nonsercure and sercure dynamic update-> next -> Finish
Nhấp phải lên Zone vưà tạo, chọn New Pointer -> nhập số cột cuối cùng cuả số IP -> Browse -> Giống bước cuả Forward Lookup Zone.

Ktra lại kquả: nhấp phải tên máy, chọn Launch nslookup nếu thấy tên domain trùng với số ip máy là ok

Cài đặt IIS:
Vào Add/Remove Programe -> Windows Component -> Application Server -> IIS

Cấu hình IIS:
Vào Administrator tools -> IIS
click vào dấu + tên máy chủ -> website -> Nhấp phải vào website chọn new -> Website ( dòng đầu tiên) -> next
Description: gõ tên trang web muốn tạo ví dụ: www.tênmáychủ.com; -> next -> Host header for this website: gõ vào domain name ví dụ www.domainname.com -> next -> path: chọn đường dẫn tới thư mục chưá trang web. -> Finish
Nhấp phải vào tên trang web ( www.domainname.com) -> property -> document -> remove tất cả, add tên trang chủ mặc định ( ví dụ index.htm) -> ok

Lúc này bạn đã hoàn tất việc cài đặt web server, bạn có thể kiểm tra bằng cách vào IE, gõ tên trang web bạn đã tạo và xem kết quả ( các máy con phải vào domain mới xem được web)

Mình tìm thấy cái này, bạn thử xem:

Firewall blocks networked printer and other device IP Addresses. Printer message is: “Printer Properties cannot be displayed. The Print Spooler is not Running”

Problem: One or more of the following:
1. When attempting to print, you get the message in title
2. When attempting to View Home Network, it is not visible; computer, drives, devices, and files and folders not available/accessible
3. Fax or other devices attached to computer no longer work properly

Possible Causes:
1. Installed a new version or new virus protection/firewall software that is blocking access to the network and the IP addresses of computers/drives/routers/drives/files.
2. Print spooler is really not running

Diagnose:
1. Turn off firewall protection temporarily – 15 minutes
2. Attempt to print and or view devices
a. If can print or view, firewall software is blocking access
b. If cannot print, then Print Spooler may really not be running.

Fix Firewall Problem. Add IP addresses of all computers, devices, routers accessed
1. Open Security Software (The Following tutorial is based on McAfee Security Center v 7.0 – One of the least intuitive, multiple replicative terms, and most difficult software navigations I have ever seen)
2. In Security Center Home of Administrative Computer – there are two Home entries on the Home page
a. In listing under Home in top left of pane, click on Manage Network until bold
b. Router IP address will be displayed. Record it.
c. Click on each computer, connection, and device to display details
i. Record the IP address of each
ii. Do not leave this window
3. While in Manage Network window, click on Configure in lower left of left pane
4. Click on Internet and Network in top right hand pane until bold
5. Click on Advanced in top right of right pane
6. Click on Trusted and Banned IPs in top left of left pane until bold
7. Check Trusted IP Addresses (choose Trusted IP Addresses at top of window) list against your recorded network IP Addresses
8. Click Add button
9. Enter IP Address, one at the time, of each device/computer/router not listed in previous pane
10. Name under description
11. Click OK
12. Repeat steps 8-11 for each IP not listed

Restart/Start/Check status Printer Spooler:
1. Click on Start button
2. Choose Run
3. type in “Services.msc” without the quote marks
4. Enter
5. Scroll to Print Spooler – should show if spooler is running
6. Select
7. On left side, click start or restart
8. Or double clidk then set start up to auto and click apply and OK

Hibernate sẽ lưu toàn bộ phiên làm việc cuả bạn vào HDD, vì thế nó sẽ chiếm bớt một phần dung lượng HDD cuả bạn, thường thì với Winxp khoảng không gian dành cho hibernate khoảng 256 Mb, tuy nhiên cái lợi là khi tắt máy xong, cho dù có rút nguồn hay mất điện đột ngột thì lần sau mở lên, phiên làm việc vẫn không mất

Thường có 3 cách để giữ lại phiên làm việc
- Stanby: dùng khi bạn phải làm 1 việc gì đó và sẽ quay lại ngay ( cái này thì nó giữ điện cho toàn hệ thống, tuy nhiên ở mức tốn ít điện năng)
- Hibernate: Dùng trong trường hợp bạn có việc gì đó phải đi lâu ( khoảng vài tiếng trở lên)
- còn 1 cách nưã ( quên mất tên gọi rồi! ^_^) cũng hoạt động như Hibernate, tuy nhiên nó lưu phiên làm việc lên Ram ( chỉ cấp điện cho ram) và sẽ mất khi mất điện.

Thường thì mình thường dùng chế độ Hibernate, tuy thời gian vào win chỉ nhanh hơn khi bị turnoff, nhưng mà an toàn hơn 2 cách còn lại. Còn nói về cái hại cuả Hibernate thì hình như ko có, ngoại trừ việc nó tạo ra 1 file trong HDD đễ lưu phiên làm việc và có thể sinh file rác, bên cạnh đ1o nếu ko có nhu cầu lưu phiên làm việc thì bạn nên turnoff máy để giải phóng RAM hoàn toàn
