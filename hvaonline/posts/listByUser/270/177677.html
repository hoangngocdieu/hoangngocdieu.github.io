Máy mình có 1 HDD duy nhất, dung lượng 20GB. Format một ổ C:\ dung lượng 10GB, còn chỗ dư 10GB chưa set gì cả, dùng nó để cài Fedora Core 9.
- Từ cái FC_9_Live_CD mình burn ra file FC9.ISO. Và đặt nó vào C:\ (định dạng FAT32)
- Tạo thư mục root trong C:\root
- Copy 2 file vmlinuz và initrd.img từ trong fle isolinux vào trong C:\root
- Tải http://jaist.dl.sourceforge.net/sourceforge/grub4dos/grub_for_dos-0.4.1.zip
Giải nén thư mục “boot” và file “grldr” từ file grub4dos. Bên trong thư mục “boot” này có một thư mục khác mang tên “grub“:
- Chép thư mục “grub” vào C:\boot
- Chép file “grldr” vào C:\
- Mở file C:\boot\grub\menu.lst thêm vào:
Code:
title Install Linux
kernel (hd0,0)/boot/vmlinuz
initrd (hd0,0)/boot/initrd.img

- Mở file C:\boot.ini và thêm dòng:
C:\grldr=”Start GRUB”
vào dưới cùng của file.

Save lại và Reset máy, vào Grub, chọn Install Linux
