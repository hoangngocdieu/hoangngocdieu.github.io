Đầu tiên, bạn download công cụ Tcpdump và thư viện lipbcap tại http://www.tcpdump.org/

Sau đó, chép vào thư mục /usr/src/redhat/SOURCES

cp tcpdump-3.9.3.tar.gz /usr/src/redhat/SOURCES

cp libpcap-0.9.3.tar.gz /usr/src/redhat/SOURCES

Thực hiện giải nén file

cd /usr/src/redhat/SOURCES

tar -zxvf tcpdump-3.9.3.tar.gz

tar -zxvf libpcap-0.9.3.tar.gz

Vào thư mục vừa giải nén

cd libpcap-0.9.3

Thực hiện cài đặt

./configure

make

make install

Thực hiện cài đặt tương tự như vậy với tcpdump-3.9.3

Cách sử dụng

Capture tất cả traffic với HOST_NAME xem như là source address hoặc dest address

tcpdump host HOST_NAME

tcpdump src host HOST_NAME

tcpdump dst host HOST_NAME

Capture traffic trên port 80, source HOST_NAME

tcpdump src host HOST_NAME port 80

Một số options

-v/-vv/-vvv : xem thông tin chi tiết

-w file_name: ghi thông tin packets vào file

-r file_name: đọc file vừa ghi

make install
