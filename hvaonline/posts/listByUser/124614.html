Thảo luận các loại thiết bị máy tính  Empty  Go to message
máy bạn bị virus rồi con virus này tên WORM_SETRO.A( tên do Macafee ) máy của mình cùng bị trường hợp như bạn.

1. Chạy Windows chế độ Safe Mode .
2.Bấm tổ hợp phím CTRL_ALT_DEL để mở Windows Task Manager
3.Bấm tab Process --> trong danh sách chương trình đang chạy có tên rose.exe (hoặc 1 cái tên file.exe)
4.Chọn rose.exe --> bấm nút End Task hoặc End Process --> đóng Windows Task Manager.
5. Bấm Start --> chọn Run --> gõ Regedit --> bấm OK .
6. Bạn tìm theo đường dẫn sau :

HKEY_LOCAL_MACHINE>SOFTWARE>Microsoft>
Windows>CurrentVersion>Run

7. Trong cửa sổ bên phải bạn xoá dòng có nội dung như sau :

dll = "C:\system32\rose.exe"

8. Đóng Registry Editor .
9. Xử dụng công cụ tìm kiếm trong thư mục Windows những file có tên và xoá : Autorun.inf , NTDETECT.COM , run.reg , systemdate.ini , systemfile.com
10. Khởi động lại máy tính
