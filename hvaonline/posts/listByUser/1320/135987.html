Trước hết là máy bạn phải vào DOS được . Điều này có thể làm bằng cách khi format ổ đĩa :
format Partition C bằng lệnh format C: /s để tạo một HDH Dos
=> Cài windows để có thêm một HDH Windows, cùng các phần mềm liên quan
=> Ghost lại một bản Image của Partition C vào Partition D:
=> Copy file ghost chạy độc lập ghost2k3.exe ( từ thư mục cài Norton Ghost 2003 ) vào Partition D:
