+ Webserver : để chạy các dịch vụ : tạo tài khoản, mua bán đồ hỗ trợ trong game ...
+ Game online server sẽ có phần : Database ( chung cho web và game ) +listener cho game ( để lắng nghe các client kết nối , sử dụng port riêng ).

Giao thức game online thường sử dụng : UDP.
