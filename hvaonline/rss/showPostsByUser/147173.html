Chào mọi người Bạn nào biết cách Disable "Local Area Connection Status" bằng command-line ? chỉ dùm mình với  
Bạn dùng lệnh sau nhé:
netsh interface set interface "Interface_Name" disabled 
Ví dụ:
netsh interface set interface "Local Area Connection" disabled 
Để Enable lại card mạng, thay disabled bằng enabled Lệnh này chỉ thực hiện được trên Windows 2003 Server, Windows Vista, không dùng được trên XP.
