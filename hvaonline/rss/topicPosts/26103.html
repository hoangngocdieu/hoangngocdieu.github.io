Máy mình cũng chạy ở chế độ Medium(16bit). Mình thấy có một lần máy không bị reset là khi có file drwatson chạy, mình dùng chương trình ProcessExplore xem, thường thì file này không chạy. Dr. Watson for Windows is a program error debugger that gathers information about your computer when an error (or user-mode fault) occurs with a program.
