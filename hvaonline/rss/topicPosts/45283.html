Server MPM:     Worker
  threaded:     yes (fixed thread count)
    forked:     yes (variable process count)
Server compiled with....
 -D APACHE_MPM_DIR="server/mpm/worker"
Code:

  core.c
  worker.c

Nhưng khi xem thông qua httpd thì kết quả vẫn là prefork Code:
Architecture:   32-bit
Server MPM:     Prefork
  threaded:     no
    forked:     yes (variable process count)
Server compiled with....
 -D APACHE_MPM_DIR="server/mpm/prefork"
Code:
Compiled in modules:
  core.c
  prefork.c
  http_core.c
  mod_so.c
