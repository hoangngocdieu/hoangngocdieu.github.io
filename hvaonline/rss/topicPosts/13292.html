Hẳn nhiều bạn đã từng nghĩ tại sao ta lại phải ngồi chờ đợi khi cài đặt windowsxp , Phải chăng nó tự động cài có phải hay không ( chỉ mất chừng vài chục phút là xong ta cứ đi chơi một lát quay về là có windows XP rồi) Sau đây ta tìm hiểu một chút về Tools trong đĩa CD Install windows Xp professional. +The winnt.exe ( clean install ) và winnt32.exe ( upgrade ) chạy trong Dos Mode + Để cài đặt tự động ta cần có answerfile là file mà bất cứ câu hỏi nào trong khi cài đặt sẽ được tự động trả lời .và nó được tạo bằng cách sau : Ta sẽ dùng setup manager ( nằm trong gói deploy.cab trong thư mục sau support\tools của CDROM WindowsXP Professional ) Bạn giải nén và chạy Setupmgr.exe sẽ có dạng như sau:
Sau đó chọn next sẽ hiện ra như hình phía dưới : Từ đây bạn có thể lựa chọn là tạo new answerfile or modify an existing answerfile
Tiếp theo là chọn OS muốn cài tự động ( trong ví dụ này là Windows XP Professional)
Tiếp sau là một vài lựa chọn : Provide defaults : các giá trị ngầm định khi cài đặt ( bạn có thể sửa đổi ) nhưng không hòan tòan là automatic. Fully automated : Là lựa chọn mà khi cài đặt sẽ hòan tòan tự động ( các câu trả lời sẽ được lấy từ file mà chúng ta đang tạo. Hide pages : Khi có yêu cầu thì sẽ được lấy câu trả lời từ file này : Câu trả lời có giá trị thì sẽ không có hiển thị messeage trái lại sẽ có messeage và yêu cầu ta trả lời cho đúng.( không hòan tòan là tự động nếu có lỗi) Read only : Việc thiết đặt giống với Hide pages nhưng với sự giới hạn là read – only . GUI attended : sẽ làm việc tự động ở chế độ TEXT nhưng đến mode GUI thì ta phải đưa thông tin vào khi có yêu cầu.
Tiếp theo sẽ chọn việc cài đặt tự động từ CD ROM haylà từ NETWORK
Cuối cùng là các thiết đặt các câu trả lời cho answer file.
Chúng ta phải trải qua vài mục như là điền key Tiếp theo là phần Computer name ( bạn có thể cho một danh sách các tên như ở dưới ( trong trường hợp bạn muốn triển khai cho cả một dàn máy tính)
Phần chọn Password cho Adminstrator ( chú ý nên chọn phần Encrypt để mã hóa password trong answer file
Sau khi hòan thành phần trả lời thì Tools sẽ tạo ra một file ( unattended.bat và unattended.txt đây là tên default).
