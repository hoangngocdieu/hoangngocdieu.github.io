<![CDATA[Latest posts for the topic "Dùng NTFS của XP trong LINUX - Case 1 "]]> http://www.hvaonline.net/hvaonline/posts/list/24.html JForum - http://www.jforum.net Dùng NTFS của XP trong LINUX - Case 1 Dùng NTFS của XP trong LINUX - Case 1 Case 1 : Cài đặt thêm gói cho kernel để access vào NTFS Tải về : http://sourceforge.net/project/showfiles.php?group_id=13956 Cài đặt : Code:
# rpm -ivh kernel-ntfs-***.rpm
phần *** sẽ là phiên bản của kernel trên hệ thống của bạn Tạo directory cho hd có dạng NTFS Code:
# mkdir /mnt/ntfs
ví dụ 1 : mount ổ C của XP Code:
# mount -t ntfs /dev/hda1 /mnt/ntfs
ví dụ 2 : mount ổ D của XP Code:
# mount -t ntfs /dev/hda2 /mnt/nfts
Tự động hóa mount khi khởi động linux vào file /etc/fstab Code:
/dev/hda1 /mnt/nfts ntfs defaults,uid=500,iocharset=utf8 0 0
/dev/hda1 : Vị trí ổ đỉa của bạn trên linux /mnt/nfts : Vị trí mount uid : User id của bấy kỳ iocharset : Mã ngôn ngữ Tham khảo : http://www.linux-ntfs.org/ ------------------ 777 vnhacker.org ]]> http://www.hvaonline.net/hvaonline/posts/list/2791.html#15394 http://www.hvaonline.net/hvaonline/posts/list/2791.html#15394 GMT
