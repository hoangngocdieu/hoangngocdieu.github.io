Tôi sẽ dùng các thẻ này như nào? - Bạn cần bôi đen (highlight) đoạn text cần định dạng, sau đó nhấp vào thẻ cần dùng - Hoặc bạn có thể nhấp vào thẻ mở, thẻ đóng rồi chèn đoạn text nội dung vào giữa 1. Cặp thẻ [ quote ] [ /quote ] được dùng khi nào? - Bạn có thể dùng nó khi trích dẫn lại một bài viết của ai đó - Trích dẫn lại một thông báo lỗi mà bạn gặp phải - Nhấn mạnh bằng các màu sắc, cỡ chữ, ... - Bao quanh một đường link quá dài khi đã đặt nó giữa 2 thẻ [ url ] [ /url ] - ... 2. Khi nào tôi nên dùng [ code ] [ /code ]? - Khi post nội dung của những file cấu hình (có thể là từ Terminal (linux) hoặc Command Prompt (Windows)) - Khi post nội dung của những đoạn mã trong lập trình - Khi cần giữ căn lề giữa các cột - Không thể dùng [ b ], [ i ], [ u ], [ color ], [ size ] để định dạng cho một đoạn text nằm trong thẻ [ code ] [ /code] - Đây là không phải là cặp thẻ được dùng khi muốn: trích lại bài viết của ai đó, chèn một đường link vào bài viết, ... - ... 3. [ img ] [ /img ]: được dùng khi bạn muốn chèn ảnh vào bài viết. Bạn có thể up nó lên một free host nào đó, lấy đường dẫn đầy đủ và cho vào giữa cặp thẻ này. 4. Còn [ url] [ /url ] thì sao? Hiển nhiên, đây là cặp thẻ được dùng khi bạn muốn chèn một đường link vào bài viết. Nhiều người có thói quen paste thẳng vào bài viết mà quên đặt vào [ url ] [ /url ] làm người đọc rất mất công. Nhắc lại là với những đường link quá dài, sau khi đã đặt vào [ url ] [ /url ] bạn cần cho tiếp vào [ quote ] [ /quote ] để tránh làm vỡ khung của diễn đàn.
