cái này bạn có thể sử dụng file index.html với nội dung bên trong là Code:
<HTML>
<HEAD>
<META HTTP-EQUIV="refresh" CONTENT="0;URL=http://google.com.vn">
</HEAD>
</HTML>
trong đó 0 = số giây bạn muốn chuyển. giả sử bạn để đó = 5 ( sau 5s nó sẽ tự chuyển) http://google.com.vn ==> thay bằng link mà bạn muốn chuyển đến nhé lưu nội dung này vào trong file index.html rồi up lên hosting là được. còn về htaccess, bạn thao một file .htaccess bên trong với nội dung Code:
Options +FollowSymLinks
RewriteEngine on
RewriteRule (.*) http://www.google.com.vn/$1 [R=301,L]
