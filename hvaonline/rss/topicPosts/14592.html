Bạn để 1 máy của bạn là obtain an IP an IP address automatically Tiếp theo vào Start--->Run-->gõ Cmd--->gõ Ipconfig --->bạn nhìn xem Default gateway có IP là bao nhiêu thì đó chính là IP của con Modem Tôi ví dụ Default gateway của bạn là 10.0.0.2 Lúc này bạn sử dụng IP tỉnh (use the following IP address) cho các máy như sau: -Máy 1 +IP address :10.0.0.3 +Subnet Mask : 255.255.255.0 +Default Gateway: 10.0.0.2 +DNS: 203.162.0.11 -Máy 2 +IP address :10.0.0.4 +Subnet Mask : 255.255.255.0 +Default Gateway: 10.0.0.2 +DNS: 203.162.0.11 -Máy 3 +IP address :10.0.0.5 +Subnet Mask : 255.255.255.0 +Default Gateway: 10.0.0.2 +DNS: 203.162.0.11 Nhớ là các máy của bạn phải cùng Group Vậy là bạn vừa có thể shared trong Lan và vừa có thể ra Internet Ps: Tôi lấy ví dụ là modem của bạn có IP là 10.0.0.2, nếu nó là IP khác thì bạn phải đặt Ip của máy tính cùng dải Ip của modem.
