Chào bạn, Từ thông tin bạn cung cấp thì tôi nghĩ bạn cần config như sau: Thứ nhất bạn phải đảm bảo là khi bạn đứng ở server sendmail bạn có thể thực hiện được những lệnh sau: # telnet <Postfix's IP> 25 MAIL FROM:nobody RCPT TO: <your email address> DATA Hi, This is a test. . Nếu bạn ko telnet được đến Postfix server và ko gửi mail thành công thì bạn cần cấu hình lại inet_interfaces của Postfix để Postfix listen trên tất cả các NIC. Nếu thành công thì thực hiện step tiếp theo như sau: Trên server sendmail: # vi /etc/mail/sendmail.cf Tìm đến dòng # "Smart" relay host (may be null) DS Sử lại thành # "Smart" relay host (may be null) DS<Postfix's IP> Rồi restart lại sendmail. Test lại xem có được ko. Chúc bạn thành công. 
Chào ngocson2vn, Tôi đã thực hiện cấu hình relay host trên Sendmail và telnet thành công đến Postfix.
