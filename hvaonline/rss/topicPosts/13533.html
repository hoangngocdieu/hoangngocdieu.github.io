rickb wrote:
Mình thấy hầu hết tất cả distro (FC, Ubuntu, RHEL ...) đều có gói này nhưng ko hiểu mục đích chính của gói này là để làm gì ? Khi nào thì ta cần phải install nó ? Gói linux-kernel-headers với lại gói kernel của Linus Torvalds trên site www.kernel.org khác nhau như thế nào ? Ai rành xin giải thích kỹ giùm Thanx 
Tớ tìm đọc và được biết là: Gói kernel-headers nói chung bao gồm các C header files cho Linux kernel. Các header files này định nghĩa các cấu trúc và hệ số cần thiết cho việc xây dựng các chương trình chuẩn. Nó cũng cần thiết cho việc rebuild kernel. Trên site của Ubuntu thì nói thế này:
This package provides headers from the Linux kernel. These headers are used by the installed headers for GNU glibc and other system libraries.
