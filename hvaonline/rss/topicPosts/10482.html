Đúng là tùy theo Version của Windows mà Recovery Agent có hoặc không có default : Microsoft Windows 2000 EFS thì Administrator là được sử dụng như là default Recovery Agent. Nó không có ngầm định thì ta tạo ra sợ gì nào : Step 1: tạo private Key 1 Log on by account muốn tạo encrypt file. 2 Start -> Run ->CMD 3 tại command prompt gõ lệnh sau: cipher/r: filename 4 Sau đó nó sẽ yêu cầu passowrd (Please type in the password to protect your .PFX file:) ( gõ 2 lần ) Step 2: Import recovery certificates and the private key 1. Log on bằng administrator. 2. Start -> Run -> gpedit.msc 3. Local Computer Policy -> Computer Configuration ->Windows Setting -> Security Settings -> Public Key Policies 4. Right-click -> click Add Data Recovery Agent. 5. Select the *.CER file mà bạn vừa tạo ra ở bước trên ( chắc vẫn còn nhớ nó nằm ở thư mục nào chứ nhỉ) 6 Finish Thê là bạn đã tạo ra một Recovery Agent rồi đấy.
