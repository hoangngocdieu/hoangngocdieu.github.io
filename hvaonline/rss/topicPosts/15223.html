<![CDATA[Latest posts for the topic "Cài đặt Ubuntu/Debian LAMP Server (vừa mần/ vừa viết)"]]> /hvaonline/posts/list/24.html JForum - http://www.jforum.net Cài đặt Ubuntu/Debian LAMP Server (vừa mần/ vừa viết) Dành cho những ai làm biếng/ không thích đọc tiếng Tây Lược dịch từ: http://howtoforge.net/ubuntu_debian_lamp_server Bài viết này sẽ hướng dẫn bạn cài một LAMP Server trên Ubuntu/Debian: - Apache2 - MySQL 5 - PHP 5 - phpMyAdmin 1. Cài Apache và PHP: Code:
sudo apt-get install apache2 php5 libapache2-mod-php5
Sau khi cài xong, file cấu hình của Apache được đặt tại: /etc/apache2/apache2.conf và web folder là /var/www 2. Cài MySQL Db Server: Code:
sudo apt-get install mysql-server mysql-client php5-mysql
File cấu hình của MySQL được đặt tại: /etc/mysql/my.cnf Để sờ-cua (hạn chế phần nào mấy thằng dở hơi ngồi hắc hiếk): change Root password Code:
mysql -u root
mysql> USE mysql;
mysql> UPDATE user SET Password=PASSWORD('new-password') WHERE user='root';
mysql> FLUSH PRIVILEGES;
hoặc đơn giản hơn thì dùng: Code:
sudo /etc/init.d/mysql reset-password
3. Cài đặt phpMyAdmin: Code:
sudo apt-get install phpmyadmin
File cấu hình sẽ được đặt tại: /etc/phpmyadmin Mở file /etc/apache2/apache2.conf, thêm vào dòng (chú ý chữ HOA, chữ thường): Code:
Include /etc/phpmyadmin/apache.conf
Restart Apache: Code:
sudo /etc/init.d/apache2 restart
Xem thành quả: Code:
http://127.0.0.1/phpmyadmin
