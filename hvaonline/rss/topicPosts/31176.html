Mr.Kas wrote:
Vấn đề này có thể là do pulseaudio, bạn thử gõ cái này vào trong Terminal xem sao: Code:
pkill pulseaudio
asoundconf unset-pulseaudio
Sau đó restart lại máy. Và nếu không được thì thử cái này xem, Code:
#!/bin/bash
# written by lambda_max
sleep 30
#pulseaudio beenden
pkill pulseaudio
#pulseaudio ausschalten
asoundconf unset-pulseaudio
exit 0
Save lại : ~/fixerror Vào trong trong terminal gõ: Code:
chmod +x ~/fixerror
Tiếp theo vào System =>Preferences =>Sessions => ở mục Startup Programs => add và thêm vào cái Name gì đấy, command thì chọn đúng cái fixerror.
