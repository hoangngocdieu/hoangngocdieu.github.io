Em có 1 server và 1 shared host , thấy speed trên shared host nhanh hơn server nên làm forum, ai ngờ làm cứ bị lỗi data (lúc trước bên server thì không bị) 
Nếu em chạy forum trên shared host và không có quyền điều khiển mysql thì bó tay. Quản lý server có thể restart mysql khi họ cần và em sẽ bị tình trạng "gone away" đều đều. Tình trạng này nói chung không có gì trầm trọng, chỉ có điểm đáng ngại là khi forum đang cập nhật cái gì đó (viết data xuống CSDL) mà họ lại restart mysql thì em có thể bị mất dữ liệu, thậm chí database bị corrupted.
