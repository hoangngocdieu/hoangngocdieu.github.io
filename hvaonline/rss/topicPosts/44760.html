Bạn thử echo %JAVA_HOME% xem biến môi trường set đúng chưa. Biến JAVA_HOME trỏ đến thư mục mẹ của thư mục bin. Sau đó, để sử dụng lệnh trong bin của JAVA_HOME mà không cần nhảy vô đường dẫn của nó thì bạn set tiếp %JAVA_HOME%\bin trong biến môi trường PATH. Cuối cùng, bạn thử kiểm tra lại bằng cách mở command prompt ra, gõ java -version Nếu lỗi ở bước nào bạn đưa lên đây lỗi bước đó chứ bạn nói chung chung là không được thì khó hình dung lắm. P/S: Bài này bạn đặt trong phân mục "Những thảo luận khác" thì không hợp lý rồi.
