Sử dụng wget thông qua Proxy Server

Có nhiều bạn sử dụng Internet trong công ty và công ty thường sử dụng một phần mềm nào đó để làm Proxy Server chẳng hạn như squid

Khi sử dụng wget để tải gói cài đặt nào đó về máy tính của bạn
Code:


wget -c http://www.debian-multimedia.org/pool/main/w/w32codecs/w32codecs_20061022-0.0_i386.deb



--06:15:54-- http://www.debian-multimedia.org/pool/main/w/w32codecs/w32codecs_20061022-0.0_i386.deb
=> `w32codecs_20061022-0.0_i386.deb'
Resolving www.debian-multimedia.org... failed: Temporary failure in name resolution.  


Có thể bạn nhận được thông báo lỗi như sau :
Code:


Temporary failure in name resolution



Sau đây là cách khắc phục.
Trên console sử dụng lệnh
Code:


export http_proxy="192.168.32.1:3128"



Nếu đây là 1 ftp proxy
Code:


export ftp_proxy="192.168.32.1:3128"


Nếu proxy được yêu cầu username và password
Code:


export http_proxy="http://username:password@192.168.32.1:3128"



Đối với anh em sử dụng cgywin trên Windows thì sử dụng
Code:


set http_proxy=http://192.168.32.1:3128



192.168.32.1 Địa chỉ IP của proxy server
3128 : Port (mặc định Squid sử dụng port 3128, bạn có thể sử dụng port khác tùy theo cấu hình của công ty bạn)
