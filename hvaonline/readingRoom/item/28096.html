Cài đặt:
Mở file source.list để add thêm source của Automatix vào:
Code:


sudo gedit /etc/apt/sources.list {nhập password root}



Edit: Copy đoạn này vào:
deb http://www.getautomatix.com/apt dapper main (nếu bạn dùng bản 6.06)
deb http://www.getautomatix.com/apt breezy main (nếu bạn dùng bản 5.10
deb http://www.getautomatix.com/apt kubuntu main (nếu bạn dùng Kubuntu
Thậm chí cả những ai đang dùng Mepis deb http://www.getautomatix.com/apt mepis main

Chạy đoạn code này trong Terminal:
Code:


wget http://www.getautomatix.com/apt/key.gpg.asc gpg --import key.gpg.asc gpg --export --armor 521A9C7C | sudo apt-key add -



Rùi bây giờ thì:
Code:


sudo apt-get update



Cuối cùng:
Code:


sudo apt-get install automatix




Vậy là kết thúc quá trình cài đặt. Để chạy thì type "Automatix" vào Terminal, hoặc nó đã có sẵn trong Applications/System Tools/
Cứ làm theo các bước hướng dẫn để update và cài đặt cái j mình muốn.
